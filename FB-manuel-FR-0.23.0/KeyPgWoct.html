WOct
 
Conversion d'un nombre dans sa représentation octale Unicode

Syntaxe

Declare Function WOct( ByVal number As Byte ) As WString
Declare Function WOct( ByVal number As Byte, ByVal digits As Integer ) As WString

Declare Function WOct( ByVal number As Short ) As WString
Declare Function WOct( ByVal number As Short, ByVal digits As Integer ) As WString

Declare Function WOct( ByVal number As Integer ) As WString
Declare Function WOct( ByVal number As Integer, ByVal digits As Integer ) As WString

Declare Function WOct( ByVal number As LongInt ) As WString
Declare Function WOct( ByVal number As LongInt, ByVal digits As Integer ) As WString

Usage

result = WOct( number [, digits ] )

Paramètres

number
Nombre à convertir dans sa représentation octale.
digits
Nombre de chiffres désirés dans la chaîne retournée.

Valeur retournée

La représentation octale Unicode du nombre, tronquée ou complètée avec des zéros ("0") au nombre de chiffres, s'il a été spécifié.

Description

Retourne représentation WString (Unicode) octale de number. Chiffres octals de 0 à 7.

Si vous spécifiez digits > 0, la chaîne résultat aura exactement cette longueur. Elle sera tronquée ou complétée avec des zéros sur la gauche, si nécessaire.

La longueur de la chaîne ne dépassera pas le nombre de bits maximum requis pour le type de number (3 caractères pour Byte (octet), 6 pour Short (entier court), 11 pour Integer (entier), et 22 pour LongInt (entier long))

Exemple

Print WOct(54321)
Print WOct(54321, 4)
Print WOct(54321, 8)


produira la sortie:

152061
2061
00152061


Différences de dialectes

  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __Woct.

Différences de plate-forme

  • Les (w)strings Unicode ne sont pas supportées dans le portage DOS de FreeBASIC.

Différences avec QB

  • Dans QBASIC Unicode n'est pas supporté.
