__FB_MT__
 
Définition intrinsèque (valeur de la macro) fixée par le compilateur

Syntaxe

__FB_MT__

Description

__FB_MT__ indique si l'option "multi-thread" -mt a été spécifiée sur la ligne de commande au moment de la compilation.

Retourne non-zéro (-1) si l'option a été spécifiée. Autrement, retourne zéro (0).

Exemple

#if __FB_MT__ 
        #print Using multi-threaded library
#else
        #print Using Single-threaded library
#endif
