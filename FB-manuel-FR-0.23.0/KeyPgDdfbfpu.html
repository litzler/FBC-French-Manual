__Fb_Fpu__
 
Définition intrinsèque fixée par le compilateur

Syntaxe

__FB_FPU__

Description

Défini comme "sse" si l'arithmétique en virgule flottante SSE est activée ou "x87" autrement.

Exemple

#if __FB_FPU__ = "sse"
  ' ... instructions only for SSE ...
#else
  ' ... instructions not for SSE ...
#endif
