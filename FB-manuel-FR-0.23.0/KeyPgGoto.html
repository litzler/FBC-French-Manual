Goto
 
Déclaration de contrôle de flux pour sauter vers une autre partie du programme

Syntaxe

Goto label

Description

L'exécution du code saute vers une ligne avec une étiquette.

Pour plus de lisibilité du code source, l'utilisation excessive de Goto doit être évitée en faveur de structures plus modernes telles que Do...Loop, For...Next, Sub et Function.

Exemple

    Goto there

backagain:
    End

there:
    Print "Welcome!"
    Goto backagain


'' Compile with -lang qb or fblite

'$lang: "qb"

1 Goto 3
2 End
3 Print "Welcome!"
4 Goto 2


Différences de dialecte

  • Les numéros de ligne ne sont permis que dans les dialectes -lang qb et -lang deprecated.

Différences avec QB

  • Aucune
