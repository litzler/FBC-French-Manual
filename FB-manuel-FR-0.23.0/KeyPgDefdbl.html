DefDbl
 
Spécifie un type de données par défaut pour une série de noms de variables

Syntaxe

DefDbl start_letter[-end_letter ][, ...]

Paramètres

start_letter
la première lettre dans la série
end_letter
la dernière lettre dans la série

Description

DefDbl précise que les variables et les tableaux qui ne sont pas déclarés avec un type de données - ou non déclarés entièrement - sont implicitement déclarés de type Double si la première lettre de leur nom correspond à une certaine lettre ou se trouve dans une plage inclusive de lettres.

Exemple

Ceci fera de aNum un nombre décimal Double-précision car sa première lettre est dans la plage a-d:
'' Compile with -lang fblite or qb

#lang "fblite"

DefDbl a-d
Dim aNum 'implicit: As Double

Print Len(aNum) ' Prints 8, the number of bytes in a double.


Différences de dialecte

  • Disponible seulement dans les dialectes -lang qb et -lang fblite.

Différences avec QB

  • Aucune
