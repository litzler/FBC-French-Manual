CVI
 
Conversion d'une chaîne de 4 octets en une variable entière

Syntaxe

Declare Function CVI ( ByRef str As Const String ) As Integer

Usage

result = CVI( str )

Paramètres

str
Une chaîne (String) ayant exactement 4 octets de longueur, copie d'une variable entière.

Valeur retournée

Une variable entière (Integer) pour contenir la copie binaire d'un ENTIER.

Description

Retourne une valeur entière (Integer) 32 bits en utilisant les données contenues dans une chaîne qui a exactement 4 octets de longueur. Une valeur entière zéro (0) est retournée si la chaîne n'a pas exactement 4 octets de longueur.

CVI est utilisé pour convertir des chaînes-4-octets crées avec MKI.

Cette fonction peut aussi convertir des valeurs entières 32 bits à partir de la mémoire ou d'un tampon fichier sans l'utilisation d'une structure Type. Cependant, tout comme avec la structure type, une attention particulière doit être prise lors de l'utilisation CVI pour convertir des chaînes qui ont été lues à partir d'un tampon. Les fonctions CVI et MKI sont dépendantes du processeur. Elles retourneront des résultats différents dépendant de l'endian du processeur.

Exemple

Dim a As Integer, b As String
a=4534
b=MKI(a)
Print a, CVI(b)
Sleep


Différences de dialectes

  • Dans le dialecte -lang qb, CVI attend des chaînes-2-octets, car un entier QB integer n'a que 16 bits. Seuls les deux premiers octets de la chaîne sont utilisés, même si la chaîne à une longueur supérieure à 2 octets.

Différences avec QB

  • Dans QB une erreur apparaît si la chaîne passée a une longueur supérieure à 2 octets.
