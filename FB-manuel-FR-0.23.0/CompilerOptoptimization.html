Option du compilateur: -O
 
Définit le niveau d'optimisation de GCC

Syntaxe

-O < level >

Parameters

level
Le niveau d'optimisation: 0, 1, 2, 3 ou max (3).

Description

Spécifie le niveau d'optimisation à passer à GCC quand on utilise -gen gcc.
