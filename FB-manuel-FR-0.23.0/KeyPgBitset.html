BitSet
 
Obtient la valeur d'un entier dont on a positionné un certain bit à un (1).

Syntaxe

#define BitSet( value, bit_number ) ((value) Or (Cast(TypeOf(Value), 1) Shl (bit_number)))

Usage

result = BitSet( value, bit_number )

Paramètres

value
La valeur entière.
bit_number
L'indice du bit à mettre à zéro.

Valeur retournée

Retourne la valeur entière avec le bit spécifié mis à 1.

Description

Cette macro se développe en une copie de la valeur entière value avec le bit bit_number "validé" (à on, ou `1`). Se comporte comme `value Or (1 Shl bit_number)`.

La plage valide de valeurs pour bit_number dépend de la taille, en bits, de`TypeOf(value)`, qui est `0` par le biais de `SizeOf(value) * 8 - 1`. Voir Limites des types de données standard pour un tableau des types de données standard et leurs tailles.

Exemple

Print BitSet(4, 0)
Print Hex(BitSet(1ull, 63))

produira la sortie:

5
8000000000000001

Différences de dialectes

  • Non disponible dans le dialecte -lang qb.

Différences avec QB

  • Nouveau pour FreeBASIC
