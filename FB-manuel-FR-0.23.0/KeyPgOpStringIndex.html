Opérateur [ ] (Indice de chaîne)
 
Renvoie une référence à un caractère dans une chaîne

Syntaxe

Declare Operator [] ( ByRef lhs As String, ByRef rhs As Integer ) As UByte
Declare Operator [] ( ByRef lhs As ZString, ByRef rhs As Integer ) As UByte
Declare Operator [] ( ByRef lhs As WString, ByRef rhs As Integer ) As T

Notez que Opérateur [ ] (Indice de chaîne) renvoie une référence. Voir CompilerFAQ.

Usage

result = lhs [ rhs ]

Paramètres

lhs
La chaîne.
rhs
Décalage, basé sur zéro, par rapport au premier caractère.
T
Le type de caractères "larges" (varie selon la plateforme).

Description

Cet opérateur renvoie une référence à un caractère spécifique dans une chaîne:
  • Cet opérateur ne doit pas être utilisé en cas de chaîne vide parce que la référence n'est pas définie (induisant une erreur d'exécution).
  • Sinon, l'utilisateur doit s'assurer que l'indice ne dépasse pas la fourchette "[0, Len(lhs) - 1]". En dehors de cette plage, les résultats sont indéfinis.

Exemple

Dim a As String = "Hello, world!"
Dim i As Integer

For i = 0 To Len(a) - 1
    Print Chr(a[i]);
Next i
Print


Affichera
Hello, world!
