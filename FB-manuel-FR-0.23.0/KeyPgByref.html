ByRef
 
Spécificateur de déclaration pour passer explicitement un paramètre par référence

Syntaxe

ByRef param As datatype

Usage

[ Declare ] { Sub | Function } proc_name ( ByRef param As datatype )

Description

Passe une variable par référence, c'est à dire son adresse, à une sous-routine ou une fonction. Quand une variable est passée par référence, le contenu de la variable peut être modifié par la cible sous-routine ou fonction.

Dans les dialectes -lang qb et -lang fblite, ByRef est la convention de passage de paramètre par défaut, sauf si l'Option ByVal est active.

L'opposé de ByVal.

Exemple

Dim MyVar As Integer

Sub ChangeVar(ByRef AVar As Integer)
    AVar = AVar + 1
End Sub

MyVar = 1
Print "MyVar: "; MyVar 'output = 1
ChangeVar MyVar
Print "MyVar: "; MyVar 'output = 2
Sleep
End


Différences de dialecte

  • Dans le dialecte -lang fb, ByVal est la convention de passage de paramètre par défaut pour tous les types intégrés à l'exception de String; Chaîne (String) et les Types définis par l'utilisateur sont passés ByRef par défaut.
  • Dans les dialectes -lang qb et -lang fblite, ByRef est la convention de passage des paramètres.
