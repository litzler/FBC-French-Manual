__Fb_Option_Gosub__
 
Définition intrinsèque (valeur de la macro) fixée par le compilateur

Syntaxe

__FB_OPTION_GOSUB__

Description

Indique comment GoSub et Return seront pris en charge au moment de la compilation. Si l'option est validée (-1) alors GoSub est permis et Return est reconnu seulement comme "retour-de-gosub". Si l'option n'est pas validée (0) alors GoSub n'est pas permis et Return est reconnu seulement comme "retour-de-procédure".

Cette valeur macro peut être changée au moment de la compilation. Option Gosub validera l'option (valide le support de gosub) et Option Nogosub annulera l'option (annulation du support de gosub).

__FB_OPTION_GOSUB__ retourne zéro (0) si l'option n'a pas été validée. Retourne non-zéro (-1) si l'option a été validée.

Exemple

#if( __FB_OPTION_GOSUB__ <> 0 )
    '' turn off gosub support
    Option nogosub
#endif


Différences de dialecte

  • Par défaut, -1 dans le dialecte -lang qb et 0 dans tous les autres dialectes.

Différences avec QB

  • Nouveau pour FreeBASIC
