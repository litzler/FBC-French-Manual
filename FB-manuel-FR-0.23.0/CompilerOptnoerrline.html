Option du compilateur: -noerrline
 
Ne pas afficher la ligne de source où l'erreur s'est produite

Syntaxe

-noerrline

Description

L'option du compilateur -noerrline force le compilateur a ne pas afficher la ligne du source où l'erreur s'est produite. Utile si un IDE analyse des messages d'erreur.
