Notions de base sur les entrées
 
Obtenez de l'information dans votre programme.

L'entrée est la vie de tout programme. Si vous ne pouvez pas obtenir quelque chose dans votre programme, comment allez-vous tirer quelque chose de lui? Ce que que vous trouverez ici, ce sont les bases sur la façon d'obtenir des informations dans votre programme FreeBASIC.

Voici un programme très basique qui vous demandera votre nom:

'Créer une place pour mettre le nom utilisateur
Dim As String strMyName

' Interroger pour le nom utilisateur et le stocker dans la chaine 'strMyName'
Input "Quel est votre nom? ", strMyName

' Attente une demi seconde
Sleep 500

' Afficher le nom
Print
Print "Je sais que votre nom est "; strMyName
Print

' Attendre quelqu'un appuie sur une touche avant de terminer
Print "Appuyez sur une touche pour terminer"
Sleep


INPUT est le moyen le plus simple pour obtenir des informations de quelqu'un. Ils ont juste à taper du texte et à appuyer sur Entrée lorsque c'est fait.

Que faire si vous ne voulez qu'une seule frappe? Le plus simple est d'utiliser la méthode GetKey. GetKey vous renvoie la valeur ASCII de la touche qui a été appuyée.

' Demande une entrée à l'utilisateur
Print "Appuyez sur votre touche favorite:"

' Définit un endroit pour garder la valeur ASCII de la touche
Dim As Integer strKeyPress

' Continue jusqu'à ce qu'une touche soit appuyée
Do
    strKeyPress = GetKey    
Loop Until strKeyPress <> 0    

' Affiche la touche appuyée par l'utilisateur
Print
Print "Votre touche favorite est: "; Chr(strKeyPress)                        
                        
' Attend qu'une touche soit appuyée pour terminer
Print
Print "Appuyez sur une touche pour terminer"
Sleep
