Option du compilateur: -profile
 
Active la fonction de profilage

Syntaxe

-profile

Description

L'option -profile du compilateur active la fonction de profilage. Après l'exécution d'un exécutable compilé avec cette option, un fichier gmon.out sera créé dans le répertoire du programme, permettant d'utiliser GPROF pour l'analyse l'exécution du programme.
