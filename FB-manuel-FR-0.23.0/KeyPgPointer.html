Pointer
 
Déclaration d'un modificateur de type de variable

Syntaxe

Dim symbolname As DataType {Pointer | Ptr}

Description

Déclare un pointeur de variable. Identique à Ptr.

Exemple

Dim p As ZString Pointer
Dim text As String
text = "Hello World!"
p = StrPtr(text) + 6
Print text
Print *p

'' Output:
'' Hello World!
'' World!


Différences de dialecte

  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __Pointer.

Différences avec QB

  • Nouveau pour FreeBASIC
