TimeValue
 
Forme une Date Serial à partir d'une chaîne heure

Syntaxe

Declare Function TimeValue ( ByRef timestring As String ) As Double

Usage

#include "vbcompat.bi"
result = TimeValue( timestring )

Paramètres

timestring
la chaîne à convertir

Valeur retournée

Retourne une Date Serial à partir d'une chaîne heure.

Description


La chaîne heure doit avoir le format "23:59:59" ou "11:59:59PM"

Le compilateur ne reconnaît cette fonction que si vbcompat.bi ou datetime.bi est inclus.

Exemple

#include "vbcompat.bi"

Dim ds As Double = TimeValue("07:12:28AM")

Print Format(ds, "hh:mm:ss")


Différences avec QB

  • N'existe pas dans QB. Cette fonction est apparue dans PDS et VBDOS
