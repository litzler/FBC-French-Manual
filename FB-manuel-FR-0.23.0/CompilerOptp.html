Option du compilateur: -p
 
Ajouter un chemin de recherche pour les bibliothèques

Syntaxe

-p < library path >

Paramètres

library path
Le chemin du répertoire, relatif ou absolu, où chercher les fichiers des bibliothèques.

Description

L'option -p du compilateur ajoute un chemin de recherche pour les bibliothèques. Par défaut, les bibliothèques sont à chercher dans le répertoire système FreeBasic des bibliothèques et dans le répertoire courant.
