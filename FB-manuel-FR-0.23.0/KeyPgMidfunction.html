Mid (Fonction)
 
Retourne une sous-chaîne d'une chaîne

Syntaxe

Declare Function Mid ( ByRef str as Const String, ByVal start as integer ) as String
Declare Function Mid ( ByVal str as Const WString Ptr, ByVal start as integer ) as WString
Declare Function Mid ( ByRef str as Const String, ByVal start as integer, ByVal n as integer ) as String
Declare Function Mid ( ByVal str as Const WString Ptr, ByVal start as integer, ByVal n as integer ) as WString

Usage

result = Mid[$]( str, start [, n ] )

Paramètres

str
La chaîne source.
start
La position de départ de la sous-chaîne dans str.
n
La longueur de la sous-chaîne, en caractères.

Description

Retourne une sous-chaîne commençant à start dans str. Si str est vide, la chaîne nulle ("") est retournée. Si start <= 0, la chaîne nulle ("") est retournée.

Dans la première forme de Mid, tous les autres caractères sont retournés. Dans la seconde forme, si n < 0 ou n >= len(str) alors tous les autres caractères sont retournés.

Exemple

Print Mid("abcdefg", 3, 2)
Print Mid("abcdefg", 3)
Print Mid("abcdefg", 2, 1)

produira la sortie:
cd
cdefg
b
Un exemple Unicode:
Wiki: Code rendu de cette façon pour permettre l'affichage des caractères Unicode.


dim text as wstring * 20
text = "Привет, мир!"
print mid(text, 6, 4) ' displays "т, м"


Différences de plate-formes

  • DOS ne supporte pas la version chaîne larges-caractères de Mid.

Différences de dialectes

  • Le suffixe de type de chaîne "$" est obligatoire dans le dialecte -lang qb.
  • Le suffixe de type de chaîne "$" est facultatif dans in the les dialectes -lang fblite et -lang fb.

Différences avec QB

  • QB ne supporte pas Unicode.
