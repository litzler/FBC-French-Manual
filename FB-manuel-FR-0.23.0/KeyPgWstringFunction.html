Wstring (Fonction)
 
Remplit une WString, d'une certaine longueur, avec un large caractère défini

Syntaxe

Declare Function WString ( ByVal count As Integer, ByVal ch_code As Integer ) As WString
Declare Function WString ( ByVal count As Integer, ByRef ch As Const WString ) As WString

Usage

result = WString( count, ch_code )
ou
result = WString( count, ch )

Paramètres

count
Un entier spécifiant la longueur de la WString à créer.
ch_code
Un entier spécifiant le caractère Unicode à utiliser pour remplir la WString.
ch
Une WString dont le premier caractère sera utilisé pour remplir la WString.

Valeur retournée

La WString créée. Une WString vide sera retournée si ch est une chaîne vide ou si count <= 0.

Description

WString génère une WString remplie de count copies d'un caractère Unicode. Cette chaîne peut être affichée à l'écran (avec Print) ou assignée à une WString dimensionnée précédemment avec Dim.

Exemple

Print WString( 4, 934 )         
Print WString( 5, WStr("Indeed") )   
End 0

Différences de plate-forme

  • Les (w)strings Unicode ne sont pas supportées dans le portage DOS de FreeBASIC

Différences de dialectes

  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __Wstring.

Différences avec QB

  • QBasic ne supporte pas Unicode
