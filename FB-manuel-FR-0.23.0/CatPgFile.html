Fonctions d'entrée/sortie fichier
 
Déclarations et les procédures pour travailler avec les fichiers et les périphériques.

Description

Ces déclarations et procédures fournissent des possibilités d'entrée/sortie fichiers et périphériques. Des numéros de fichier peuventt être liés à des fichiers ou des périphériques, qui peuvent être lues ou écrites à l'aide de données formatées (mode texte) ou non-formatée (mode binaire). En mode binaire, les fichiers et périphériques peuvent être lus ou écrits à des emplacements arbitraires. Pour applications "multi-thread", les fichiers et les périphériques peuvent également être verrouillés.

Ouverture de fichiers ou de périphériques
Procédures et autres mots-clés qui permettent l'accès, en lecture ou écriture, à un fichier ou un périphérique.
Lecture et écriture de fichiers ou de périphériques
Procédures pour lire et écrire des données dans un fichier ouvert ou un périphérique.
Position dans un fichier et autres informations
Procédures qui déterminent où la lecture et l'écriture aura lieu au sein d'un fichier ouvert.

Ouverture de fichiers ou de périphériques
FreeFile
Obtient un numéro de fichier disponible qui peut être utilisé pour lire ou écrire à partir de fichiers ou de périphériques.
Open
Lie un numéro de fichier à un fichier physique pour fournir des capacités de lecture et d'écriture.
Open Com
Lie un numéro de fichier à un port de communication.
Open Cons
Lie un numéro de fichier aux flux d'entrée ou de sortie standard.
Open Err
Lie un numéro de fichier aux flux d'entrée et d'erreur standard.
Open Lpt
Lie un numéro de fichier à un périphérique d'impression.
Open Pipe
Lie un numéro de fichier aux flux d'entrée ou de sortie d'un processus.
Open Scrn
Lie un numéro de fichier directement à la console.
Close
Déconnecte un numéro de fichier d'un fichier ou d'un périphérique.
Reset
Déconnecte tous les numéros de fichiers actifs.

Modes d'entrée/sortie fichier
Input (Mode fichier)
Les données de texte peuvent être lues à partir du fichier.
Output
Les données de texte peuvent être écrites dans le fichier.
Append
Les données de texte sont ajoutées à la fin d'un fichier qui est déjà ouvert (Output).
Binary
Des données binaires peuvent être lues ou écrites dans le fichier.
Random
Des blocs de données d'une certaine taille peuvent être lus et écrits dans le fichier.

Privilèges d'accès aux fichiers
Access
Un aperçu des privilèges d'accès aux fichiers.
Read (Accès fichier)
Les données binaires peuvent seulement être lues à partir du fichier.
Write (Accès fichier)
Les données binaires peuvent seulement être écrites dans le fichier.
Read Write(Accès fichier)
Les données binaires peuvent être lues et écrites dans le fichier..

Encodage des caractères
Encoding
Spécifie l'encodage des caractères d'un fichier.
Lecture et écriture de fichiers ou de périphériques
Input #
Lit une liste de valeurs à partir d'un fichier ou un périphérique.
Write #
Ecrit une liste de valeurs dans un fichier ou un périphérique.
Input()
Lit un nombre de caractères à partir d'un fichier ou d'un périphérique.
Winput()
Lit un nombre de "caractères larges" à partir d'un fichier ou d'un périphérique.
Line Input #
Lit une ligne de texte à partir d'un fichier ou d'un périphérique.
Print #
Écrit des données texte dans un fichier ou un périphérique.
Print # Using
Écrit des données formatées dans un fichier ou un périphérique.
Put #
Écrit des données binaires vers un fichier ou un périphérique.
Get #
Lit des données binaires à partir d'un fichier ou d'un périphérique.

Position dans un fichier et autres informations
LOF
Obtient la longueur (en octets) d'un fichier.
LOC
Obtient la position dans un fichier de la dernière lecture ou écriture.
EOF
Renvoie "Vrai" si toutes les données ont été lues à partir d'un fichier.
Seek (Déclaration)
Définit la position dans un fichier de la prochaine opération de lecture ou d'écriture.
Seek (Fonction)
Obtient la position dans un fichier de la prochaine opération de lecture ou d'écriture.
Lock
Limite l'accès en lecture/écriture sur un fichier ou une partie d'un fichier.
Unlock
Supprime une restriction d'accès précédente imposée par la commande Lock.
