Hex
 
Retourne la valeur hexadécimale du nombre donné

Syntaxe

Declare Function Hex Overload ( ByVal number As Byte ) As String

Declare Function Hex ( ByVal number As Short ) As String

Declare Function Hex ( ByVal number As Integer ) As String
Declare Function Hex ( ByVal number As Integer, ByVal digits As Integer ) As String

Declare Function Hex ( ByVal number As LongInt ) As String
Declare Function Hex ( ByVal number As LongInt, ByVal digits As Integer ) As String


Usage

result = Hex[$]( number [, digits ] )

Paramètres

number
Un nombre ou expression évalué comme nombre. Un nombre à virgule flottante sera converti en LongInt.
digits
Facultatif: nombre de chiffres à retourner.

Valeur retournée

Une String (chaîne) contenant la représentation hexadécimale non-signée de number.

Description

Retourne la représentation hexadécimale non-signée de l'entier number. Chiffres héxadécimaux de 0 à 9, ou A à F.

Si vous spécifiez digits > 0, la chaîne résultat aura exactement cette longueur. Elle sera tronquée ou complètée avec des zéros sur la gauche, si nécessaire.

La longueur de la chaîne ne dépassera pas le nombre de bits maximum requis pour le type de number (8 pour un Integer (entier), 16 pour un LongInt (entier long)).

Si vous désirez l'opposé, par exemple convertir une chaîne binaire vers un nombre, la meilleure façon de le faire est d'ajouter "&H", au début de la chaîne, et la convertir en utilisant ValInt ou ValLng, de façon similaire à une chaîne numérique normale. Par exemple ValInt("&HFF")

Exemple

'54321 is D431 in hex
Print Hex(54321)
Print Hex(54321, 2)
Print Hex(54321, 5)

produira la sortie:

D431
31
0D431

Différences de dialectes

  • Le suffixe de type chaîne "$" est obligatoire dans le dialecte -lang qb.
  • Le suffixe de type chaîne "$" est facultatif dans les dialectes -lang fblite et -lang fb.

Différences avec QB

  • Dans QBASIC, il n'y avait aucun moyen de spécifier le nombre de chiffres retournés.
  • La taille de la chaîne retournée était limitée à 32 bits, ou 8 chiffres hexadécimaux.
