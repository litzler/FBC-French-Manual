__FB_WIN32__
 
Définition intrinsèque fixée par le compilateur

Syntaxe

__FB_WIN32__

Description

Définition sans valeur créée au moment de la compilation, si compilation vers une cible Win32. Par défaut dans la version Win32 hébergée, ou actif si l'option -target win32 de la ligne de commande est utilisée. Peut être utilisé pour compiler des parties d'un programme que si la cible est Win32.

Exemple

#ifdef __FB_WIN32__
  ' ... instructions only for Win32 ...
  ' ... GetProcAddress ...
#else
  ' ... instructions not for Win32 ...
#endif
