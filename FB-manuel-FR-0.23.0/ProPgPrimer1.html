Premier livre #1 FreeBASIC
 
Ce livre élémentaire est destiné aux "débutants débutants", pour ceux qui commencent tout juste à apprendre à programmer et à utiliser FreeBASIC dans ce but.

L'apprentissage du langage

Apprendre un langage de programmation, c'est apprendre à écrire des mots et savoir ce qu'ils veulent dire quand ils sont écrits. Nous n'avons pas besoin de les apprendre tous à la fois. Mais apprendre quelques mots importants qui font quelque chose va nous aider à démarrer. Ici nous allons nous contenter de se concentrer sur ces mots clés:

  • Dim
  • Print
  • Input
  • For...Next
  • If...Then
  • Do...Loop


Bonjour le monde! (Hello World!)

Aucune référence débutants n'est complet sans cet exemple.

Print "Hello World!"

Le texte entre la paire de guillemets est une chaîne littérale. La déclaration Print est utilisée pour la sortie du texte à l'écran. Si vous pouvez éditer, compiler et exécuter cet exemple, vous êtes sur la bonne voie.

Utilisation d'une variable pour stocker des données

Parfois, dans un programme, nous voulons stocker des informations quelque part, dans la mémoire, puis l'utiliser plus tard. Pour stocker quelque chose dans la mémoire, nous utilisons une variable. Toutes les variables dans FreeBasic sont d'un type spécifique, comme un nombre ou une chaîne. Nous utilisons l'instruction Dim pour déclarer un nom de variable et préciser quel type d'information nous voulons y entreposer.

Dim text As String
text = "Hello World!"
Print text

Nous utilisons Dim pour faire savoir au compilateur que nous voulons utiliser une variable appelée text dans notre programme et que nous allons déposer dedans des données de type String (Chaîne). Ensuite nous attribuons (copie) "Hello World!" dans la variable. Enfin, nous utilisons Print pour l'afficher à l'écran.

Utilisation d'une variable dans une expression

Une expression est un terme générique pour décrire une partie du code source qui peut être évaluée. Après qu'une expression ait été évaluée, nous pouvons alors faire quelque chose avec elle, comme l'assigner (copier) à une variable.

Dim a As String, b As String, text As String
a = "Hello"
b = "World"
text = a + " " + b + "!"
Print text

Nous affectons certaines données aux variables a et b. Puis nous utilisons les variables a et b dans une qui est affectée à text. Enfin, nous affichons le résultat à l'écran.

Saisir des données de la part de l'utilisateur

Souvent, nous n'avons aucune idée des données nécessaires pour le programme car c'est l'utilisateur qui doit les fournir. Nous ne pouvons les mettre dans notre code source du programme puisque nous ne les connaîtrons que lorsque l'utilisateur utilisera le programme et nous dira ce qu'il en est.

Dim answer As String
Input "Tapez quelque chose et appuyez enter:", answer
Print "Vous avez tape: '"; answer; "'"

Tout d'abord, l'instruction Input affichera certaines informations à l'écran, puis attendra que l'utilisateur fournisse certaines données au programme. Dans cet exemple, nous afficherons à l'écran exactement ce que l'utilisateur aura tapé.


Faire des maths

Variables et expressions ne sont pas seulement limités à des chaînes. La plupart des premiers langages ne géraient pas très bien les chaînes. L'écriture d'expressions mathématiques est semblable à la façon dont elles peuvent être écrites avec un crayon et du papier.

Dim a As Integer, b As Integer, c As Integer

a = 5
b = 7
c = a + b

Print "a = "; a
Print "a = "; b
Print "a + b = "; c

Nous attribuons des valeurs aux variables a, b et c. Nous utilisons le type Integer (Entier) pour les variables. Un entier ne peut être positif ou négatif, mais il n'y a pas de fractions.

Faire des maths avec saisie de données

Ceci est similaire à l'exemple précédent, Ceci est similaire à l'exemple précédent, sauf que nous laissons l'utilisateur choisir les nombres que nous allons additionner.

Dim a As Integer, b As Integer, r As Integer
Input "Entrez un nombre:", a
Input "Entrez un autre nombre:", b

r = a + b
Print "La somme des nombres est "; r

Dim permet au compilateur de savoir quels noms de variables nous voulons utiliser et qu'elles vont contenir des données de type Integer. Nous utilisons Input pour que l'utilisateur donne les valeurs des nombres et Print pour afficher le résultat.

Faire plus de maths avec saisie de données

Les variables numériques ne sont pas limitées seulement à des nombres entiers. Nous pouvons également utiliser des types de données Single ou Double précision qui peuvent représenter des fractions. Dans cet exemple nous saisirons des entrées de l'utilisateur correspondant à un poids en livres pour une conversion en kilogrammes.

Dim lb As Single, kg As Single
Input "Entrez un poids en pounds:", lb

kg = lb * 0.454
Print lb; " lb. est egal a "; kg; " kg"


Instructions de répétition

En utilisant l'instruction For...Next nous pouvons dire au programme de faire quelque chose en le répétant un certain nombre de fois. Par exemple, disons que nous voulons ajouter tous les nombres de 1 à 100.

Dim total As Integer
Dim number As Integer
total = 0
For number = 1 To 100
  total = total + number
Next
Print "La somme des nombres de 1 a 100 est "; total



Prendre une décision

Un programme peut choisir les instructions à exécuter en utilisant une instruction conditionnelle comme If...Then. Nous pouvons utiliser les valeurs d'une variable ou les résultats d'une expression pour décider si nous devrions ou ne devrions pas, exécuter une ou plusieurs instructions.

Dim number As Integer
Input "Entrez un nombre : ", number
Print "Votre nombre est ";
If number < 0 Then
  Print "negative"
ElseIf number > 0 Then
  Print "positive"
Else
  Print "zero"
End If


Après la saisie d'un nombre par l'utilisateur, nous allons afficher un mot (positive, négative ou zéro) en se basant sur l'instruction qui correspond à la condition.

Instructions de répétition (encore)

Ici, nous allons utiliser une autre structure de boucle Do...Loop pour répéter certaines instructions. Comment le programme sait-il qu'il doit cesser de répéter les instructions? Nous utiliserons If...Then pour prendre la décision de sortir de la boucle.

Dim total As Single, count As Single, number As Single
Dim text As String

Print "Ce programme calculera calculate la somme et la moyenne pour une"
Print "liste de nombres.  Entrez une valeur vide pour finir."
Print

Do
  Input "Entrez un nombre : ", text
  If text = "" Then
    Exit Do
  End If

  count = count + 1
  total = total + Val(text)

Loop

Print
Print "Vous avez entre "; count; " numbers"
Print "La somme est "; total
If count <> 0 Then
  Print "La moyenne est "; total / count
End If
