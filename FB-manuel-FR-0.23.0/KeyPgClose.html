Close
 
Fonction flux Entrée/Sortie pour terminer l'accès à un périphérique

Syntaxe

Close [[#]filenum ] [, [#]filenum ...]
ou
result = Close( [#filenum] )

Paramètres

filenum
Liste des numéros de fichiers à fermer.

Description

Ferme les fichiers dont les numéros sont passés en arguments. Si un numéro inutilisé est passé, Close renvoie une erreur.

Close sans arguments ferme tous les fichiers ouverts.

Terminer un programme en utilisant End fermera automatiquement tous les fichiers.

Valeur retournée

Close retourne zéro (0) si succès et sinon un code d'erreur différent de zéro.

Exemple

' Create a string and fill it.
Dim buffer As String, f As Integer

buffer = "Hello World within a file."

' Find the first free file number.
f = FreeFile

' Open the file "file.ext" for binary usage, using the number "f".
Open "file.ext" For Binary As #f

  ' Place our string inside the file, using number "f".
  Put #f, , buffer

' Close the file.  We could also do 'Close #f', but it's only necessary if more than one number is open.
Close

' End the program. (Check the file "file.ext" upon running to see the output.)
End
