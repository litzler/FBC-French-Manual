Int
 
Renvoie le plus grand entier qui est inférieur ou égal à un nombre

Syntaxe

Declare Function Int ( ByVal number As Double ) As Double

Usage

result = Int( number )

Paramètres

number
le nombre à virgule flottante à arrondir

Return Value

Retourne le plus grand entier qui est inférieur ou égal à number.

Description

Int retourne le plus grand entier qui est inférieur ou égal à number. Par exemple, Int(4.9) retournera 4.0 et Int(-1.3) retournera -2.0.

L'Operator unaire Int peut être surchargé par un type défini par l'utilisateur.

Exemple

Print Int(1.9)  '' will print  1
Print Int(-1.9) '' will print -2 


Différences de dialecte

  • Dans le dialecte -lang qb, cet opérateur ne peut pas être surchargé.

Différences avec QB

  • Aucune
