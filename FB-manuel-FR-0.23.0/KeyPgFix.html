Fix
 
Retourne la partie entière d'un nombre, arrondi vers zéro

Syntaxe

Declare Function Fix ( ByVal number As Double ) As Double

Usage

result = Fix( number )

Paramètres

number
le nombre à virgule flottante à tronquer

Valeur retournée

Retourne la partie entière de number, arrondi vers zéro.

Description

Equivalent à: Sgn(number) * Int(Abs(number)). Par exemple, Fix(1.3) retournera 1.0, et Fix(-4.9) retournera -4.0.

Note: cette fonction est également équivalente à number - Frac(number).

L'Operator unaire Fix peut être surchargé avec des types définis par l'utilisateur.

Exemple

Print Fix(1.9)  '' will print  1
Print Fix(-1.9) '' will print -1 


Différences de dialecte

  • Dans le dialecte -lang qb, cet opérateur ne peut pas être surchargé.

Différences avec QB

  • Aucune
