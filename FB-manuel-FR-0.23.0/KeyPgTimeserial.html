TimeSerial
 
Forme une Date Serial avec les heures, minutes et secondes spécifiées

Syntaxe

Declare Function TimeSerial ( ByVal hour As Integer, ByVal minute As Integer, ByVal second As Integer ) As Double

Usage

#include "vbcompat.bi"
result = TimeSerial( hours, minutes, seconds )

Paramètres

hours
nombre d'heures, de 0 à 23
minute
nombre de minutes
second
nombre de secondes

Valeur retournée

Retourne une date serial contenant l'heure formée avec les valeurs des paramètres hours, minutes et seconds. La "date serial" retournée n'a pas de partie entière.

Description


hours doit avoir une valeur de 0 à 23

Le compilateur ne reconnaît cette fonction que si vbcompat.bi ou datetime.bi est inclus.

Exemple

#include "vbcompat.bi"

Dim ds As Double = DateSerial(2005, 11, 28) + TimeSerial(7, 30, 50)

Print Format(ds, "yyyy/mm/dd hh:mm:ss")


Différences avec QB

  • N'existe pas dans QB. Cette fonction est apparue dans PDS et VBDOS
