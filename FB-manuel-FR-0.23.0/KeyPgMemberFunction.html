Function (Membre)
 
Déclare ou définit une procédure membre qui renvoie une valeur.

Syntaxe

{ Type | Class | Union } typename
Declare [ Static | Const ] Function fieldname [calling convention specifier] [ Alias external_name ] ( [ parameters ] ) As datatype [ Static ]
End { Type | Class | Union }

Function typename.fieldname ( [ parameters ] ) As datatype
statements
End Function

Paramètres

typename
nom de Type, Class ou Union
fieldname
nom de la procédure
external_name
nom de champ vu quand on établit des liens externes
parameters
les paramètres à passer à la procédure
calling convention specifier
peut être: cdecl, stdcall ou pascal

Description

Les membres de Function sont accessibles avec Operator . (Accès de membre) ou Operator -> (Pointeur vers Accès de membre) pour appeler une procédure membre qui retourne une valeur. La procédure peut éventuellement accepter des paramètres ByVal ou ByRef. typename peut être surchargé, sans utilisation explicite du mot-clé Overload.

typename est le nom du type avec lequel la Function est déclarée et définie. La résolution de noms pour typename suit les mêmes règles que pour les procédures en cas d'utilisation dans un Namespace = Espace de noms.

Un paramètre caché This ayant le même type que typename est passé aux procédures membres non statiques. This est utilisé pour accéder aux champs de Type, Class ou Union.

Un Static (Membre) peut être déclaré en utilisant le spécificateur Static. A Const (Membre) peut être déclaré en utilisant le spécificateur Const.

Exemple

#include "vbcompat.bi"

Type Date

  value As Double

  Declare Static Function Today() As Date

  Declare Function Year() As Integer
  Declare Function Month() As Integer
  Declare Function Day() As Integer

End Type

Function Date.Today() As Date
  Return Type(Now())
End Function

Function Date.Year() As Integer
  Return ..Year(value)
End Function

Function Date.Month() As Integer
  Return ..Month(value)
End Function

Function Date.Day() As Integer
  Return ..Day(value)
End Function

Dim d As Date = Date.Today

Print "Year  = "; d.Year
Print "Month = "; d.Month
Print "Day   = "; d.Day



Différences de dialecte

  • Disponible seulement dans le dialecte -lang fb.
