Erfn
 
Signaler une erreur survenue dans une fonction

Syntaxe

Declare Function Erfn ( ) As ZString Ptr

Usage

result = Erfn ( )

Valeur retourné

Retourne un pointeur vers la chaîne de caractères identifiant la fonction où l'erreur s'est produite.

Retourne NULL si la source n'est pas compilé avec l'option de compilation -exx.
Description

Une fonction de rapport d'erreur retournant un pointeur sur le nom de la fonction.

Exemple

'' test.bas
'' compile with fbc -exx -lang fblite test.bas

#lang "fblite"

Sub Generate_Error
  On Error Goto Handler
  Error 1000
  Exit Sub
Handler:
  Print "Error Function: "; *Erfn()
  Print "Error Module  : "; *Ermn()
  Resume Next
End Sub

Generate_Error

Affichage:
Error Function: GENERATE_ERROR
Error Module  : test.bas


Différences de dialecte

  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __Erfn.
