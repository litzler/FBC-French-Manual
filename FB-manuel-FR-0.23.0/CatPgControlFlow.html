Déclarations de contrôle de flux
 
Les déclarations qui dirigent le flux d'exécution du programme.

Déclarations de transfert
Déclarations qui donnent le contrôle à une autre partie du programme.
Déclarations de branchement
Déclarations qui exécutent une parmi plusieurs possibilités de branchement.
Déclarations de boucle
Déclarations qui exécutent du code avec répétition.

Déclarations de transfert
Goto
Transfère l'exécution à un autre point dans le code, défini par une étiquette de texte.
GoSub
Transfère temporairement l'exécution à un autre point dans le code, défini par une étiquette de texte.
On Goto
Transfert de l'exécution, basé sur la valeur d'un expression, à un parmi un certain nombre de points du code définis par des étiquettes.
On Gosub
Transfert temporaire de l'exécution, basé sur la valeur d'un expression, à un parmi un certain nombre de points du code définis par des étiquettes.
Return
Retourne à partir d'un appel à l'aide de GoSub ou d'une procédure retournant une valeur.

Déclarations de branchement
If..End If
Exécute un bloc d'instructions si une condition est remplie.
..Else If..
Exécute un bloc de code si une condition est remplie et que toutes les conditions précédentes ne sont pas remplies..
..Else..
Exécute un bloc de code si toutes les conditions précédentes ne sont pas remplies.
Select..End Select
Exécute une parmi un certain nombre d'instructions en utilisant un ensemble de conditions.
..Case..
Exécute un bloc de code si une condition est remplie.
..Case Else..
Exécute un bloc de code si toutes les conditions précédentes ne sont pas remplies.

Contrôle intra-branche
Exit Select
Sort prématurément d'une déclaration Select..End Select.
Déclarations de boucle
While..Wend
Exécute un bloc d'instructions tant qu'une condition est remplie.
For..Next
Exécute un bloc d'instructions tant qu'un itérateur est inférieur ou supérieur à une expression.
Do..Loop
Exécute un bloc d'instructions tant qu'/jusqu'à ce qu'une condition est/soit remplie.

Contrôle intra-boucle
Continue While, Continue For et Continue Do
Re-rentre prématurément dans une boucle.
Exit While, Exit For et Exit Do
Sort prématurément d'une boucle.
