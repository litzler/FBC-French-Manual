Index des en-têtes de bibliothèques
 
Index des fichiers d'en-tête distribués avec FreeBasic pour une utilisation avec des bibliothèques externes. Seules les en-têtes (fichiers include) sont distribuées avec FreeBasic. Pour obtenir une bibliothèque externe nécessaire, veuillez consulter le lien indiqué ci-dessous sur l'information pour chaque bibliothèque.

Graphiques
allegro - Bibliothèque de programmation de jeux.
DUGL - Bibliothèque graphique et jeu pour DOS.
caca - Une bibliothèque ASCII d'art couleur.
cairo - Bibliothèque graphique 2D avec le support de plusieurs périphériques de sortie.
cgui - Bibliothèque pour faire des interfaces graphiques de manière simple.
Curses - Bibliothèque console normalisée d'interface utilisateur.
dislin - Bibliothèque de sous-routines et de fonctions qui affichent les données sous forme graphique.
FreeImage - Bibliothèque Open Source pour la prise en charge de formats graphiques populaires.
Freetype2 - Un moteur de polices libre, de haute qualité et portable.
gd - Bibliothèque "code source libre" pour la création dynamique d'images par les programmeurs.
gif_lib - GIFLIB est un ensemble d'outils portables et une bibliothèque de routines pour travailler avec des images GIF.
grx - Bibliothèque graphique 2D.
GTK+ - Bibliothèque graphique multiplate-forme, interface utilisateur.
IL - Une Bibliothèque image complète multi-plateforme.
IUP - Boîte à outils portable pour créer des interfaces utilisateur.
japi - Boîte à outils GUI "code source libre" utilisant la boîte à outils Java AWT.
jpeglib - Bibliothèque multiplate-forme pour la lecture/écriture des images jpeg.
jpgalleg - JPGalleg est un petit module complémentaire qui ajoute capacités de traitement des images JPG à la bibliothèque Allegro.
libpng - Permet la lecture et l'écriture des images PNG.
OpenGL - Bibliothèque graphique 3D multiplate-forme.
pdflib - PDFlib est un outil de développement qui permet à votre logiciel l'utilisation de PDF.
SDL - Bibliothèque multiplate-forme multimédia.
tinyptc - Une petite bibliothèque graphique permettant l'utilisation facile du tampon vidéo.
wx-c - Bibliothèque graphique d'interface utilisateur multiplate-forme.

Base de données
mysql - Moteur de base de données de haute qualité couramment utilisé.
postgresql - Logiciel libre de système de gestion de base de données, objet-relationnel.
sqlite - Petite bibliothèque C qui implémente un moteur de base de données SQL interne, embarquable, sans configuration.

Musique / son
BASS - Bibliothèque audio pour une utilisation sous Windows avec une version bêta pour Linux.
BASSMOD - BASSMOD est seulement un MOD (XM, IT, S3M, MOD, MTM, UMX) version de BASS
FMOD - Bibliothèque audio prenant en charge à peu près n'importe quel format.
OpenAL - API audio 3D multiplate-forme.

Aide au développement
bfd - Permet aux applications d'utiliser les mêmes routines pour fonctionner sur des fichiers objets quel que soit le format de fichier objet.
CUnit - Système léger pour l'écriture, l'administration et l'exécution des tests unitaires en C.
gdsl -"Generic Data Structures Library" est une collection de routines pour les structures de données génériques.
GNU ASpell - Vérificateur orthographique libre.

Langages intégrables
jni - L'interface de programmation standard pour l'écriture de méthodes Java natif et l'incorporation de la machine virtuelle Java * dans les applications natives.
Lua - Moteur de script léger embarquable utilisant le langage Lua.
spidermonkey - Moteur javascript intégrable.

Cryptographie
cryptlib - Une trousse d'outils de sécurité puissante qui permet même aux programmeurs inexpérimentés crypto d'ajouter facilement des services de chiffrement et d'authentification à leurs logiciels.

Mathématiques
big_int - Bibliothèque pour utiliser des entiers de taille arbitraire.
gmp - Bibliothèque libre pour l'arithmétique de précision arbitraire, opérant sur des entiers signés, des nombres rationnels et nombres à virgule flottante.
gsl - Fournit un large éventail de routines mathématiques tels que les générateurs de nombres aléatoires, des fonctions spéciales et des "des moindres carrés".
Newton - Une solution intégrée pour la simulation en temps réel des environnements physiques.
ode - Bibliothèque "code source libre" haute performance pour la simulation dynamique des corps rigides.

Réseaux
cgi-util - Petite bibliothèque C pour créer des programmes CGI pour site web.
curl - Bibliothèque de transfert URL côté client libre et facile à utiliser supportant presque tous les protocoles.
FastCGI - Extension libre de CGI qui fournit des performances élevées sans les limitations d'API serveur spécifique.

eXtensible Markup Language (XML)
expat - Bibliothèque flux orientée analyseur XML avec plusieurs fonctionnalités utiles.
libxml - Bibliothèque standard d'accès aux de fichiers xml.
libxslt - XSLT est lui-même un langage XML pour définir des transformations XML.
mxml - Petite bibliothèque d'analyse XML que vous pouvez utiliser pour lire les données XML et les fichiers de données XML dans votre application.

Expressions régulières
pcre - Filtrage d'expression régulières utilisant la même syntaxe que Perl.
tre - Bibliothèque légère, robuste, efficace et conforme aux expressions régulières POSIX.

Compression
LZO - Compression rapide et décompression très rapide.
QuickLZ - Bibliothèque de compression très rapide
zlib - Bibliothèque de compression sans perte de données non handicapée par des brevets.

C Runtime Library
DOS API
disphelper - Bibliothèque de d'assistance pour utiliser des objets COM en pur langage C.
