MKD
 
Copie binaire d'une variable Double précision en une String (chaîne), réglage de sa longueur à 8 octets

Syntaxe

Declare Function MKD ( ByVal number As Double ) As String

Usage

result = MKD[$]( number )

Paramètres

number
Une variable Double précision dont on fait une copie binaire vers une String (chaîne).

Valeur retournée

Retourne une String (chaîne) copie binaire d'un nombre Double précision.

Description

Exécute une copie binaire d'une variable DOUBLE précision vers une chaîne dont la longueur est de 8 octets. La chaîne résultat peut être lue à nouveau comme un DOUBLE par CVD

Cette fonction est utile pour écrire une valeur numérique dans un tampon sans utiliser de définition de TYPE.

Exemple

Dim n As Double, e As String
n = 1.2345
e = MKD(n)
Print n, CVD(e)


Différences de dialectes

  • Le suffixe de type de chaîne "$" est obligatoire dans le dialecte -lang qb.
  • Le suffixe de type de chaîne "$" est facultatif dans les dialectes -lang fblite et -lang fb.

Différences avec QB

  • Aucune
