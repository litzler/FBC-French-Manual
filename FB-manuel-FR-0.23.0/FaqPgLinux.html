Questions sur Linux et FreeBASIC


FreeBASIC me donne une erreur 'ld: can't find -lX11' ou quelque chose de similaire!
FreeBASIC utilise "ld" pour lier les fichier sous Linux. Ce programme exige que toute les bibliothèques que vous utilisez aient leur version de développement d'installée (-dev). Par exemple, pour le message d'erreur ci-dessus, vous devez installer "xlib-dev" de votre distribution. Les autres erreurs courantes sont pour "glibc" qui nécessite "glibc-dev" et "sdl" qui nécessite "sdl-dev". Vous les trouverez facilement sur le média d'installation de la plupart des distributions.
