Opérateur . (Accès Membre)
 
Renvoie une référence à un membre d'une référence à un objet

Syntaxe

Declare Operator . ( ByRef lhs As T ) As U

Notez que Opérateur . (Accès membre) renvoie une référence. En date du 27/02/07, la syntaxe FreeBASIC ne supporte pas le retour de références.

Usage

result = lhs . rhs

Paramètres

lhs
Un objet.
T
Un type défini par l'utilisateur.
rhs
Le nom du membre auquel on veut accéder.
U
Le type auquel rhs se réfère.

Valeur retournée

Renvoie une référence au membre spécifié par rhs.

Description

Opérateur . (Accès membre) renvoie une référence à un membre d'un objet.

Opérateur . (Accès membre) peut également être utilisé pour accéder aux membres d'un objet implicite dans un bloc With..End With.

Exemple


Type T
    As Integer a, b
End Type

Dim x As T

'' Access the member 'a' of x.
x.a = 10

'' Access the member 'b' of x.
With x
    .b = 20
End With


Différences de dialectes

  • Dans le dialecte -lang qb, cet opérateur ne peut pas être surchargé.

Différences avec QB

  • Aucune
