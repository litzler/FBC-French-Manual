DefShort
 
Spécifie un type de données par défaut pour une série de noms de variables

Syntaxe

DefShort start_letter[-end_letter ][, ...]

Paramètres

start_letter
la première lettre dans la série
end_letter
la dernière lettre dans la série

Description

DefShort précise que les variables et les tableaux qui ne sont pas déclarés avec un type de données - ou non déclarés entièrement - sont implicitement déclarés de type Short si la première lettre de leur nom correspond à une certaine lettre ou se trouve dans une plage inclusive de lettres.

Exemple

Ceci fera de sNumber un nombre Short car sa première lettre est s.
'' Compile with -lang fblite or qb

#lang "fblite"

DefShort s
Dim sNumber


Différences de dialecte

  • Disponible dans le dialecte -lang fblite.
  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __Defshort.

Différences avec QB

  • Nouveau pour FreeBASIC
  • Dans QBasic, pour que les variables aient par défaut 2 octets entiers, utilisez DEFINT.
