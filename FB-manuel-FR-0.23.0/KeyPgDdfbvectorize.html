__Fb_Vectorize__
 
Définition intrinsèque fixée par le compilateur

Syntaxe

__FB_VECTORIZE__

Description

Défini comme le numéro de niveau de vectorisation fixé par l'option -vec de la ligne de commande.

Exemple

#if __FB_VECTORIZE__ = 2
  ' ... instructions only for vectorization level 2...
#else
  ' ...
#endif
