CShort
 
Convertit une expression numérique ou une chaîne en nombre entier (Short)

Syntaxe

Declare Function CShort ( ByVal expression As datatype ) As Short

Type typename
Declare Operator Cast ( ) As Short
End Type

Usage

result = CShort( numeric expression )
result = CShort( string expression )
result = CShort( user defined type )

Paramètres

expression
Une expression numérique, chaîne ou pointeur à transformer (cast) en valeur Short
datatype
Tout type de données numérique, chaîne ou pointeur.
typename
Un type défini par l'utilisateur.

Valeur retournée

Une valeur Short.

Description

La fonction CShort arrondit la partie entière et retourne une valeur Short 16 bits. La fonction ne vérifie pas s'il y a un dépassement, alors assurez-vous de ne pas avoir une valeur inférieure à -32768 ou supérieure à 32767. Le nom vient de 'Convert to Short'.

Si l'argument est une expression de chaîne, il est converti en numérique à l'aide de ValInt.

Exemple

' Using the CSHORT function to convert a numeric value

'Create an SHORT variable
Dim numeric_value As Short

'Convert a numeric value
numeric_value = CShort(-4500.66)

'Print the result, should return -4501
Print numeric_value
Sleep
