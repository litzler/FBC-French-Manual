UInteger
 
Type de données standard: nombre de 32 bits non-signé

Syntaxe

Dim variable As UInteger

Description

Nombre entier de 32 bits non-signé. Peut contenir des valeurs de 0 à 4294967295.

Exemple

  Dim x As UInteger = 0
  Dim y As UInteger = &HFFFFFFFF
  Print "UInteger Range = "; x; " to "; y

Affichage:
UInteger Range = 0 to 4294967295


Différences de dialecte

  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __Uinteger.

Différences avec QB
