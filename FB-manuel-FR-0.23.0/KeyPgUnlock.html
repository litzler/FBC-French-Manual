Unlock
 
Supprime une restriction d'accès précédente (lock) sur un fichier

Syntaxe

Unlock #filenum, record
Unlock #filenum, start To end

Paramètres

filenum
Le numéro de fichier utilisé pour Open (=ouvrir) le fichier.
record
L'enregistrement (fichiers en mode Random) à déverrouiller.
start
La première position, dans une plage de positions, en octets (fichiers en mode Binary) à déverrouiller.
end
La dernière position, dans une plage de positions, en octets (fichiers en mode Binary ) à déverrouiller.

Description

Unlock supprime la restriction d'accès temporaire mise en place par Lock.

Il est fortement recommandé d'utiliser les mêmes arguments que ceux utilisés dans le précédent Lock.

Note: Cette commande ne fonctionne pas toujours, ni comme dans la documentation, ni comme espéré. Elle apparaît comme "cassée" pour le moment.

Exemple

Pour un exemple voir Lock.

Différences avec QB

  • Actuellement, FB ne peut pas implicitement déverrouiller l'ensemble des fichiers
  • En mode Random, FB ne peut pas déverrouiller un ensemble d'enregistrements
