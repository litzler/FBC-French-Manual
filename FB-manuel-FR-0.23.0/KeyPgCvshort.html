CVShort
 
Conversion d'une chaîne de 2 octets en une variable Short

Syntaxe

Declare Function CVShort ( ByRef str As Const String ) As Short

Usage

result = CVShort( str )

Paramètres

str
Une chaîne (String) ayant exactement 2 octets de longueur, copie d'une variable Short.

Valeur retournée

Retourne une variable Short qui contient la copie binaire d'un Short.

Description

Effectue une copie d'une chaîne-2-octets vers une variable Short (court). Le résultat n'aura un sens que si la chaîne contient une valeur Short correctement formatée, comme celles produites par MKShort.

Cette fonction est utile pour lire des valeurs à partir de tampons sans utiliser une définition de Type.

Exemple

Dim a As Short, b As String
a=4534
b=MKShort(a)
Print a, CVShort(b)
Sleep

Différences de dialectes

  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __Cvshort.
