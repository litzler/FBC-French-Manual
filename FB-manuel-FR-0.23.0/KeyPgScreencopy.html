ScreenCopy
 
Copie le contenu d'une page graphique dans une autre page graphique

Syntaxe

Declare Sub ScreenCopy ( ByVal frompage As Integer = -1, ByVal topage As Integer = -1 )

Usage

ScreenCopy [ from_page ] [, to_page ]

Paramètres

frompage
page à copier
topage
page vers laquelle copier

Description

from_page est la page que l'on copie. Si cet argument est omis, la page de travail courante est utilisée.

to_page est la page vers laquelle on copie. Si cet argument est omis, la page visible courante est utilisée.

Les numéros de page varient de 0 à num_pages spécifié lors du réglage du mode graphique moins 1

Vous pouvez utiliser cette fonction pour ajouter un double tampon à vos graphiques. Tous les modes graphiques supportent cette fonction; vous devez fournir des numéros de page valides sinon ScreenCopy n'aura aucun effet.

Il existe deux autres fonctions similaires: Flip et PCopy. Flip est conçu pour fonctionner en mode OpenGL, alors que PCopy supporte les pages console sur certaines plates-formes. Les deux font la mÃªme chose que ScreenCopy dans les modes graphiques normaux.

Exemple

'' 320x200x8, avec 3 pages
Screen 13,,3

'' image pour page #1
ScreenSet 1, 0
Cls
Circle( 160, 100 ), 90, 1 ,,,, f
Circle( 160, 100 ), 90, 15
Print "Press 2 to copy page #2 to visible page"
Print "Press escape to exit"

'' image pour page #2
ScreenSet 2, 0
Cls
Line( 50, 50 )-( 270, 150 ), 2, bf
Line( 50, 50 )-( 270, 150 ), 15, b
Print "Press 1 to copy page #1 to visible page"
Print "Press escape to exit"

'' page #0 est la page visible
ScreenSet 0, 0
Cls
Print "Press 1 to copy page #1 to visible page"
Print "Press 2 to copy page #2 to visible page"
Print "Press escape to exit"

Dim k As String

Do
  k = Inkey
  Select Case k
  Case Chr(27)
    Exit Do
  Case "1"
    ScreenCopy 1, 0
  Case "2"
    ScreenCopy 2, 0
  End Select

  Sleep 25
Loop



Différences de dialecte

  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __Screencopy.

Différences avec QB

  • Nouveau pour FreeBASIC. QB a PCopy comme fonction équivalente.
