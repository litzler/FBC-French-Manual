DefSng
 
Spécifie un type de données par défaut pour une série de noms de variables

Syntaxe

DefSng start_letter[-end_letter ][, ...]

Paramètres

start_letter
la première lettre dans la série
end_letter
la dernière lettre dans la série

Description

DefSng précise que les variables et les tableaux qui ne sont pas déclarés avec un type de données - ou non déclarés entièrement - sont implicitement déclarés de type Single si la première lettre de leur nom correspond à une certaine lettre ou se trouve dans une plage inclusive de lettres.

Exemple

Ceci fera de sNumber et yNumber un nombre décimal Single-précision car sa première lettre est dans la plage s-z.
'' Compile with -lang fblite or qb

#lang "fblite"

DefSng s-z
Dim sNumber, yNumber


Différences de dialecte

  • Disponible seulement dans les dialectes -lang qb et -lang fblite.

Différences avec QB

  • Aucune
