h Compiler Option: -d
Option du compilateur: -d
 
Ajouter une définition préprocesseur

Syntaxe

-d < name=value >

Paramètres

name
Nom de la macro du préprocesseur à définir.
value
Valeur à donner à la macro.

Description

L'option -d du compilateur ajoute une macro préprocesseur à tous les fichiers source. Identique à l'utilisation de la directive préprocesseur #define ou #macro.
