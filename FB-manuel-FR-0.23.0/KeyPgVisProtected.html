Protected: (Contrôle d'accès)
 
Spécifie le contrôle d'accès protégé (protected) à un membre de Type ou Class

Syntaxe

Type typename
Protected:
member declarations
End Type

Paramètres

typename
nom de Type ou Class
member declarations
déclarations pour les champs, fonctions ou énumérations

Description

Protected: indique que les member declarations qui suivent ont un accès protégé. Les membres protégés ne sont accessibles qu'à l'intérieur d'une fonction membre pour Type ou Class et les classes dérivées de Type ou Class.

member declarations qui suivent Protected: sont protégés jusqu'à ce qu'un spécificateur de contrôle d'accès différent ne soit donné, comme Private: ou Public:.

Les membres d'une déclaration Type sont Public: par défaut si aucun spécificateur de contrôle d'accès n'est donné.

NOTE: Ce mot-clef est supporté dans le dernier compilateur fbc, cependant, son utilité est limitée puisque les classes et l'héritage ne sont pas encore supportés.
