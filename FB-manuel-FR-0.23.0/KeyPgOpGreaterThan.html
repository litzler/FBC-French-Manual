Opérateur > (Supérieur à)
 
Compare une expression à une autre pour savoir si elle lui est supérieure

Syntaxe

Declare Operator > ( ByRef lhs As T, ByRef rhs As T ) As Integer

Usage

result = lhs > rhs

Paramètres

lhs
L'expression côté gauche à comparer.
rhs
L'expression côté droit à comparer.
T
N'importe quel type de pointeur.

Valeur retournée

Retourne une valeur négative (-1) si l'expression côté gauche est supérieure à celle de droite ou zéro (0) si elle est inférieure ou égale.

Description

L'Opérateur > (Supérieur à) est un opérateur binaire qui compare une expression à une autre pour savoir si elle est plus grande et retourne un résultat - une valeur boléenne sous la forme d'un Entier (Integer): negatif (-1) si c'est vrai et zéro (0) si c'est faux. Les arguments ne sont en aucun cas modifiés.

Cet opérateur peut être surchargé pour accepter aussi bien des types définis par l'utilisateur.

Exemple

L'Opérateur <= (Inférieur ou égal) est complémentaire à l'Opérateur > (Supérieur à) et est fonctionnellement identique si combiné avec l'Opérateur Not (Bit-wise Complement).

   If (420 > 69) Then Print "(420 > 69) is true."
   If Not (420 <= 69) Then Print "not (420 <= 69) is true."


Différences de dialectes

  • Dans le dialecte -lang qb, cet opérateur ne peut pas être surchargé.

Différences avec QB

  • Aucune
