Option du compilateur: -g
 
Ajouter les informations de débogage

Syntaxe

-g

Description

L'option du compilateur -g insère des symboles de débogage dans les fichiers produits, pour les utiliser avec les débogueurs compatibles avec GDB.

La macro intrinsèque __FB_DEBUG__ est fixée à non-zéro (-1) si l'option a été spécifiée, et mise à zéro (0) autrement.
