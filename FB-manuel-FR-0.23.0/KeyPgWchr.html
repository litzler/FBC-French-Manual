WChr
 
Retourne une chaîne de larges caractères contenant un ou plusieurs caractères Unicode

Syntaxe

Declare Function Wchr ( ByVal ch As Integer [, ... ] ) As WString

Usage

result = WChr( ch0 [, ch1 ... chN ] )

Paramètres

ch
La valeur entière Unicode d'un caractère.

Valeur retournée

Retourne une chaîne de larges caractères.

Description

WChr retourne une chaîne de larges caractères contenant le(s) caractère(s) représenté(s) par la(les) valeur(s) Unicode qui lui sont passée(s).

Quand WChr est utilisé avec des constantes numériques ou littérales, le résultat est évalué lors de la compilation, de sorte qu'il peut être utilisé dans des initialiseurs de variable.

Tous les caractères Unicode ne sont pas forcément affichables sur n'importe quelle machine, les caractères accessibles dépendent de la police de caractères utilisée sur la console. Les modes graphiques ne peuvent pas afficher les caractères Unicode, car la police intégrée GfxLib n'est pas Unicode.

Exemple

Print "the character represented by the UNICODE code of 934 is:"; WChr(934)
Print "multiple UNICODE characters:"; WChr(933, 934, 935)

produira la sortie:

Φ
ΥΦΧ

Différences de plate-forme

  • DOS ne supporte pas WChr.


Différences de dialecte

  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __Wchr.

Différences avec QB

  • Nouveau pour Freebasic
