Environ
 
Renvoie la valeur d'une variable d'environnement système

Syntaxe

Declare Function Environ ( ByRef varname As Const String ) As String

Usage

result = Environ( varname )

Paramètres

varname
Le nom d'une variable d'environnement.

Valeur retournée

Retourne la valeur du texte de la variable d'environnement ou la chaîne vide ("") si la variable n'existe pas.

Description

Environ retourne la valeur texte d'une variable d'environnement système.

Exemple

'e.g. to show the system variable "path":

Print Environ("path")


Différences avec QB

  • La déclaration Environ est maintenant appelée SetEnviron.
