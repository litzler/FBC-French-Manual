$Dynamic
 
Métacommande pour changer la façon dont les tableaux sont alloués

Syntaxe

'$Dynamic
ou
Rem $Dynamic

Description

'$Dynamic est une métacommande qui implique que toutes les déclarations de tableaux qui suivent, qu'elles aient été déclarées avec un intervalle d'indices constant ou non, sont de longueur variable. Ceci demeure en vigueur pour le reste du module dans lequel '$Dynamic est utilisé, et peut être modifié avec $Static.

Exemple

' compile with -lang fblite or qb

#lang "fblite"

'$DYNAMIC
Dim a(100)
'......
ReDim a(200)

Différences de dialectes

  • Disponible seulement dans les dialectes -lang fblite et -lang qb.

Différences avec QB

  • Lorsqu'il est utilisé dans les commentaires il doit être le premier symbole
