ValUInt
 
Conversion d'une chaîne en nombre entier non signé

Syntaxe

Declare Function ValUInt Overload ( ByRef strnum As Const String ) As UInteger
Declare Function ValUInt ( ByVal strnum As Const WString Ptr ) As UInteger

Usage

result = ValUInt ( strnum )

Param√®tres

strnum
la chaîne à convertir

Valeur retournée

Retourne un UInteger (entier non-signé) de la chaîne convertie

Si le premier caractère de la chaîne n'est pas valide, ValUInt retournera 0.

Description

Par exemple, ValUInt("10") retournera 10, et ValUInt("10.60") retournera 10. La fonction analyse la chaîne à partir de la gauche, en sautant tout espace blanc, et retourne le plus long nombre qu'elle peut lire, s'arrêtant au premier caractère non-approprié, qu'elle trouve. Tout caractère non-numérique, inclus les points décimaux et les spécificateurs d'exposant, sont considérés comme non-appropriés. Par exemple, ValUInt("23.1E+6") retournera seulement 23.

ValUInt peut convertir des nombres entiers en format Binaire / Octal / Hexadécimal, s'il y a devant l'identificateur pertinent ("&B" / "&O" / "&H"), par exemple: ValUInt("&HFF") retourne 255.

Si vous voulez convertir un nombre en une chaîne, utilisez la fonction Str.

Example

Dim a As String, b As UInteger
a = "20xa211"
b = ValUInt(a)
Print a, b


20xa211   20
