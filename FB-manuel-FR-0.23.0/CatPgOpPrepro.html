Opérateurs préprocesseur
 
Opérateurs exécutés par le préprocesseur

Ces opérateurs contrôlent comment le texte est interprété par le préprocesseur.

Opérateur # (Stringize)
Retourne le texte opérande converti en littéral String.
Opérateur ## (Concaténation)
Concatène deux opérandes texte.
Opérateur ! (Escaped String Literal)
Indique que le littéral chaîne immédiatement après doit être traité comme séquence d'échappement.
Opérateur $ (Non-Escaped String Literal)
Indique que le littéral chaîne immédiatement après ne doit pas être traité comme séquence d'échappement.
