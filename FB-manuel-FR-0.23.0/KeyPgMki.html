MKI
 
Copie binaire d'une variable Integer (entière) en une String (chaîne), réglage de sa longueur à 4 octets

Syntaxe

Declare Function MKI ( ByVal number As Integer ) As String

Usage

result = MKI[$]( number )

Paramètres

number
Une variable Integer (entière) dont on fait une copie binaire vers une String (chaîne).

Valeur retournée

Retourne une String (chaîne) copie binaire d'un nombre Integer (entier).

Description

Exécute une copie binaire d'une variable Integer (entière) vers une String (chaîne) dont la longueur est de 4 octets. La chaîne résultat peut être lue à nouveau comme un Integer (entier) par CVI

Cette fonction est utile pour écrire une valeur numérique dans un tampon sans utiliser de définition de Type.

Exemple

Dim a As Integer, b As String
a=4534
b=MKI(a)
Print a, CVI(b)
Sleep


Différences de dialectes

  • Dans le dialecte -lang qb, MKI retourne une chaîne de 2 octets, car un entier QB n'a que 16 bits.
  • Le suffixe de type de chaîne "$" est obligatoire dans le dialecte -lang qb.
  • Le suffixe de type de chaîne "$" est facultatif dans les dialectes -lang fblite et -lang fb.
