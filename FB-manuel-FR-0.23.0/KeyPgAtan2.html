Atan2
 
Renvoie l'arc tangente du rapport de deux composantes

Syntaxe

Declare Function Atan2 ( ByVal y As Double, ByVal x As Double ) As Double

Usage

result = ATan2( y, x )

Param√®tres

y
Composante verticale du rapport.
x
Composante horizontale du rapport.

Valeur retournée

L'angle dont la tangente est y/x, en radians, dans l'intervalle [-Pi .. Pi].

Description

ATan2 renvoie l'arc tangente du rapport y/x comme un Double dans l'intervalle de -Pi à Pi. L'arc tangente est l'inverse de la fonction Tan. L'angle renvoyé est mesuré en radians (non en degrés).

Exemple

Print Atan2 ( 4, 5 )     'this is the same as PRINT ATN ( 4 / 5 )


L'affichage sera:
0.6747409422235527
