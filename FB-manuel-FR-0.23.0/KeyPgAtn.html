Atn
 
Renvoie l'arc tangente d'un nombre

Syntaxe

Declare Function Atn ( ByVal number As Double ) As Double

Usage

result = Atn( number )

Paramètres

number
Un nombre.

Valeur retournée

L'angle, en radians, dont la tangente est number, dans l'intervalle [-Pi/2..Pi/2].

Description

Atn renvoie l'arc tangente de l'argument number comme un Double dans l'intervalle de -Pi/2 à Pi/2. L'arc tangente est l'inverse de la fonction Tan. L'angle renvoyé est mesuré en radians (non pas en degrés).

Exemple

Print "Pi ="; Atn ( 1.0 ) * 4
Print Atn ( 4 / 5 )


Affichera:
Pi = 3.141592653589793
0.6747409422235527
