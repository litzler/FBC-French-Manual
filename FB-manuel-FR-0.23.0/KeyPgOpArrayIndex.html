Opérateur () (Indice de tableau)
 
Retourne une référence à un élément d'un tableau

Syntaxe

Declare Operator () ( lhs() As T, ByRef rhs As Integer, ... ) As T

Notez que Opérateur () (Indice de tableau) retourne une référence. Voir FAQ du compilateur.

Usage

result = lhs ( rhs [, ...] )

Paramètres

lhs
Un tableau.
rhs
L'indice d'un élément du tableau.
T
N'importe quel type de donnée.

Description

Cet opérateur retourne une référence à un élément d'un tableau. Pour les tableaux multi-dimensionnels, des indices multiples seront spécifiés (au maximum égal au nombre total des dimensions du tableau).

Pour toute dimension d dans un tableau a, tout indice en dehors de la plage (LBound(a(), d), UBound(a(), d)) donnera une erreur de fonctionnement.

Exemple


Dim array(4) As Integer = { 0, 1, 2, 3, 4 }

For index As Integer = 0 To 4
    Print array(index)
Next

produira l'affichage:

	0 1 2 3 4

Différences avec QB

  • Aucune
