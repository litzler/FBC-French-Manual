Option du compilateur: -Wa
 
Passe les options de l'assembleur pour l'utilisation de l'émetteur d'assemblage (-gen gas), la valeur par défaut.

Syntaxe

-Wa < options >

Paramètres

options
Options supplémentaires à passer à l'assembleur.

Description

L'option du compilateur -Wa passe des options supplémentaires à GAS, l'assembleur. Les options doivent être séparées seulement par des virgules.
Par exemple:
fbc -Wa -o,output.o,--verbose
