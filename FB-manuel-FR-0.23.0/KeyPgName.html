Name
 
Renomme un fichier sur le disque

Syntaxe

Declare Function Name( ByRef oldname As Const String, ByRef newname As Const String ) As Integer

Usage

result = Name( oldname, newname )

Paramètres

oldname
Nom d'un fichier existant.
newname
Nouveau nom du fichier.

Valeur retournée

Retourne zéro (0) si succès et non-zéro en cas d'échec.

Description

Renomme newname un fichier ou un dossier appelé à l'origine oldname.

Exemple

Dim OldName As String
Dim NewName As String
Dim result As Integer 

OldName = "dsc001.jpg"
NewName = "landscape.jpg"

result = Name( OldName, NewName )
If 0 <> result Then 
    Print "error renaming " & oldname & " to " & newname & "."
End If


Différences avec QB

  • Dans QB, NAME nécessite AS plutôt qu'une virgule entre ancien et nouveau noms.
