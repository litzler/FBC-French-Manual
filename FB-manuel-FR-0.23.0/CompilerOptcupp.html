Option du compilateur: -C
 
Ne pas supprimer le(s) fichier(s) objet

Syntaxe

-C

Description

L'option -C du compilateur implique que le(s) fichier(s) objet qui est/sont généré(s) pendant le processus de compilation de n'est/ne sont pas supprimé(s).
