RSet
 
Justification à droite d'une chaîne de caractères

Syntaxe

Declare Sub RSet Overload ( ByRef dst As String, ByRef src As Const String )
Declare Sub RSet ( ByVal dst As WString Ptr, ByVal src As Const WString Ptr )

Usage

RSet dst, src

Paramètres

dst
Un tampon String ou WString pour copier le texte dedans.
src
La String ou WString source à justifier à droite.

Description

RSet justifie le texte à droite dans le tampon chaîne dst, en remplissant la partie droite de la chaîne avec src et la partie gauche avec des espaces.

Exemple

Dim buffer As String
buffer = Space(10)
RSet buffer, "91.5"
Print "-[" & buffer & "]-"


Différences avec QB

  • Dans QB, la syntaxe est RSet dst = src
