Opérateur \ (Division entière)
 
Divise deux expressions Integer (=entier)

Syntaxe

Declare Operator \ ( ByRef lhs As Integer, ByRef rhs As Integer ) As Integer

Usage

result = lhs \ rhs

Paramètres

lhs
Le dividende, argument côté gauche.
rhs
Le diviseur, argument côté droit.

Valeur retournée


Retourne le quotient d'un dividende et d'un diviseur Integer (=entier).

Description

Operator \ (Division entière) divise deux expressions Integer (=entier) et retourne le résultat. Les valeurs nuériques à virgule sont converties en Integer (=entier) en arrondissant par excès ou par défaut, et la partie décimale du quotient résultant est tronquée.

Si le diviseur (rhs) est zéro (0), une erreur "division par zéro" (plantage!) est affichée.

Dans tous les cas, aucun opérande n'est modifié.

Cet opérateur peut être surchargé pour accepter les types définis par l'utilisateur.

Exemple

Dim n As Double
Print n \ 5
n = 7 \ 2.6  '' => 7 \ 3  => 2.33333  => 2
Print n
n = 7 \ 2.4  '' => 7 \ 2 => 3.5 => 3
Print n
Sleep

Affiche:
0
2
3

Différences de dialecte

  • Dans le dialecte -lang qb, cet opérateur ne peut pas être surchargé.

Différences avec QB

  • Aucune
