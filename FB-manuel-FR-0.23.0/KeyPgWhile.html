While
 
Déclaration de contrôle de flux.

Syntaxe

Do While condition
[statement block]
Loop
ou
Do
[statement block]
Loop While condition
ou
While [condition]
[statement block]
Wend

Description

While spécifie qu'un bloc de boucle continuera si la condition qui suit est vraie (true). Cette condition est vérifiée pour chaque itération de la boucle.
