CSng
 
Convertit une expression numérique ou une chaîne vers un nombre Single précision à virgule flottante.

Syntaxe

Declare Function CSng ( ByVal expression As datatype ) As Single

Type typename
Declare Operator Cast ( ) As Single
End Type

Usage

result = CSng( numeric expression )
result = CSng( string expression )
result = CSng( user defined type )

Paramètres

expression
Une expression numérique, chaîne ou pointeur à transformer (cast) en valeur Single
datatype
Tout type de données numérique, chaîne ou pointeur.
typename
Un type défini par l'utilisateur.

Valeur retournée

Une valeur Single précision.

Description

La fonction CSng retourne une valeur Single 32 bits. La fonction ne vérifie pas s'il y a un dépassement, alors assurez-vous qu'elle ne soit pas en dehors de la plage valide du type de données Single. Le nom vient de 'Convert to SiNGle'.

Si l'argument de CSng est une expression chaîne, elle sera convertie en numérique avec Val.

Exemple

' Using the CSNG function to convert a numeric value

'Create an SINGLE variable
Dim numeric_value As Single

'Convert a numeric value
numeric_value = CSng(-12345.123)

'Print the result, should return -12345.123
Print numeric_value
