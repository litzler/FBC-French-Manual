UCase
 
Retourne une copie en lettres majuscules d'une chaîne de caractères

Syntaxe

Declare Function UCase Overload ( ByRef str As Const String ) As String
Declare Function UCase ( ByRef str As Const WString ) As WString

Usage

result = UCase[$]( str )

Paramètres

str
Chaîne de caractères à convertir en majuscules.

Valeur retournée

Copie en lettres majuscules de str.

Description

Retourne une copie de str avec toutes les lettres converties en lettres majuscules.

Si str est vide, une chaîne nulle ("") est retournée.

Exemple

Print UCase("AbCdEfG")

produira la sortie:

ABCDEFG

Différences de plate-forme

  • La version large chaîne WString de UCase n'est pas supportée pour la cible DOS.

Différences de dialecte

  • Le suffixe de type de chaîne "$" est obligatoire dans le dialecte -lang qb.
  • Le suffixe de type de chaîne "$" est facultatif dans les dialectes -lang fblite et -lang fb.

Différences avec QB

  • QB ne supporte pas Unicode.
