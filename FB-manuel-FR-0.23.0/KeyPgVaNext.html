va_next
 
Renvoie un pointeur sur l'argument suivant dans une liste d'arguments variable

Syntaxe

Argument_Pointer = va_next (Argument_List, datatype )

Description

La macro va_next pointe vers l'argument suivant dans la liste Argument_List de type datatype.

Exemple

Function Avg cdecl (Count As Integer, ... ) As Double
    Dim ARG As Any Ptr 
    Dim SUM As Double = 0
    Dim i As Integer
    
    ARG = va_first()

    For i = 1 To COUNT
        SUM += va_arg(ARG, Double)
        ARG = va_next(ARG,Double)
    Next i
    Return SUM/COUNT
End Function

Print AVG (4, 3.4,5.0,3.2,4.1)
Print AVG (2, 65.2,454.65481)
