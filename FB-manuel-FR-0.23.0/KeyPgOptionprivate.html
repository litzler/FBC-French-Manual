Option Private
 
Spécifie la liaison interne par défaut pour les déclarations de procédure

Syntaxe

Option Private

Description

Option Private est une déclaration qui donne à toutes les déclarations de procédure suivantes une liaison interne par défaut, comme si elles étaient déclarées avec Private. Ceci reste en vigueur pour le reste du module dans lequel Option Private est utilisé et ne peut être annulé qu'en déclarant les procédures avec Public.

Exemple

'' Compile with the "-lang fblite" compiler switch

#lang "fblite"

Sub ProcWithExternalLinkage()
   ' ...
End Sub

Option Private

Sub ProcWithInternalLinkage()
   ' ...
End Sub

Public Sub AnotherProcWithExternalLinkage()
   ' ...
End Sub

Différences de dialecte

  • Disponible seulement dans les dialectes -lang fblite et -lang qb.

Différences avec QB

  • Nouveau pour FreeBASIC
