Step
 
Modification de déclaration.

Syntaxe

For iterator = initial_value To end_value Step increment

Line [ buffer, ] Step ( x1, y1 ) - Step ( x2, y2 ) [, [ color ][, [ B|BF ][, style ] ] ]

Circle [ target, ] Step ( x, y ), radius [, [ color ][, [ start ][, [ end ][, [ aspect ][, F] ] ] ] ]

Paint [ target, ] STEP ( x, y ) [, [ paint ][, [ border_color ] ] ]

Description

Dans une déclaration For, Step indique l'incrémentation de l'itération de la boucle à chaque boucle.

Dans une déclaration Line, Circle ou Paint, Step indique que les coordonnées qui suivent ont des valeurs relatives à la position du curseur graphique.

Exemple

Dim i As Integer
For I=10 To 1 Step -1
Next



Line -Step(10,10),13
