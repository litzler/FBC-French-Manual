ScreenList
 
Détermine les modes plein écran valides

Syntaxe

Declare Function ScreenList ( ByVal depth As Integer = 0 ) As Integer

Usage

result = ScreenList( [ depth ] )

Paramètres

depth
la profondeur de couleur pour laquelle on détermine la liste de modes (les profondeurs supportées sont 8, 15, 16, 24 and 32)

Valeur retournée

retourne 0, s'il n'y a aucune résolution à lire.

Fonctionnement identique à celui de la fonction Dir : au premier appel de la fonction le paramètre depth est obligatoire, la fonction renvoie la résolution la plus basse supportée par cette profondeur de couleur. Les appels suivants à ScreenList sans argument renvoie les résolutions suivantes. Quand il n'y a plus de résolution valable, ScreenList retourne 0.

Le résultat de ScreenList est encodé dans un entier, la largeur d'écran dans le mot de poids fort et la hauteur d'écran dans le mot de poids faible.

Les résolutions supportées sont renvoyées de la plus faible à la plus forte.

Il est préférable d'appeler cette fonction avant qu'un mode graphique soit validé.

Dim As Integer mode, w, h
'' Cherche quelles resolutions 8bit sont supportees
mode = ScreenList(8)
While (mode)
    w = HiWord(mode)
    h = LoWord(mode)
    Print Str(w) + "x" + Str(h)
    mode = ScreenList
Wend
