OffsetOf
 
Retourne la position (offset) d'un champ dans un type.

Syntaxe

#define OffsetOf(typename, fieldname) CInt( @Cast( typename Ptr, 0 )->fieldname )

Usage

result = OffsetOf( typename, fieldname )

Paramètres

typename
Nom du type qui a été défini avec Type...End Type.
fieldname
Nom du type de champ qui a été défini dans le type.

Description

OffsetOf retournera la position, en octets, de fieldname depuis le début de typename.

Example

Type MyType
  x As Single
  y As Single
  Union
    b As Byte
    i As Integer
  End Union
End Type

Print "OffsetOf x = "; OffsetOf(MyType, x)
Print "OffsetOf y = "; OffsetOf(MyType, y)
Print "OffsetOf b = "; OffsetOf(MyType, b)
Print "OffsetOf i = "; OffsetOf(MyType, i)


Output
OffsetOf x =  0
OffsetOf y =  4
OffsetOf b =  8
OffsetOf i =  8


Différences de dialecte

  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __Offsetof.

Différences avec QB
