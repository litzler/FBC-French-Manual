CVS
 
Conversion d'une chaîne de 4 octets en une variable simple-précision

Syntaxe

Declare Function CVS ( ByRef str As Const String ) As Single

Usage

result = CVS( str )

Paramètres

str
Une chaîne (String) ayant exactement 4 octets de longueur, copie d'une variable simple-précision.

Valeur retournée

Retourne une variable Single pour contenir la copie binaire d'un nombre SIMPLE PRECISION.

Description

Effectue une copie d'une chaîne-4-octets vers une variable Single (simple précision). Le résultat n'aura un sens que si la chaîne contient une valeur SIMPLE PRECISION formatée IEEE754, comme celles produites par MKS.

Cette fonction est utile pour lire des valeurs à partir de tampons sans utiliser une définition de Type.

Exemple

Dim a As Single, b As String
a=4534.4243
b=MKS(a)
Print a, CVS(b)
Sleep
