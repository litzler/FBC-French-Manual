This
 
Paramètre instance caché passé aux membres non-statiques des fonctions d'un Type ou d'une Class

Syntaxe

This.fieldname
or
With This
.fieldname
End With

Description

This est un paramètre caché passé à toutes les fonctions membres non-statiques d'une Type ou Class. Les fonctions membres non-statiques sont des procédures déclarées à l'intérieur du corps d'un Type ou d'une Class et comprennent Sub, Function, Constructor, Destructor, opérateurs d'assignation ou cast, et les procédures Property.

Le paramètre This a le même type de donnée que le Type ou la Class où la procédure est déclarée.

Exemple

Type sometype
    Declare Sub MyCall()
    value As Integer
End Type

Dim example As sometype

'' Set element test to 0
example.value = 0
Print example.value

example.MyCall()

'' Output should now be 10
Print example.value

End 0

Sub sometype.MyCall()
    This.value = 10
End Sub
