Unsigned
 
Modificateur du type de données Integer (= entier)

Syntaxe

Dim variable As Unsigned {integer-based data type}

Description

Force un nombre entier de base à être non-signé (ne peut contenir de nombres négatifs, mais voit sa valeur maximum doublée).

Exemple

'e.g. notice what is displayed:

Dim x As Unsigned Integer
x = -1
Print x 

'output is 4294967295



Différences de dialecte

  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __Unsigned.

Différences avec QB

  • Nouveau pour FreeBASIC
