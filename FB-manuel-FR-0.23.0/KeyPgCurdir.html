CurDir
 
Retourne le répertoire/dossier actuel

Syntaxe

Declare Function CurDir ( ) As String

Usage

result = CurDir

Valeur retournée

Une chaîne (String) qui contient le nom du répertoire/dossier actuel.

Description

Retourne le nom du répertoire/dossier actuel.

Exemple

Print CurDir

L'affichage peut varier.

Différences de dialecte

  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __Curdir.

Différences avec QB

  • Nouveau pour FreeBASIC
