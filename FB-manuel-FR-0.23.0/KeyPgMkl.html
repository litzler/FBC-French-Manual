MKL
 
Copie binaire d'une variable Long en une String (chaîne), réglage de sa longueur à 4 octets

Syntaxe

Declare Function MKL ( ByVal number As Integer ) As String

Usage

result = MKL( number )

Paramètres

number
Une variable Integer (entière) dont on fait une copie binaire vers une String (chaîne).

Valeur retournée

Retourne une String (chaîne) copie binaire d'un nombre entier Long.

Description

Exécute une copie binaire d'une variable entière long(ue) vers une String (chaîne) dont la longueur est de 4 octets. La chaîne résultat peut être lue à nouveau comme un Long par CVL.

Cette fonction est utile pour écrire une valeur numérique dans un tampon sans utiliser de définition de TYPE.

Exemple

Dim a As Long, b As String
a = 4534
b = MKL(a)
Print a, CVL(b)
Sleep


Différences de dialectes

  • Le suffixe de type de chaîne "$" est obligatoire dans le dialecte -lang qb.
  • Le suffixe de type de chaîne "$" est facultatif dans les dialectes -lang fblite et -lang fb.

Différences avec QB

  • Aucune
