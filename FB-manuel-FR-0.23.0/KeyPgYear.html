Year
 
Récupère l'année à partir d'une Date Serial

Syntaxe

Declare Function Year ( ByVal date_serial As Double ) As Integer

Usage

#include "vbcompat.bi"
result = Year( date_serial )

Paramètres

date_serial
la date

Valeur retournée

Retourne l'année à partir d'une variable contenant une date sous forme de Date Serial.

Description


Le compilateur ne reconnaît cette fonction que si vbcompat.bi est inclus.

Exemple

#include "vbcompat.bi"

Dim a As Double = DateSerial (2005, 11, 28) + TimeSerial(7, 30, 50)

Print Format(a, "yyyy/mm/dd hh:mm:ss "); Year(a)


Différences avec QB

  • N'existe pas dans QB. Cette fonction est apparue dans PDS et VBDOS
