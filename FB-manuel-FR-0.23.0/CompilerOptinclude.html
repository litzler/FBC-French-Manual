Option du compilateur: -include
 
Inclure un fichier en-tête dans chaque source compilé

Syntaxe

-include < include file >

Paramètres

include file
Le nom du fichier d'en-tête, à inclure, avec l'extension et éventuellement un chemin.

Description

L'option -include a pour effet ajouter une directive #include du préprocesseur au début de chaque fichier source avant de le traiter. Lorsqu'elle est utilisée à plusieurs reprises, les fichiers seront inclus dans l'ordre où ils sont inscrits sur la ligne de commande.

Pour inclure le fichier header.bi lors de la compilation de file1.bas et file2.bas, tapez:

fbc -include header.bi file1.bas file2.bas
