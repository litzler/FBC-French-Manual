Trim
 
Enlève les sous-chaînes qui entourent ou les caractères sur les côtés gauche et droit d'une chaîne.

Syntaxe

Declare Function Trim Overload ( ByRef str As Const String, [ Any ] ByRef trimset As Const String = " " ) As String
Declare Function Trim ( ByRef str As Const WString, [ Any ] ByRef trimset As Const WString = WStr(" ") ) As WString

Usage

result = Trim[$]( str [, [ Any ] trimset ] )

Paramètres

str
La chaîne source.
trimset
La sous-chaîne à enlever.

Valeur retournée

Retourne la chaîne "nettoyée" à gauche et à droite.

Description

Cette procédure élimine les caractères entourant à gauche (début) et à droite (fin) de la chaîne source. Les sous-chaînes correspondant à trimset seront éliminées si spécifiées, autrement les espaces (ASCII code 32) sont éliminés.

Si le mot-clef Any est utilisé, chaque caractère correspondant à un caractère dans trimset sera éliminé.

Toutes les comparaisons sont sensibles à la casse.

Exemple

Dim s1 As String = " ... Stuck in the middle ... "
Print "'" + Trim(s1) + "'"
Print "'" + Trim(s1, Any " .") + "'"

Dim s2 As String = "BaaBaaaaB With You aaBBaaBaa"
Print "'" + Trim(s2, "Baa") + "'"
Print "'" + Trim(s2, Any "Ba") + "'"

produira la sortie:

'... Stuck in the middle ...'
'Stuck in the middle'
'aaB With You aaB'
' With You '

Différences de plate-forme

  • DOS version/cible de FreeBASIC ne supporte pas la version larges-caractères de Trim.

Différences de dialectes

  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __Trim.
  • Le suffixe de type de chaîne "$" est obligatoire dans le dialecte -lang qb.
  • Le suffixe de type de chaîne "$" est facultatif dans les dialectes -lang fblite et -lang fb.

Différences avec QB

  • Nouveau pour FreeBASIC
