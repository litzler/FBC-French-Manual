Option du compilateur: -z
 
Définit des options de compilation diverses ou expérimentales.

Syntaxe

-z < value >

Paramètres

value
Diverses options du compilateur.

Description

L'option du compilateur -z définit des options diverses, obscures, temporaires ou expérimentales utilisées par les développeurs. Il n'y a aucune garantie que ces options seront prises en charge dans les futures versions du compilateur.

-z gosub-setjmp
Spécifie que la mise en oeuvre de setjmp/longjmp de GoSub devrait être utilisé même lorsque GAS est utilisé. Par défaut, GoSub sera supporté dans -gen gas en utilisant les instructions d'assemblage CALL/RET et dans -gen gcc en utilisant les fonctions C setjmp/longjmp.
