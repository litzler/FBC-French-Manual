Write (Accès fichiers)
 
Spécificateur d'accès fichier

Syntaxe

Open filename As String For Binary Access Write As #filenum As Integer

Description

Spécificateur pour la clause d'Access de la déclaration Open. Write spécifie que le fichier est accessible en sortie.
