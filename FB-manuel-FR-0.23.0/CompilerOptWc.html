Option du compilateur: -Wc
 
Passe les options pour le compilateur C lors de l'utilisation de l'émetteur d'assemblage C (-gen gcc).

Syntaxe

-Wc < options >

Paramètres

options
Options supplémentaires à passer au compilateur C.

Description

L'option du compilateur -Wc passe des options supplémentaires pour GCC, le compilateur C. Les options doivent être séparées seulement par des virgules.
Par exemple:
fbc -gen gcc -Wc -m32,--verbose,-include,some-header.h
