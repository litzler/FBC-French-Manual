Short
 
Type de données standard: nombre entier signé de 16 bits

Syntaxe

Dim variable As Short

Description

Type de données: nombre entier signé de 16 bits. Peut contenir des valeurs de -32768 à 32767.

Exemple

  Dim x As Short = CShort(&H8000)
  Dim y As Short = CShort(&H7FFF)
  Print "Short Range = "; x; " to "; y

Affichage:
Short Range = -32768 to  32767


Différences de dialecte

  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __Short.

Différences avec QB

  • Le nom "short" est nouveau pour FreeBASIC, toutefois, ils sont identiques aux entiers de QB
