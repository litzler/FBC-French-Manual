DateAdd
 
Décale une date avec un intervalle donné

Syntaxe

Declare Function DateAdd ( ByRef interval As Const String, ByVal number As Double, ByVal date_serial As Double ) As Double

Usage

#include "vbcompat.bi"
result = DateAdd( interval, number, date_serial )

Paramètres

interval
chaîne indiquant à quelle période de temps correspond une unité de number
number
le nombre d'intervalles à ajouter à la date de base. Le nombre sera arrondi à l'entier le plus proche.
date_serial
La date de base

Valeur retournée

Retourne une Date Serial correspondant à la date_serial reçue plus le nombre (number) d' intervalles (intervals).

Description

L'intervalle est spécifié comme ci-dessous:

valeurintervalle
yyyyannée
qtrimestre
mmois
wwsemaines
d,w,yjours
hheures
nminutes
ssecondes


Le compilateur ne reconnaît cette fonction que si vbcompat.bi ou datetime.bi est inclus.

Exemple

#include "vbcompat.bi"

Const fmt = "ddddd ttttt"
Dim d As Double
d = Now()

Print "1 hour from now is ";
Print Format( DateAdd( "h", 1, d ), fmt )

Print "1 day from now is ";
Print Format( DateAdd( "d", 1, d ), fmt )

Print "1 week from now is ";
Print Format( DateAdd( "ww", 1, d ), fmt )

Print "1 month from now is ";
Print Format( DateAdd( "m", 1, d ), fmt )


Différences avec QB

  • N'existe pas dans QB. Cette fonction est apparue dans Visual Basic.
