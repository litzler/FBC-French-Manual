SAdd
 
Retourne un pointeur vers les données d'une variable chaîne

Syntaxe

Declare Function Overload SAdd ( ByRef str As String ) As Any Ptr
Declare Function SAdd ( ByRef str As WString ) As Any Ptr
Declare Function SAdd ( ByRef str As ZString ) As Any Ptr

Usage

result = SAdd( str )

Paramètres

str
l'expression chaîne ou variable dont on veut obtenir l'adresse

Valeur retournée

Un pointeur vers les données associées à str.

Description

Retourne la position en mémoire des données dans la variable chaîne.

Exemple

Dim s As String

Print SAdd(s)
s = "hello"
Print SAdd(s)
s = "abcdefg, 1234567, 54321"
Print SAdd(s)
