WSpace
 
Crée une WString de longueur donnée remplie d'espaces (" ")

Syntaxe

Declare Function WSpace( ByVal count As Integer ) As WString

Usage

result = WSpace( count )

Paramètres

count
Un entier spécifiant la longueur de la chaîne à créer.

Valeur retournée

La WString créée. Une chaîne vide est retournée si count <= 0.

Description

WSpace crée une wstring (une chaîne large-caractère - Unicode) avec le nombre spécifié d'espaces.

Exemple

Dim a As WString * 10
a = "x" + WSpace(3) + "x"
Print a ' prints: x   x


Différences de plate-forme

  • (w)strings Unicode ne sont pas supportées dans le portage DOS de FreeBASIC.

Différences de dialectes

  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __Wspace.

Différences avec QB

  • Nouveau pour FreeBASIC
