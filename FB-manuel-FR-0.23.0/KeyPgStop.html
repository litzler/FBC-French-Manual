Stop
 
Arrête l'exécution du programme et attend qu'une touche soit appuyée avant de terminer le programme.

Syntaxe

Declare Sub Stop ( ByVal retval As Integer = 0 )

Usage

Stop

Paramètres

retval
Code d'erreur retourné au système.

Description

Arrête l'exécution du programme et attend. Prévu comme une aide au débogage, car il préserve la mémoire et ne ferme pas les fichiers. Pour terminer normalement le programme, le mot-clé End ne doit être utilisé. Une valeur de retour facultative, un nombre entier, peut être spécifiée pour retourner un code d'erreur au système. Si aucune valeur de retour n'est donnée, une valeur de 0 est automatiquement retournée.

Exemple

Print "this text is shown"
Sleep
Stop
Print "this text will never be shown"
