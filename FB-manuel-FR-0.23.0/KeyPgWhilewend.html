While
 
Déclaration de contrôle de flux pour une boucle.

Syntaxe

While [condition]
[statement block]
Wend

Description

La déclaration While entraînera l'ensemble suivant des instructions de statement block à s'exécuter répétitivement tant que l'expression condition est vraie (true).

Si condition est évaluée à faux (false) quand l'instruction WHILE est exécutée la première fois, le statement block est sauté et l'exécution reprend immédiatement après le Wend de fin de structure.

Si une déclaration Exit While est rencontrée dans le statement block, la boucle se termine et l'exécution reprend immédiatement après le Wend de fin de structure. Si une déclaration Continue While est rencontrée, le reste du statement block est ignorée et l'exécution reprend à la déclaration While.

Comme toutes les déclaration de contrôle de flux, l'instruction While peut être imbriquée, c'est à dire qu'elle peut être utilisée dans un bloc d'instructions d'une autre déclaration While.

Note: le mot-clef While est aussi utilisé dans la déclaration Do...Loop pour indiquer le type de comparaison. Utilisée de cette façon, la déclaration DO devient fonctionnellement équivalente à l'instruction WHILE, donc il ne faut pas confondre les mots-clés entourant Loop et Wend, respectivement.

Exemple

Dans cet exemple, une boucle While est utilisée pour inverser une chaîne par itération à travers elle en arrière. La boucle s'arrête si l'indice est inférieur à 0 (0 étant le premier indice de la chaîne).
Dim As String sentence                          '' string to reverse
sentence = "The quick brown fox jumps over the lazy dog."

Dim As String ecnetnes
Dim As Integer index
index = Len( sentence ) - 1                     '' point to last character
While( index >= 0 )                             '' stop after first character
  ecnetnes += Chr( sentence[index] )           '' append character to new string
  index -= 1
Wend

Print "original: """ ; sentence ; """"
Print "reversed: """ ; ecnetnes ; """"

End 0


Différences de dialectes

  • Dans les dialectes -lang qb et -lang fblite, les variables déclarées dans une boucle WHILE..WEND ont une portée fonction comme dans QB.
  • Dans les dialectes -lang fb et -lang deprecated, les variables déclarées dans une boucle While..Wend ne sont visibles que dans ce bloc et inaccessibles en-dehors.

Différences avec QB

  • Aucune
