Fonctions chaînes
 
Déclarations et procédures qui travaillent sur les chaînes.

Description

Ces déclarations et procédures fournissent de nombreux moyens pour créer et manipuler chaîne et sous-chaînes. Les nombres peuvent être convertis en chaînes et vice-versa. Des procédures sont également fournies pour faciliter la sérialisation des données numériques, même pour le stockage persistant.

Création de chaînes
Types de données chaîne et procédures qui céent de nouvelles chaînes.
Conversions de caractères
Procédures qui convertissent des codes carctères en chaînes et vice-versa.
Conversions numérique vers chaîne
Procédures qui convertissent des valeurs numériques en chaînes.
Conversions chaîne vers numérique
Procédures qui convertissent des chaînes en valeurs numériques.
Sérilisations numériques
Procédures qui convertissent des données numériques brutes vers/depuis des chaînes appropriées pour le stockage.
Travail avec les sous-chaînes
Procédures qui retournent des sous-ensembles de chaînes, ou qui modifient des sous-ensembles de chaînes.

Création de chaînes
String
Type de données standard : chaîne de caractères 8 bit.
String (Function)
Retourne une String de plusieurs caractères.
ZString
Type de données standard : chaîne de caractères 8 bit terminée par "null".
WString
Type de données standard : chaînes de caractères larges (2 octets).
Wstring (Function)
Retourne une WString de multiples caractères.
Space
Retourne une String remplie d'espaces.
WSpace
Retourne une WString remplie d'espaces.
Len
Retourne la longueur en caractères de la chaîne.

Conversions de caractères
Asc
Retourne un Integer (=entier) représentation d'un caractère.
Chr
Retourne une chaîne d'un ou plusieurs caractères à partir de leur représentation Integer ASCII.
WChr
Retourne une WString d'un ou plusieurs caractères à partir de leur représentation Integer Unicode.

Conversions numérique vers chaîne
Bin
Retourne une String binaire représentation d'une valeur entière.
WBin
Retourne une WString binaire représentation d'une valeur entière.
Hex
Retourne une String hexadécimale représentation d'une valeur entière.
WHex
Retourne une WString hexadécimale représentation d'une valeur entière.
Oct
Retourne une String octale représentation d'une valeur entière.
WOct
Retourne une WString octale représentation d'une valeur entière.
Str
Retourne la String représentation d'une valeur numérique.
WStr
Retourne la WString représentation d'une valeur numérique.
Format
Retourne une String formatée représentation d'un Double.

Conversions chaîne vers numérique
Val
Retourne le Double conversion d'une chaîne numérique.
ValInt
Retourne l'Integer conversion d'une chaîne numérique.
ValLng
Retourne le Long conversion d'une chaîne numérique.
ValUInt
Retourne le UInteger conversion d'une chaîne numérique.
ValULng
Retourne le Ulong conversion d'une chaîne numérique.

Sérialisations numériques
MKD
Retourne une String de huit octets représentation d'un Double.
MKI
Retourne une String de quatre octets représentation d'un Integer.
MKL
Retourne une String de quatre octets représentation d'un Long.
MKLongInt
Retourne une String de huit octets représentation d'un LongInt.
MKS
Retourne une String de quatre octets représentation d'un Single.
MKShort
Retourne une String de deux octets représentation d'un Short.
CVD
Retourne un Double représentation d'une String de huit octets.
CVI
Retourne an Integer représentation d'une String de quatre octets.
CVL
Retourne a Long représentation d'une String de quatre octets.
CVLongInt
Retourne a LongInt représentation d'une String de huit octets.
CVS
Retourne a Single représentation d'une String de quatre octets.
CVShort
Retourne a Short représentation d'une String de deux octets.

Travail avec les sous-chaînes
Left
Retourne une sous-chaîne des caractères les plus à gauche d'une chaîne.
Mid (Function)
Retourne une sous-chaîne d'une chaîne.
Right
Retourne une sous-chaîne des caractères les plus à droite d'une chaîne.
LCase
Retourne une copie d'une chaîne convertie en caractères alphabétiques minuscules.
UCase
Retourne une copie d'une chaîne convertie en caractères alphabétiques majuscules.
LTrim
Enlève les sous-chaines qui entourent ou les caractères sur le côté gauche d'une chaîne.
RTrim
Enlève les sous-chaines qui entourent ou les caractères sur le côté droit d'une chaîne.
Trim
Enlève les sous-chaines qui entourent ou les caractères sur les côtés gauche et droit d'une chaîne.
InStr
Repère la première occurrence d'une sous-chaine ou d'un caractère dans une chaîne.
InStrRev
Repère la dernière occurrence d'une sous-chaine ou d'un caractère dans une chaîne.
Mid (Statement)
Copie une sous-chaine vers une sous-chaine d'une chaîne.
LSet
Justification à gauche d'une chaîne.
RSet
Justification à droite d'une chaîne.
