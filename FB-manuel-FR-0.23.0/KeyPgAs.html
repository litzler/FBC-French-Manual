As
 
Partie facultative d'une déclaration qui spécifie un type de données, ou une partie de l'instruction Open, qui spécifie un identificateur de fichier.

Syntaxe

symbolname As datatype

Open ... As #filenumber
Type ... As datatype

Description

As est utilisé pour déclarer le type d'une variable, d'un champ ou d'un argument et est également utilisé dans Open pour déterminer l'identificateur d'un fichier. As est aussi utilisé dans la syntaxe de Type (Alias), similaire à l'instruction typedef du langage C.

Exemple

'' don't try to compile this code, the examples are unrelated
Declare Sub mySub (X As Integer, Y As Single, Z As String)
' ...

Dim X As Integer
' ...

Type myType
  X As Integer
  Y As Single
  Z As String
End Type
' ...

Type TheNewType As myType
' ...

Open "test" For Input As #1
' ...


Différences avec QB

  • La syntaxe Type (Alias) n'est pas admise dans QB.
