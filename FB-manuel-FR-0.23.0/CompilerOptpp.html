Option du compilateur: -pp
 
Produit le fichier d'entrée traîté seulement par le préprocesseur, ne compile pas

Syntaxe

-pp

Description

L'option -pp du compilateur permet seulement le mode préprocesseur. Le code est analysé et contrôlé comme d'habitude, mais n'est pas compilé. Une version traitée pré-processeur de chaque fichier source.bas est générée, nommée source.pp.bas.
