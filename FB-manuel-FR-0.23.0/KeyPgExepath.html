ExePath
 
Retourne le chemin du programme en cours

Syntaxe

Declare Function ExePath ( ) As String

Usage

result = ExePath

Valeur retournée

Une variable String contenant le chemin du programme en cours.

Description

Renvoie le chemin (l'emplacement) du programme appelant. Ce n'est pas nécessairement le même que CurDir.

Exemple

Dim pathname As String = ExePath
Print "This program's initial directory is: " & pathname



Différences de dialecte

  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __Exepath.

Différences avec QB

  • Nouveau pour FreeBASIC
