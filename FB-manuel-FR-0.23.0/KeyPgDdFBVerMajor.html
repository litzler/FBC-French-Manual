__FB_VER_MAJOR__
 
Définition intrinsèque (valeur de la macro) fixée par le compilateur

Syntaxe

__FB_VER_MAJOR__

Description

__FB_VER_MAJOR__ retournera la version majeure de FreeBASIC utilisée actuellement. Par exemple, la version majeure est 0 pour Freebasic 0.18, et restera 0 jusqu'à FreeBasic 1.0.

Exemple

Dim fbMajorVersion As Integer
Dim fbMinorVersion As Integer

fbMajorVersion = __FB_VER_MAJOR__
fbMinorVersion = __FB_VER_MINOR__

Print "Welcome to freebasic ";fbMajorVersion;".";fbMinorVersion


Différences avec QB

  • Nouveau pour FreeBASIC
