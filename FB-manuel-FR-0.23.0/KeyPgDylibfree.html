DyLibFree
 
Décharge une bibliothèque à liens dynamiques de la mémoire

Syntaxe

Declare Sub DyLibFree ( ByVal library As Any Pointer )

Usage

DyLibFree( library )

Paramètres

library
Le nom du pointeur d'une bibliothèque à décharger.

Description

DyLibFree est utilisé pour libérer les bibliothèques d'exécution préalablement liées à votre programme avec DyLibLoad. L'argument est le nom du pointeur de la bibliothèque retourné par DyLibLoad.

Exemple

Dim hndl As Any Ptr

hndl=DyLibLoad("MYLIB.DLL")
' ...
DyLibFree(hndl)



Différences de plate-forme

  • Les bibliothèques à liens dynamiques ne sont pas disponibles dans DOS, car le système d'exploitation ne les prend pas en charge.

Différences de dialecte

  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __Dylibfree.

Différences avec QB

  • Nouveau pour FreeBASIC
