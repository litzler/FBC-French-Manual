Acos
 
Détermine l'arc cosinus d'un angle

Syntaxe

Declare Function Acos ( ByVal number As Double ) As Double

Usage

result = Acos( number )

Paramètres

number
Une valeur de cosinus dans l'intervalle [-1..1].

Valeur retournée

L'arc cosinus de number en radians, dans l'intervalle [0..Pi].

Description

Acos renvoie l'arc cosinus de l'argument number dans une valeur Double dans l'intervalle de 0 à Pi. L'arc cosinus est l'inverse de la fonction Cos. L'angle renvoyé est mesuré en radians (non pas en degrés).

Exemple

Dim h As Double
Dim a As Double
Input "Please enter the length of the hypotenuse of a triangle: ", h
Input "Please enter the length of the adjacent side of the triangle: ", a
Print ""
Print "The angle between the sides is"; Acos ( a / h )
Sleep

L'affichage sera:
Please enter the length of the hypotenuse of a triangle: 5
Please enter the length of the adjacent side of the triangle: 4

The angle between the sides is 0.6435011087932843
