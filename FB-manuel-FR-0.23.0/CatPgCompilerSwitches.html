Commutateurs du compilateur
 
Déclarations qui affectent la façon dont le code est compilé.

Description

Ces déclarations affectent la façon dont le compilateur déclare les variables, les tableaux et les procédures, analyse les chaînes littérales, passe les paramètres de procédure, etc.

Métacommandes
  • '$Dynamic
  • '$Include
  • '$Static
  • '$Lang

Options du compilateur
  • Option Base
  • Option ByVal
  • Option Dynamic
  • Option Escape
  • Option Explicit
  • Option Gosub
  • Option Nogosub
  • Option NoKeyword
  • Option Private
  • Option Static

Configuration par défaut des types de données
  • DefByte
  • DefDbl
  • DefInt
  • DefLng
  • Deflongint
  • DefShort
  • DefSng
  • DefStr
  • DefUByte
  • DefUInt
  • Defulongint
  • DefUShort


 
Différences de dialecte

  • Deflongint et Defulongint ne sont disponibles qu'avec le dialecte -lang fblite.
  • Les déclarations OPTION ne sont disponibles que dans les dialectes -lang fblite et -lang qb.
