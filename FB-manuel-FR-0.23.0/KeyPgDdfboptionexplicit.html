__FB_OPTION_EXPLICIT__
 
Définition intrinsèque (valeur de la macro) fixée par le compilateur

Syntaxe

__FB_OPTION_EXPLICIT__

Description

__FB_OPTION_EXPLICIT__ indique si Option Explicit a été utilisée précédemment dans le source.

Retourne zéro (0) si l'option n'a pas été validée. Retourne non-zéro (-1) si l'option a été validée.

Exemple

#if( __FB_OPTION_EXPLICIT__ = 0 )
  #error Option Explicit must used With This module
#endif
