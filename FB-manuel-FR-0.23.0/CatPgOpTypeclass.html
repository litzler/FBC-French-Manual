Opérateurs de Type ou Classe
 
Opérateurs qui travaillent avec les objets

Ces Opérateurs retournent des références sur des objets, donnent un objet ou son adresse mémoire.

Opérateur . (Accès membre)
Retourne une référence vers un membre.
Opérateur -> (Pointeur vers accès membre)
Retourne une référence vers un membre à partir d'un pointeur.
