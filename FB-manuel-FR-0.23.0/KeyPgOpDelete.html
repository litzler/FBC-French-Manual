Opérateur Delete
 
Opérateur pour détruire des données allouées avec l'opérateur New

Syntaxe

Declare Operator Delete ( buf As Any Ptr )
Declare Operator delete[] ( buf As Any Ptr )

Usage

Delete expression
ou
Delete[] expression

Paramètres

buf
Une adresse mémoire à libérer.
expression
Une adresse mémoire à libérer.

Description

Delete est utilisé pour détruire et libérer la mémoire d'un objet créé avec New. Quand on Delete un TYPE, son destructeur est appelé. Delete doit être utilisé uniquement avec des adresses retournées par New.

La version Delete pour les tableaux, Delete[], est utilisé pour détruire un tableau d'objets créés précédemment avec New[]. Les destructeurs seront là aussi appelés.

Delete doit être utilisé avec des adresses retournées par New, et Delete[] par New[]. Vous ne pouvez mélanger les différentes versions de ces opérateurs.

Exemple

Type Rational
    As Integer    numerator, denominator
End Type

Scope

    ' Create and initialize a Rational, and store it's address.
    Dim p As Rational Ptr = New Rational(3, 4)

    Print p->numerator & "/" & p->denominator

    ' Destroy the rational and give its memory back to the system. 
    Delete p

End Scope

Scope

    ' Allocate memory for 100 integers, store the address of the first one.
    Dim p As Integer Ptr = New Integer[100]

    ' Assign some values to the integers in the array.
    For i As Integer = 0 To 99
        p[i] = i
    Next

    ' Free the entire integer array.
    Delete[] p

End Scope

Différences de dialecte

  • Valable seulement dans le dialecte -lang fb.

Différences avec QB

  • Nouveau pour FreeBASIC
