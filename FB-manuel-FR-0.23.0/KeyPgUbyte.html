UByte
 
Type de données standard: nombre de 8 bits non-signé

Syntaxe

Dim variable As UByte

Description

Nombre entier de 8 bits non-signé. Peut contenir des valeurs de 0 à 255.

Exemple

Dim ubytevar As UByte
ubytevar = 200
Print "ubytevar= ", ubytevar

Example

  Dim x As UByte = 0
  Dim y As UByte = &HFF
  Print "UByte Range = "; x; " to "; y

Affichage:
UByte Range = 0 to 255


Différences de dialecte

  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __Ubyte.

Différences avec QB

  • Nouveau pour FreeBASIC
