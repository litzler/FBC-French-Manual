Abs
 
Calcule la valeur absolue d'un nombre

Syntaxe

Declare Function Abs Overload ( ByVal number As Integer ) As Integer
Declare Function Abs ( ByVal number As LongInt ) As LongInt
Declare Function Abs ( ByVal number As Double ) As Double

Usage

result = Abs( number )

Paramètres

number
Valeur pour laquelle on évalue la valeur absolue.

Valeur retournée

La valeur absolue de number.

Description

La valeur absolue d'un nombre est sa grandeur non signée. Par exemple, Abs(-1) et Abs(1) retournent tous les deux 1. L'argument requis number peut être n'importe quelle expression numérique.
L'Operator unaire Abs peut être surchargé avec des types définis par l'utilisateur..

Exemple

Dim n As Integer

Print Abs( -1 )
Print Abs( -3.1415 )
Print Abs( 42 )
Print Abs( n )

n = -69

Print Abs( n )

Affiche:
1
3.1415
42
0
69

Différences de dialecte

  • Dans le dialecte -lang qb, cet opérateur ne peut pas être surchargé.
