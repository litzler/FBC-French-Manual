DefUByte
 
Spécifie un type de données par défaut pour une série de noms de variables

Syntaxe

DefUByte start_letter[-end_letter ][, ...]

Paramètres

start_letter
la première lettre dans la série
end_letter
la dernière lettre dans la série

Description

DefUByte précise que les variables et les tableaux qui ne sont pas déclarés avec un type de données - ou non déclarés entièrement - sont implicitement déclarés de type UByte si la première lettre de leur nom correspond à une certaine lettre ou se trouve dans une plage inclusive de lettres.

Exemple

Ceci fera de uNumber un nombre UByte car sa première lettre est u.
'' Compile with -lang fblite

#lang "fblite"

DefUByte u
Dim uNumber


Différences de dialecte

  • Disponible seulement dans le dialecte -lang fblite.
  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __Defubyte.

Différences avec QB

  • Nouveau pour FreeBASIC
