Option du compilateur: -map
 
Enregistrer la carte de l'édition des liens sous un nom de fichier

Syntaxe

-map < map file >

Paramètres

map file
Nom du fichier pour sauver la carte générée pedant l'édition des liens.

Description

L'option de compilateur -map enregistre dans un fichier la carte de l'édition des liens de l'exécutable .
