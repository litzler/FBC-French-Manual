ScreenSync
 
Synchronises les mises à jour d'affichage avec le "hardware"

Syntaxe

Declare Function ScreenSync ( ) As Integer

Usage

result = ScreenSync

Valeur retournée

Zéro si réussi, ou non-zéro si un mode graphique n'a pas encore été validé.

Description

Cette fonction GfxLib arrÃªtes l'exécution du programme tant que la carte graphique n'a pas terminé le balayage complet de l'écran et va en commencer un autre. Si le programme utilise le très petit intervalle de temps entre deux affichages pour redessiner l'image, le scintillement est grandement atténué.

L'utilisation de la forme QB-compatible WAIT &H3DA,8 est obsolète.

Exemple

'boucle principale
Do
  
  ' effectue entree utilisateur
  ' calcule_une_frame
   
  ScreenSync
  
  ' dessine_ une_ frame  
  
Loop Until Inkey <> ""



Différences de dialectes

  • Non disponible en dialecte -lang qb sauf si référencé avec l'alias __Screensync.

Différences avec QB

  • Nouveau pour FreeBASIC.
  • QBasic utilisait WAIT &H3DA,8 pour cela.
