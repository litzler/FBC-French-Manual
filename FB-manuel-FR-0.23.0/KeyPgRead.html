Read
 
Lecture des valeurs stockées avec Data.

Syntaxe

Read variable_list

Description

Lit les données stockées dans l'application avec la commande Data.

Les éléments de variable_list doivent être des types basiques, numériques, chaînes ou éléments de tableaux et de types définis par l'utilisateur.

Toutes les déclarations données ( Data ) dans le programme constituent une liste unique, après la lecture du dernier élément d'une déclaration de données, le premier élément de la déclaration de données suivante sera lu. Le programme ne doit pas tenter de lire ( Read ) après le dernier élément de données. Si c'est le cas, les résultats sont (dans tous les dialectes) indéfinis, et le programme peut se "planter" (erreur de page).

Des données constantes ne peuvent être que des types simples (numériques ou chaînes). Une chaîne lue dans une variable numérique sera évaluée par la fonction Val.

L'instruction "Restore label" oblige la lecture du premier item de données après label, permettant à l'utilisateur de choisir des sections spécifiques de données à lire.

Exemple

' Cree un tableau de 5 entiers et une chaine pour stocker les donnees.
Dim As Integer h(4)
Dim As String hs
Dim As Integer readindex

' Boucle 5 fois (pour 5 nombres... charge les donnees)
For readindex = 0 To 4

  ' Lecture d'un entier.
  Read h(readindex)

  ' Affichage de celui-ci.
  Print "Number" ; readindex ; " = " ; h(readindex)

Next readindex

' Separation.
Print

' Lecture d'une chaine.
Read hs

' Affiche celle-ci.
Print  "String = " + hs

' Attend appui sur touche.
Sleep

' Sortie programme.
End

' Block de donnees.
Data 3, 234, 4354, 23433, 87643, "Bye!"
