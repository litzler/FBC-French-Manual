__FILE_NQ__
 
Définition intrinsèque (valeur de la macro) fixée par le compilateur

Syntaxe

__FILE_NQ__

Description

Remplacé par le nom du fichier source entre guillemets où il est utilisé.

Exemple

#print __FILE_NQ__
