Option du compilateur: -m
 
Fichier principal sans extension pour indiquer le module principal

Syntaxe

-m < source file >

Paramètres

source file
Le nom sans extension du fichier source du module principal

Description

L'option du compilateur -m spécifie un point d'entrée principal pour un fichier source; l'argument est le nom d'un fichier source moins son extension. Si "-m" ou "-entry" n'est pas spécifié, le premier fichier source de la liste est donné comme point d'entrée principal. Lorsque vous utilisez l'option "-c", "-m" ou "-entry" doivent être spécifiés lors de la compilation d'un fichier source principal. Même chose que pour l'option "-entry" du compilateur.

La macro intrinsèque __FB_MAIN__ La macro intrinsèque est définie dans le module principal et pas définie dans les autres modules.
