Indexation des tableaux
 
Un indice de tableau est le numéro utilisé pour accéder à un tableau de variables créé en utilisant la commande Dim.

Description

Les exemples suivants illustrent l'utilisation d'éléments d'un tableau.

Si nous avons un tableau myArray avec des éléments de 1 à 10, rempli avec des données aléatoires:
Indice       Données
1              5
2              2
3              6
4              5
5              9
6              1
7              0
8              4
9              5
10             7

On peut accéder à chaque élément de données séparément en pointant l'indice de l'élément de tableau:
    Print myArray(5)
    

Afficher les données contenues dans le cinquième élément de myArray se traduit par l'affichage de:
	9
Pour modifier le contenu d'un tableau, utilisez-le comme n'importe quelle autre variable:
    myArray(3) = 0
    

Pour afficher le contenu de myArray(3), utilisez la commande:
    Print myArray(3)
    

ce qui affichera:
	0
Les éléments du tableau peuvent être indexés à l'aide d'une autre variable. Dans cet exemple, nous avons mis tous les les éléments de notre tableau à zéro:
    Dim a As Integer
    For a = 1 To 10
      myArray(a) = 0
    Next a
    

Pour modifier un élément de tableau aléatoire par une valeur aléatoire:
    Dim Index As Integer
    Dim Value As Integer
    index = Int(Rnd(1) * 10) + 1 'This line will simply return a random value between 1 and 10
    Value = Int(Rnd(1) * 10) + 1 'This line will do the same
    myArray(index) = Value
