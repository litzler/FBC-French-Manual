CsrLin
 
Retourne la position-ligne du curseur

Syntaxe

Declare Function CsrLin ( ) As Integer

Usage

result = CsrLin

Valeur retournée

Un Integer (entier) spécifiant la ligne courante du curseur.

Description

Retourne la ligne courante où se trouve le curseur (c'est à dire "la ligne curseur"). La première ligne en haut a le numéro 1.

Exemple

Print "The cursor is on row:"; CsrLin


Différences avec QB

  • Aucune.
