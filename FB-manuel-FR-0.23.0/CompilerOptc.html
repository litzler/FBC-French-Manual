Option du compilateur: -c
 
Compile et assemble seulement le fichier source, ne lie pas

Syntaxe

-c

Description

L'option -c spécifie que tous les fichiers source listés doivent être compilés et assemblés en fichiers objet et non liés dans un fichier exécutable (comportement par défaut).

Pour compiler les fichiers source file1.bas et file2.bas en fichier objet file1.o et apple.o, tapez:

fbc -c file1.bas file2.bas -o apple.o
