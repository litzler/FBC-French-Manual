Bin
 
Retourne une chaîne binaire (base 2) représentation d'un entier

Syntaxe

Declare Function Bin Overload ( ByVal number As Byte ) As String

Declare Function Bin ( ByVal number As Short ) As String

Declare Function Bin ( ByVal number As Integer ) As String
Declare Function Bin ( ByVal number As Integer, ByVal digits As Integer ) As String

Declare Function Bin ( ByVal number As LongInt ) As String
Declare Function Bin ( ByVal number As LongInt, ByVal digits As Integer ) As String

Usage

result = Bin[$]( number [, digits ] )

Paramètres

number
Un nombre ou une expression transformé en nombre. Un nombre en virgule flottante sera converti en LongInt.
digits
Nombre de chiffres de la chaîne renvoyée.

Valeur retournée

Une chaîne contenant la représentation binaire non-signée de number.

Description

Retourne une chaîne binaire représentation de la valeur binaire non-signée de l'entier number. Chiffres binaires de 0 à 1.

Si vous spécifiez digits > 0, la chaîne résultat aura exactement cette longueur. Elle sera tronquée ou complètée par des zéros sur la gauche, si nécessaire.

La longueur de la chaîne ne dépassera pas le nombre de bits maximum requis pour le type de number (32 pour un Integer (entier), 64 pour un LongInt (entier long)).

Si vous désirez l'opposé, par exemple convertir une chaîne binaire vers un nombre, la meilleure façon de le faire est d'ajouter "&B" au début de la chaîne, et la convertir en utilisant ValInt ou ValLng, de façon similaire à une chaîne numérique normale. Par exemple ValInt("&B101")

Exemple

Print Bin(54321)
Print Bin(54321, 5)
Print Bin(54321, 20)

produira la sortie:

1101010000110001
10001
00001101010000110001
Différences de dialecte

  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __Bin.

Différences avec QB

  • Nouveau pour Freebasic
