__FB_OPTION_ESCAPE__
 
Définition intrinsèque (valeur de la macro) fixée par le compilateur

Syntaxe

__FB_OPTION_ESCAPE__

Description

Indique si par défaut, les chaînes de littéraux sont prévues pour les caractères d'échappement quand elles ne sont pas explicitement préfixées avec $ Operator pour les chaînes non-caractères-d'échappement, ou ! Operator pour les chaînes-à-caractères-d'échappement.

La méthode par défaut pour traîter les chaînes littérales est fixée par l'utilisation de l'option -lang de la ligne de commande pendant la compilation ou l'utilisation de l'Option Escape dans le fichier source.

__FB_OPTION_ESCAPE__ retourne zéro (0) si l'option n'est pas valide. Retourne non-zéro (-1) si l'option est valide.

Exemple

#if( __FB_OPTION_ESCAPE__ <> 0 )
  #error Option Escape must Not be used With This include file
#endif
