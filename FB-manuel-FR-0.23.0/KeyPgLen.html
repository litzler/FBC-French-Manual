Len
 
Retourne la longueur d'un type de variable ou de données

Syntaxe

Declare Function Len ( ByRef variable As Any ) As Integer
Declare Function Len ( datatype ) As Integer

Usage

result = Len( variable )
ou
result = Len( DataType )

Paramètres

variable
Une variable de n'importe quel type.
datatype
Un DataType.

Valeur retournée

Retourne la taille en octets d'une variable ou d'un DataType.

Description

Len retourne la longueur en octets d'une variable ou d'un DataType.

Dans la première forme, si variable est du type String, WString ou ZString, la longueur de la chaîne, en caractères, est retournée. Sinon, la taille du type de variable est retournée.

Dans la seconde forme, si datatype est ZString ou WString, la taille d'un caractère ASCII ou Unicode est retournée, respectivement. Si datatype est String, la taille du descripteur de type est retournée.

S'il y a, à la fois, un type défini par l'utilisateur et une variable qui ont le même nom dans le même champ d'application, le type défini par l'utilisateur a la priorité sur la variable. Pour s'assurer que Len prend la variable au lieu du type défini par l'utilisateur, mettez l'argument de Len entre parenthèses pour le forcer à être considéré comme une expression. Par exemple Len((variable)).

Exemple

Print Len("hello world") 'returns "11"
Print Len(Integer) ' returns 4

Type xyz
    a As Integer
    b As Integer
End Type

Print Len(xyz) ' returns 8


Différences de dialectes

  • Len n'admet que des expressions dans le dialecte -lang qb.
  • Peut être utilisé avec les types intégrés et les types définis par l'utilisateur dans les dialectes -lang fb et -lang fblite.

Différences avec QB

  • Peut être utilisé avec les types intégrés et les types définis par l'utilisateur dans les dialectes -lang fb et -lang fblite.
  • Aucune dans le dialecte -lang qb.
