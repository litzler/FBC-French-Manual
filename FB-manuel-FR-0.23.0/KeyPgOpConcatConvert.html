Opérateur & (Concaténation de chaînes avec conversion)
 
Concatène deux chaînes, en convertissant, si nécessaire, les non-chaînes en chaînes

Syntaxe

Declare Operator & ( ByRef lhs As T, ByRef rhs As U ) As V

Usage

result = lhs & rhs

Paramètres

lhs
La chaîne de gauche à concaténer.
T
N'importe quel type de donnée standard ou défini par l'utilisateur qui peut être converti en un type de donnée standard.
rhs
La chaîne de droite à concaténer.
U
N'importe quel type de donnée standard ou défini par l'utilisateur qui peut être converti en un type de donnée standard.
V
Le type de la chaîne résultante (varie avec les opérandes).

Description

Cet opérateur concatène deux expressions. Si l'une des expressions n'est pas de type chaîne, elle est convertie en String (Chaîne) avec Str.

Si l'une des expressions est une WString, une WString est retournée, sinon une String est retournée.

Exemple

Dim As String A,C
Dim As Single B
A="The result is: "
B=124.3
C=A & B
Print C
Sleep

Affiche:
The result is: 124.3
