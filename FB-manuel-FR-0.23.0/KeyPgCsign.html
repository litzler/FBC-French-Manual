CSign
 
Convertit une expression en une valeur signée

Syntaxe

CSign ( expression )

Usage

variable = CSign ( expression )

Description

Convertit une expression non-signée en une expression signée, utile pour forcer le comportement signé de la division ou de la multiplication (y compris avec Shl et Shr).

C'est l'opposé de CUnsg.

Exemple

Dim value As UShort = 65535
Print CSign(value)  '' will print -1


Différences de dialecte

  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __Csign.
