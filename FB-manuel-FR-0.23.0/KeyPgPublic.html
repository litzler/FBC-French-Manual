Public
 
Indique une procédure ayant de liens externes.

Syntaxe

Public Sub procedure_name [Overload] [cdecl|stdcall|pascal] [Alias "external_name"] [([parameter_list])] [Constructor [priority]] [Static] [Export]
..procedure body..
End Sub

Public Function procedure_name [Overload] [cdecl|stdcall|pascal] [Alias "external_name"] [([parameter_list])] As return_type [Static] [Export]
..procedure body..
End Function

Description

Dans les définitions de procédure, Public précise qu'une procédure a un lien externe, ce qui signifie son nom est visible à des modules externes. Si Public ou Private n'est pas spécifié, une procédure est définie comme si Public avait été spécifié.

Exemple

Private Sub i_am_private
End Sub

Public Sub i_am_public
End Sub
