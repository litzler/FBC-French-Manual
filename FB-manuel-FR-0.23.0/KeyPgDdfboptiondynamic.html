__FB_OPTION_DYNAMIC__
 
Définition intrinsèque (valeur de la macro) fixée par le compilateur

Syntaxe

__FB_OPTION_DYNAMIC__

Description

__FB_OPTION_DYNAMIC__ est définie comme vraie (un négatif (-1)) si une récente déclaration Option Dynamic ou métacommande '$Dynamic a été émise. Sinon, elle est définie comme zéro.

Exemple

#if __FB_OPTION_DYNAMIC__ <> 0
#error This module must Not use Option Dynamic
#endif


Différences avec QB
