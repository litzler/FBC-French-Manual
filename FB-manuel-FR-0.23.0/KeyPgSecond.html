Second
 
Récupère les secondes de l'heure à partir d'une Date Serial

Syntaxe

Declare Function Second ( ByVal date_serial As Double ) As Integer

Usage

#include "vbcompat.bi"
result = Second( date_serial )

Paramètres

date_serial
la "date serial"

Valeur retournée

Retourne les secondes à partir d'une variable contenant une date sous forme de Date Serial.

Description


Le compilateur ne reconnaît cette fonction que si vbcompat.bi est inclus.

Exemple

#include "vbcompat.bi"

Dim ds As Double = DateSerial(2005, 11, 28) + TimeSerial(7, 30, 50)

Print Format(ds, "yyyy/mm/dd hh:mm:ss "); Second(ds)


Différences avec QB

  • N'existe pas dans QB. Cette fonction est apparue dans PDS et VBDOS
