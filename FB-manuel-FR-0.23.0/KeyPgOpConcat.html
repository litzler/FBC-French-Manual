Opérateur + (Concaténation de chaînes)
 
Concatène deux chaînes

Syntaxe

Declare Operator + ( ByRef lhs As String, ByRef rhs As String ) As String
Declare Operator + ( ByRef lhs As ZString, ByRef rhs As ZString ) As ZString
Declare Operator + ( ByRef lhs As WString, ByRef rhs As WString ) As WString

Usage

result = lhs + rhs

Paramètres

lhs
La chaîne de gauche à concaténer.
rhs
La chaîne de droite à concaténer.

Description

Cet opérateur concatène deux chaînes. Contrairement à Opérateur & (Concaténation de chaînes avec conversion) les deux expressions doivent être des chaînes, et ne seront pas converties (en fait, toute tentative de concaténer une chaîne avec une non-chaîne ou deux non-chaînes se traduira par une erreur de type, sauf si la surcharge d'opérateur est utilisée dans un type défini par l'utilisateur).

Exemple

Dim As String a = "Hello, ", b = "World!"
Dim As String c
c = a + b
Print c

Affiche:
Hello, World!
