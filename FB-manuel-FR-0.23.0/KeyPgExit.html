Exit
 
Déclaration de contrôle de flux pour sortir d'un bloc composé d'instructions

Syntaxe

Exit {Do | For | While | Select }
Exit {Sub | Function | Operator | Property }

Exit {Do [, Do [ , ...] ] |
For [, For [ , ...] ] |
While [, While, [...] ] |
Select [, Select [ , ...] ] }


Description

Quitte un bloc de code comme Sub, Function, Do...Loop, For...Next, While...Wend, ou Select Case. L'exécution ignore le reste du bloc et continue sur la ligne après sa fin.

Là où il y a plusieurs blocs Do / For / While / Select imbriqués, exit fera passer à la fin du bloc de ce type, le plus intérieur. Vous pouvez aller directement à la fin de plusieurs blocs du même type en donnant en même temps plusieurs mots, séparés par des virgules.
Par exemple: Exit While, While

Exemple

'e.g. the print command will not be seen

Do
    Exit Do ' Exit the Do...Loop and continues to run the code after Loop
    Print "I will never be shown"
Loop


Dim As Integer i, j
For i = 1 To 10
    
    For j = 1 To 10
        
        Exit For, For
        
    Next j
    
    Print "I will never be shown"
    
Next i


Différences avec QB

  • EXIT WHILE et EXIT SELECT sont nouveaux pour FreeBASIC.
