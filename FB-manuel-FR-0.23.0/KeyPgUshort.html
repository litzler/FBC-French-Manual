UShort
 
Type de données standard: nombre de 16 bits non-signé

Syntaxe

Dim variable As UShort

Description

Nombre entier de 16 bits non-signé. Peut contenir des valeurs de 0 à 65535.

Exemple

  Dim x As UShort = 0
  Dim y As UShort = &HFFFF
  Print "UShort Range = "; x; " to "; y

Affichage:
UShort Range = 0 to 65535


Différences de dialecte

  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __Ushort.

Différences avec QB

  • Nouveau pour FreeBASIC
