Option Gosub
 
Active le support de GoSub et On Gosub.

Syntaxe

Option Gosub

Description

Option Gosub active le support de GoSub et Return (depuis gosub).

Parce que Return pourrait signifier retour-de-gosub ou retour-de-procédure, Option Gosub et Option Nogosub peuvent être utilisés pour activer et désactiver le support de GoSub. Quand le support de GoSub est désactivé, Return est alors reconnu comme retour-de-procédure.

Exemple

'' Compile with the "-lang fblite" compiler switch

#lang "fblite"

'' turn on gosub support
Option GoSub

GoSub there
backagain:
    Print "backagain"
    End

there:
    Print "there"
    Return


Différences de dialecte

  • Disponible seulement dans les dialectes -lang fblite et -lang qb.

Différences avec QB

  • Nouveau pour FreeBASIC
