DateValue
 
Retourne une Date Serial à partir d'une chaîne

Syntaxe

Declare Function DateValue ( ByRef date_string As String ) As Double

Usage

#include "vbcompat.bi"
result = DateValue( date_string )

Paramètres

date_string
la chaîne à convertir en "Date Serial"

Valeur retournée

Retourne une Date Serial à partir d'une chaîne.

Description

La chaîne de la date doit être dans le format défini dans les paramètres régionaux du système d'exploitation.

DateValue( Date() ) ne fonctionnera correctement que si les paramètres régionaux spécifiés ont le même format court de date utilisé par QB (mm-dd-yyyy). Pensez à utiliser la fonction Now dans l'expression Fix(Now()) pour obtenir la date courante sous forme de "date serial".

Le compilateur ne reconnaît cette fonction que si vbcompat.bi ou datetime.bi est inclus.

Exemple

#include "vbcompat.bi"

Dim As Integer v1, v2
Dim As String  s1, s2

Print "Enter first date: ";
Line Input s1

If IsDate( s1 ) = 0 Then
  Print "not a date"
  End
End If

Print "Enter second date: ";
Line Input s2

If IsDate( s2 ) = 0 Then
  Print "not a date"
  End
End If

'' convert the strings to date serials
v1 = DateValue( s1 )
v2 = DateValue( s2 )

Print "Number of days between dates is " & Abs( v2 - v1 )


Différences avec QB

  • N'existe pas dans QB. Cette fonction est apparue dans PDS et VBDOS
