__FB_VER_PATCH__
 
Définition intrinsèque (valeur de la macro) fixée par le compilateur

Syntaxe

__FB_VER_PATCH__

Description

__FB_VER_PATCH__ retournera le patch/subversion/numéro de révision de la version de FreeBasic actuellement utilisé. Pour FreeBASIC 0.18, par exemple, il y a eu les subversions 1, 2, 3, 4, 5 et 6.

Exemple

Dim fbMajorVersion As Integer
Dim fbMinorVersion As Integer
Dim fbPatchVersion As Integer

fbMajorVersion = __FB_VER_MAJOR__
fbMinorVersion = __FB_VER_MINOR__
fbPatchVersion = __FB_VER_PATCH__

Print "Welcome to FreeBASIC ";fbMajorVersion;".";fbMinorVersion;" , Revision ";fbPatchVersion
