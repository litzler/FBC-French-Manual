Byte
 
Type de données standard: 8 bits signé

Syntaxe

Dim variable As Byte

Description

type de données: nombre entier 8-bits signé. Peut contenir une valeur dans la plage de -128 à 127.

Exemple

  Dim bytevar As Byte
  bytevar = 100
  Print "bytevar= ", bytevar


Dim x As Byte = CByte(&H80)
Dim y As Byte = CByte(&H7F)
Print "Byte Range = "; x; " to "; y

Affichage:
Byte Range = -128 to  127

Différences de dialecte

  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __Byte.

Différences avec QB

  • Nouveau pour FreeBASIC
