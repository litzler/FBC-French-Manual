defined
 
Fonction du préprocesseur pour tester si un symbole a déjà été défini

Syntaxe

defined (symbol_name)

Paramètres

symbol_name
Nom du symbole à tester

Valeur retournée

Retourne non-zéro (-1) si le symbole a été défini, autrement retourne zéro (0).

Description

Etant donné le nom du symbole, la fonction préprocesseur defined() retourne vrai (true) si le symbole a été défini - ou faux (false) si le symbole est inconnu.

Utilisé principalement avec #if.

Similaire à #ifdef sauf qu'il permet de faire plus d'une vérification en raison de sa flexibilité.

Exemple

'e.g. - which symbols are defined out of a, b, c, and d ?

Const a = 300
#define b 12
Dim c As Single

#if defined(a)
 Print "a is defined"
#endif
#if defined(b)
 Print "b is defined"
#endif
#if defined(c)
 Print "c is defined"
#endif
#if defined(d)
 Print "d is defined"
#endif
