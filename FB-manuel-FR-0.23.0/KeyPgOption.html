Option()
 
Spécifie des attributs et/ou des caractéristiques additionnels de symboles.

Syntaxe

Option( "literal-text" )

Paramètres

literal-text
Le texte littéral spécifiant l'option. Voir description.

Description

Option() permet au programmeur de préciser des attributs ou caractéristiques supplémentaires. Dans cette syntaxe la chaîne doit être encadrée par des guillemets ("). Les options inconnues seront ignorées.

Option peut également être utilisé comme instruction pour préciser d'autres options de compilation. Voir Commutateurs du compilateur.

Les options individuelles sont expliquées ci-dessous.

SSE

Option("SSE") indique qu'une valeur à virgule flottante (Single ou Double) retournée par une fonction est chargée dans le registre xmm0. Option("Sse") est ignorée tant que le code source n'est pas compilé avec l'option de ligne de commande -fpu SSE. Cette option peut être utilisée immédiatement après le retour dans une déclaration de fonction ou une définition de fonction. Cette option est une optimisation et n'est pas requise pour compiler des programmes utilisant l'option de ligne de commande -fpu SSE.

Declare Function ValueInXmm0 () As Double Option("sse")


FPU

Option("FPU") indique qu'une valeur à virgule flottante (Single ou Double) retournée par une fonction est chargée dans le registre st(0). Cette option peut être utilisée immédiatement après le retour dans une déclaration de fonction ou une définition de fonction.

Declare Function ValueInStZero () As Double Option("fpu")
