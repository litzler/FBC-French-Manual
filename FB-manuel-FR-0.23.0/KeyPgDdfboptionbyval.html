__FB_OPTION_BYVAL__
 
Définition intrinsèque (valeur de la macro) fixée par le compilateur

Syntaxe

__FB_OPTION_BYVAL__

Description

Indique si les paramètres d'une Function ou d'une Sub sont passés par référence, comme avec ByRef, ou par valeur comme avec ByVal, par défaut, quand le spécificateur by value (par valeur) / by reference (par référence) n'est pas explicitement établi.

__FB_OPTION_BYVAL__ est fixé à non-zéro (-1) si par les paramètres par défaut sont passés de valeur et zéro (0) si les paramètres par défaut sont passés par référence.

La valeur par défaut pour passer des paramètres par référence ou par valeur est déterminée par l'option -lang de la ligne de commande utilisée pendant la compilaton ou l'utilisation de Option ByVal dans le fichier source.

Exemple

#if( __FB_OPTION_BYVAL__ <> 0 )
  #error Option ByVal must Not be used With This source
#endif
