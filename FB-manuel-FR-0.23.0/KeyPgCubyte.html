CUByte
 
Convertit une expression numérique ou une chaîne vers un octet non-signé (UByte)

Syntaxe

Declare Function CUByte ( ByVal expression As datatype ) As UByte

Type typename
Declare Operator Cast ( ) As UByte
End Type

Usage

result = CUByte( numeric expression )
result = CUByte( string expression )
result = CUByte( user defined type )

Paramètres

expression
Une expression numérique, chaîne ou pointeur à transformer (cast) en valeur UByte
datatype
Tout type de données numérique, chaîne ou pointeur.
typename
Un type défini par l'utilisateur.

Valeur retournée

Une valeur UByte.

Description

La fonction CUByte arrondit la partie entière et retourne une valeur UByte 8 bits. La fonction ne vérifie pas s'il y a un dépassement, alors assurez-vous de ne pas avoir une valeur inférieure à 0 / supérieure à 255. Le nom peut être expliqué par 'Convert to Unsigned Byte'.

Si l'argument est une expression chaîne, il est converti en valeur numérique à l'aide de ValUInt.

Exemple

' Using the CUBYTE function to convert a numeric value

'Create an UNSIGNED BYTE variable
Dim numeric_value As UByte

'Convert a numeric value
numeric_value = CUByte(123.55)

'Print the result, should return 124
Print numeric_value
Sleep
