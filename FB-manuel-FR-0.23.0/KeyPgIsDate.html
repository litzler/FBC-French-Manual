IsDate
 
Teste si une chaîne est convertible en Date Serial

Syntaxe

Declare Function IsDate ( ByRef stringdate As Const String ) As Integer

Usage

#include "vbcompat.bi"
result = IsDate( stringdate )

Paramètres

stringdate
la chaîne à tester

Valeur retournée

Retourne une valeur non-nulle (-1) si la chaîne-date est convertible en Date Serial, sinon retourne zéro (0).

Description

Les chaines-date doivent être dans le format enregistré dans les paramètres régionaux du système d'exploitation pour être considérées comme des dates valides.

IsDate(Date) retournera une valeur non-nulle (-1) seulement si les paramètres régionaux spécifient le même format de date que celui utilisé par QB.

Le compilateur ne reconnaît cette fonction que si vbcompat.bi ou datetime.bi est inclus.

Exemple

#include "vbcompat.bi"

Dim s As String, d As Integer

Do
  Print
  Print "Enter a date: "

  Line Input s

  If s = "" Then Exit Do

  If IsDate( s ) = 0 Then
    Print "'"; s; "' is not a valid date"
  Else
    d = DateValue( s )
    Print "year  = "; Year( d )
    Print "month = "; Month( d )
    Print "day   = "; Day( d )
  End If

Loop
