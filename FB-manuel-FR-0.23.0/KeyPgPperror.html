#Error
 
Directive préprocesseur de diagnostique

Syntaxe

#error error_text

Paramètres

error_text
Le message affiché

Description

#error arrête la compilation et affiche error_text quand le compilateur détecte une erreur.

Ce mot-clef doit être encadré par #if <condition> ...#endif ainsi le compilateur peut atteindre #error seulement si <condition> est vérifiée.

Exemple

#define c 1

#if c = 1
  #error Bad value of c 
#endif
