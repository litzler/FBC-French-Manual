__FB_LANG__
 
Définition intrinsèque (valeur de la macro) fixée par le compilateur

Syntaxe

__FB_LANG__

Description

__FB_LANG__ indique la langue dans laquelle l'option de compatibilité a été fixée au moment de la compilation d'un module. Par défaut __FB_LANG__ sera réglé sur "fb". L'option de compatibilité langue peut être changée en utilisant une (ou plusieurs) des méthodes suivantes:
    • l'option -lang de la ligne de commande
    • l'option -forcelang de la ligne de commande
    • l'instruction #lang
    • la méta-commande $Lang

Renvoie une chaîne en minuscules avec l'une des valeurs suivantes:
valeurdescription
''fb''Compatibilité FreeBASIC (valeur par défaut)
''qb''Compatibilité QBASIC
''fblite''Compatibilité langage FreeBASIC, avec plus de codage en style compatible-QBASIC
''deprecated''Compatibilité FBC version 0.16


__FB_LANG__ est toujours défini.

Exemple

'' Set option explicit always on

#ifdef __FB_LANG__
  #if __FB_LANG__ <> "fb"
    Option Explicit
  #endif
#else
  '' Older version - before lang fb
  Option Explicit
#endif


Différences avec QB

  • Nouveau pour FreeBASIC
