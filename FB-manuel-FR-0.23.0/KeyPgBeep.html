Beep
 
Émet un signal sonore.

Syntaxe

Declare Sub Beep ( )

Usage

Beep

Description

Beep appelle le système pour qu'il émette un bruit de bip. Notez que cela ne fonctionne pas sur certaines plates-formes. Cette commande n'est pas efficace car il n'y a aucun moyen de spécifier la fréquence et la durée, vous pouvez l'éviter en faveur d'autres/meilleures solutions, par exemple: Son par Yetifoot.

Exemple

Beep


Différences avec QB

  • Dans QB, c'était un bruit de tonalité unique généré par le haut-parleur PC. Maintenant, cela pourrait ne pas être le cas.
