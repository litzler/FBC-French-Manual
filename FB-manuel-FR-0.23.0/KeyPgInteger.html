Integer
 
Type de données standard: 32 bits signé

Syntaxe

Dim variable As Integer

Description

Type de données: nombre entier signé de 32 bits. Peut contenir des valeurs de -2147483648 à 2147483647.

Exemple

  Dim x As Integer = &H80000000
  Dim y As Integer = &H7FFFFFFF
  Print "Integer Range = "; x; " to "; y

Affichage:
Integer Range = -2147483648 to  2147483647

Différences de dialecte

  • Dans les dialectes -lang fb et -lang fblite, le type de donnée Integer a 32 bits.
  • Dans les dialecte -lang qb, le type de donnée Integer a 16 bits.

Différences avec QB

  • Le type INTEGER a une largeur de 16 bits dans QB.
  • Aucune, si compilation dans le dialecte -lang qb.
