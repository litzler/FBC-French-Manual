__FB_OPENBSD__
 
Définition intrinsèque fixée par le compilateur

Syntaxe

__FB_OPENBSD__

Description

Définition sans valeur créée au moment de la compilation dans la version OpenBSD du compilateur ou quand l'option -target openbsd de la commande en ligne est utilisée. Peut être utilisé pour compiler des parties du programme si la cible est OpenBSD.

Exemple

#ifdef __FB_OPENBSD__
  '...instructions only for OpenBSD...
#else
  '...instructions not for OpenBSD...
#endif 


Différences avec QB
