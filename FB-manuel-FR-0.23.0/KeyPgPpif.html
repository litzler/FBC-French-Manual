#if
 
Directive conditionnelle du préprocesseur

Syntaxe

#if (expression)
' Déclarations incluses sous condition
#endif

Description

Déclarations incluses sous condition au moment de la compilation.

Les déclarations contenues dans le bloc #if / #endif sont incluses si expression est évaluée comme Vrai (True) (non-zéro) et exclues (ignorées) si expression est évaluée comme Faux (False) (0).

Cette directive conditionnelle diffère de la déclaration conditionnelle If car #if est évalué au moment de la compilation alors que If est évalué à l'exécution du programme.

Exemple

#define DEBUG_LEVEL 1
#if (DEBUG_LEVEL >= 2)
  ' This line is not compiled since the expression is False
  Print "Starting application"
#endif
