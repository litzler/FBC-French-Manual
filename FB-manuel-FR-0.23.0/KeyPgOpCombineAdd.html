Opérateur += (Addition et Affectation)
 
Ajoute et attribue une valeur à une variable

Syntaxe

Declare Operator += ( ByRef lhs As T1, ByRef rhs As T2 )

Declare Operator += ( ByRef lhs As T Ptr, ByRef rhs As Integer )

Declare Operator += ( ByRef lhs As String, ByRef rhs As String )
Declare Operator += ( ByRef lhs As WString, ByRef rhs As WString )

Usage

lhs += rhs

Paramètres

lhs
La variable à affecter.
T1
Tout type numérique.
rhs
La valeur à ajouter à lhs.
T2
Tout type numérique.
T
Tout type de données.

Description

Cet opérateur ajoute et attribue une valeur à une variable. Il est fonctionnellement équivalent à:

lhs = lhs + rhs

Pour les types numériques, l'expression de droite (rhs) sera convertie dans le type de gauche (T1).

Pour les types chaîne, cet opérateur est fonctionnellement équivalent à Opérateur &= (Concaténation et Affectation).

Cet opérateur peut être surchargé pour des types définis par l'utilisateur.

Exemple

Dim n As Double
n = 6
n += 1
Print n
Sleep

Affiche:
7

Différences de dialecte

  • Dans le dialecte -lang qb, cet opérateur ne peut pas être surchargé.
