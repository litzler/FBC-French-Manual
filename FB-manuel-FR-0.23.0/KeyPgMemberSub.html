Sub (Membre)
 
Déclare ou définit un membre de procédure.

Syntaxe

{ Type | Class | Union } typename
Declare [ Static ] Sub fieldname [calling convention specifier] [ Alias external_name ] ( [ parameters ] ) [ Static ]
End { Type | Class | Union }

Sub typename.fieldname ( [ parameters ] )
statements
End Sub

Paramètres

typename
nom de Type, Class, ou Union
fieldname
nom de la procédure
external_name
nom du champ tel qu'il est vu quand lié extérieurement
parameters
les paramètres à passer à la procèdure
spécificateur de convention d'appel
peut être: cdecl, stdcall ou pascal

Description

Les membres de Sub sont accessibles avec "Operator . (Accès de membre)" ou "Operator -> (Pointeur vers Accès membre)" pour appeler des membres de procédure et peuvent éventuellement accepter des paramètres comme ByVal ou ByRef. typename peut être surchargé sans l'utilisation explicite du mot-clef Overload.

typename est le nom du type sous lequel la méthode Sub a été déclarée et définie. La résolution de nom de typename suit les mêmes règles que les procèdures quand elles sont utilisées dans un Namespace.

Le paramètre caché This qui a le même type que typename est passé aux membres non-statics des procédures. This est utilisé pour accéder aux champs du Type, de la Class, ou de l'Union.

Un Static (Membre) peut être déclaré en utilisant le spécificateur Static. Un Const (Membre) peut être déclaré en utilisant le spécificateur Const.

Exemple

Type Statistics
  count As Single
  sum As Single
  Declare Sub AddValue( ByVal x As Single )
  Declare Sub ShowResults( )
End Type

Sub Statistics.AddValue( ByVal x As Single )
  count += 1
  sum += x
End Sub

Sub Statistics.ShowResults( )
  Print "Number of Values = "; count
  Print "Average          = ";
  If( count > 0 ) Then
    Print sum / count
  Else
    Print "N/A"
  End If
End Sub

Dim stats As Statistics

stats.AddValue 17.5
stats.AddValue 20.1
stats.AddValue 22.3
stats.AddValue 16.9

stats.ShowResults

Affichage:
Number of Values =  4
Average          =  19.2

Différences de dialecte

  • Disponible seulement dans le dialecte -lang fb.
