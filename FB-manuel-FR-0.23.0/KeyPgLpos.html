Lpos
 
Retourne le nombre de caractères envoyés au port d'imprimante depuis la dernière déclaration LPrint.

Syntaxe

Declare Function Lpos ( ByVal printer As Integer ) As Integer

Usage

result = LPOS(printer)

Paramètres

printer
0, 1, 2 ou 3. Représente le port imprimante (LPT#)

Valeur retournée

Retourne le nombre de caractères envoyés.

Description

Utilisé pour déterminer, à partir du dernier LPrint, combien de caractères ont été envoyés au port d'imprimante.

Exemple

' compile with -lang fblite or qb

#lang "fblite"

Dim test As String = "LPrint Example test"

Print "Sending '" + test + "' to LPT1 (default)"
LPrint test
Print "LPT1 last recieved " + Str(LPOS(1)) + " characters"
Print "String sent was " + Str(Len(test)) + " characters long"

Sleep
