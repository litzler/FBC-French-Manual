Pos
 
Retourne la position horizontale (de gauche à droite) du curseur texte

Syntaxe

Declare Function Pos Overload ( ) As Integer
Declare Function Pos ( ByVal dummy As Integer ) As Integer

Usage

result = Pos[ ( dummy ) ]

Paramètres

dummy
Un paramètre inutilisé pour conserver la compatibilité descendante avec QBASIC.

Valeur retournée

Retourne la position horizontale du curseur texte.

Description

Retourne la position horizontale (de gauche à droite) du curseur texte. La colonne la plus à gauche a le numéro 1.

Exemple

Dim As Integer p

'' print starting column position
p = Pos()
Print "position: "; p

'' print a string, without a new-line
Print "ABCDEF";

'' print new column position:
p = Pos()
Print: Print "position: "; p
Print

''position changes after each Print:
Print "Column numbers: "
Print Pos(), Pos(), Pos(), Pos(), Pos()


Différences avec QB

  • Le paramètre dummy n'était pas facultatif dans QBASIC.
