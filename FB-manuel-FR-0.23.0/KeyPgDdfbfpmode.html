__Fb_Fpmode__
 
Définition intrinsèque fixée par le compilateur

Syntaxe

__FB_FPMODE__

Description

Défini comme "fast" = "rapide" si l'arithmétique rapide SSE est activée, ou "precise" = "précis" autrement.

Exemple

#if __FB_FPMODE__ = "fast"
  ' ... instructions for using fast-mode math ...
#else
  ' ... instructions for using normal math ...
#endif
