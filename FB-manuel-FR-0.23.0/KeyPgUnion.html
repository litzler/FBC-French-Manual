Union
 
Déclare un type union défini par l'utilisateur.

Syntaxe

Union typename
fieldname as datatype
Declare member function declaration ...
...
End Union

Paramètres

typename
Nom de l'Union
fieldname
Nom d'un membre champ de données
member function declaration
N'importe quelle fonction membre supportée

Description

Une Union est similaire à un Type structure, sauf que les éléments d'une Union occupent le même espace dans la mémoire. La taille de l'Union est celle du plus grand élément de données. Un élément de données peut être d'un Type sans nom. Comme ils occupent le même espace, un seul élément peut être utilisé.

Des Union sans nom peuvent être imbriquées dans un type structure et des types sans nom peuvent être imbriqués à l'intérieur d'une Union. Voir Exemple.

Les Union supportent des fonctions membres inclus Constructor, Destructor, Function, Operator, Property et Sub. Tous les membres d'une Union sont publics et le contrôle d'accès n'est pas supporté.

Une Union peut être passée comme un type défini par l'utilisateur pour surcharger des fonctions opérateurs.

Exemple

'define our union
Union AUnion
    a As UByte
    b As Integer
End Union
'define a composite type
Type CompType
    s As String * 20
    ui As Byte 'Flag to tell us what to use in union.
    Union 
        au As UByte
        bu As Integer
    End Union
End Type

'Flags to let us know what to use in union.
'You can only use a single element of a union.
Const IsInteger = 1
Const IsUByte = 2

Dim MyUnion As AUnion
Dim MyComposite As CompType

'Can only set one value in union
MyUnion.a = 128

MyComposite.s = "Type + Union"
MyComposite.ui = IsInteger 'Tells us this is an integer union
MyComposite.bu = 1500

Print "Union: ";MyUnion.a

Print "Composite: ";
If MyComposite.ui = IsInteger Then
    Print MyComposite.bu
ElseIf MyComposite.ui = IsUByte Then
    Print MyComposite.au
Else
    Print "Unknown type."
End If

Différences de dialecte

  • Les fonctionnalités relatives aux objets telles que les fonctions déclarées à l'intérieur de blocs Union ne sont supportées que dans le dialecte -lang fb.
  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __Union.
