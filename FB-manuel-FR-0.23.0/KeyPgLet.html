Let
 
Indique l'opérateur d'affectation.

Syntaxe

Let variable = value
ou
Let( variable1 [, variable2 [, ... ]] ) = udt
ou
Operator typename.Let ( [ ByRef | ByVal ] rhs As datatype )
statements
end operator

Description

Commande ayant pour but d'aider le programmeur à distinguer une instruction d'affectation (par exemple, Let a = 1) d'un test d'égalité (par exemple, If a = 1 then ...). Comme le compilateur ne l'exige pas, il est souvent omis.

Let peut être utilisé comme un opérateur de gauche pour affecter les membres d'un type défini par l'utilisateur à plusieurs variables. Voir Operator Let() (Affectation)

Let est utilisé avec un opérateur surchargé pour désigner l'opérateur d'affectation. Voir Operator Let (Affectation)

Exemple

'' Compile with -lang fblite or qb

#lang "fblite"

' these two lines have the same effect:
Let x = 100
x = 100


Différences de dialecte

  • L'utilisation de Let pour indiquer une instruction d'affectation (Let variable = expr) n'existe pas dans le dialecte -lang fb.
  • L'affectation multi-variables définie par l'utilisateur Let n'est disponible que dans le dialecte -lang fb.
  • La surcharge des opérateurs n'est pas disponible dans les dialectes -lang qb et -lang fblite.

Différences avec QB

  • Aucune dans le dialecte -lang fb.
  • L'opérateur Let est nouveau pour FreeBASIC.
  • L'affectation multi-variables définie par l'utilisateur est nouvelle pour FreeBASIC.
