__TIME__
 
Définition intrinsèque (valeur de la macro) fixée par le compilateur

Syntaxe

__TIME__

Description

Remplacé, où il est utilisé, par l'heure de compilation, dans le format 24 heures.

Exemple

Print "CompileTime: ";__TIME__

Compile Time: 11:42:57
