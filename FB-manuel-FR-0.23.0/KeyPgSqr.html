Sqr
 
Retourne la racine carrée d'un nombre

Syntaxe

Declare Function Sqr ( ByVal number As Double ) As Double

Usage

result = Sqr( number )

Paramètres

number
le nombre (supérieur ou égal à zéro)

Valeur retournée

Retourne la racine carrée de l'argument number.

Si number égale zéro, Sqr retourne zéro.

Si number inférieur à zéro, Sqr retourne une valeur spéciale signifiant "non défini", affichée "NaN" ou "IND", le texte exact dépendant de la plate-forme de travail.

Description

C'est identique à l'élévation de l'argument number à la puissance un demi: y = x ^ (1/2) . L'argument obligatoire number peut ê tre n'importe expression numérique valide supérieure ou égale à zéro.

Exemple

'' Example of sqr function: Pythagorean theorem 
Dim As Single a , b

Print "Pythagorean theorem , right triangle"
Print
Input "Please enter one leg side length: ", a
Input "Please enter the other leg side length: ", b
Print 
Print "The hypotenuse has a length of: " ; Sqr ( a * a + b * b )

Sleep


Affichera:
Pythagorean theorem , right triangle

Please enter one leg side length: 3
Please enter the other leg side length: 4

The hypotenuse has a length of: 5
