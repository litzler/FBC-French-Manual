Seek (Fonction)
 
Obtient la position de la prochaine opération de lecture/écriture pour un fichier ou un périphérique

Syntaxe

Declare Function Seek ( ByVal filenum As Integer ) As LongInt

Paramètres

filenum
numéro de fichier d'un fichier ouvert

Valeur retournée

La position à laquelle la prochaine lecture ou écriture sur un fichier va se produire.

Description

La position est donnée en enregistrements si le fichier a été ouvert en mode d'accès Random en octets dans les autres cas. La position est basée sur 1 -- le 1er enregistrement est à la position 1.

La fonction Seek est utilisée pour déterminer la position de la prochaine opération de lecture ou d'écriture.

Exemple

Dim f As Integer, position As Integer

f = FreeFile
Open "file.ext" For Binary As #f

position = Seek(f)

Close #f
