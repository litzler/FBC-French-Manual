Stick
 
Lit la position des axes à partir de périphériques de jeu

Syntaxe

Declare Function Stick ( ByVal axis As Integer ) As Integer

Usage

result = Stick( axis )

Paramètres

axis
le numéro de l'axe dont on recherche la position

Valeur retournée

Renvoie un nombre entre 1 et 200 pour l'axis spécifié, sinon zéro (0), si le périphérique n'est pas lié.

Description

Stick permettra de récupérer la position de l'axe pour le premier et second axes des premier et deuxième périphériques de jeu. axis doit être un nombre entre 0 et 3 ont la signification suivante:

AxesRetourne
0Position X du dispositif de jeu A
1Position Y du dispositif de jeu A quand STICK(0) a été appelé
2Position X du dispositif de jeu B lorsque STICK (0) a été appelé
3Position Y du dispositif de jeu B lorsque STICK (0) a été appelé


Stick(0) doit d'abord être appelé en premier pour obtenir les positions des autres axes.

Exemple

'' Compile with -lang qb

'$lang: "qb"

Screen 12

Do
    Locate 1, 1
    Print "Joystick A-X position : "; Stick(0); "   "
    Print "Joystick A-Y position : "; Stick(1); "   "
    Print "Joystick B-X position : "; Stick(2); "   "
    Print "Joystick B-Y position : "; Stick(3); "   "
    Print
    Print "Button A1 was pressed : "; Strig(0); "  "
    Print "Button A1 is pressed  : "; Strig(1); "  "
    Print "Button B1 was pressed : "; Strig(2); "  "
    Print "Button B1 is pressed  : "; Strig(3); "  "
    Print "Button A2 was pressed : "; Strig(4); "  "
    Print "Button A2 is pressed  : "; Strig(5); "  "
    Print "Button B2 was pressed : "; Strig(6); "  "
    Print "Button B2 is pressed  : "; Strig(7); "  "
    Print
    Print "Press ESC to Quit"

    If Inkey$ = Chr$(27) Then
        Exit Do
    End If

    Sleep 1

Loop
