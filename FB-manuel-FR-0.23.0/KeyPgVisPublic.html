Public: (Contrôle d'accès)
 
Spécifie un contrôle d'accès public aux membres de Type or Class

Syntaxe

Type typename
Public:
member declarations
End Type

Paramètres

typename
nom de Type ou Class
déclarations des membres
déclarations pour les champs, fonctions ou énumérations

Description

Public: indique que les member declarations qui suivent ont un accès public. Des membres public sont accessibles pour toute utilisation de Type ou Class.

Les member declarations qui suivent Public: sont publics tant qu'aucun contrôle d'accès différent n'est spécifié, comme Private: ou Protected:

Les membres d'une déclaration Type sont Public: par défaut si aucun contrôle d'accès différent n'est spécifié.

Exemple

Type testing
  Private:
    nome As String
  Public:
    number As Integer
  Declare Sub setNome( ByRef newnome As String )
End Type

Sub testing.setnome( ByRef newnome As String )
  this.nome = newnome 
End Sub

Dim As testing myVariable

'' We can access these members anywhere since
'' they're public
myVariable.number = 69 ''
myVariable.setNome( "FreeBASIC" )
