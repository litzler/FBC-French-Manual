Opérateur = (égal)
 
Compare deux expressions pour l'égalité

Usage

result = lhs = rhs

Paramètres

lhs
L'expression côté gauche à comparer.
rhs
L'expression côté droit à comparer.
T
N'importe quel type de pointeur.

Valeur retournée

Retourne une valeur négative (-1) si les deux expressions sont égales ou zéro (0) si inégales.

Description

L'Opérateur = (Egalité) est un opérateur binaire qui compare deux expressions pour l'égalité et retourne le résultat - une valeur booléenne sous forme d'un Entier (Integer): négatif (-1) pour vrai et zéro (0) pour faux. Les arguments ne sont en aucun cas modifiés.

Cet opérateur peut être surchargé pour accepter aussi bien des types définis par l'utilisateur.

L'Opérateur = (Egalité) ne doit pas être confondu avec des initialisations ou des affectations, qui elles aussi utilisent aussi le symbole "=".

Exemple


Dim i As Integer = 0    '' initialization: initialise i with a value of 0
i = 420                 '' assignment: assign to i the value of 420

If (i = 69) Then        '' equation: compare the equality of the value of i and 69
    Print "serious error: i should equal 420"
    End -1
End If

L'Opérateur <> (Inégalité) est complémentaire à l'Opérateur = (Egalité) et est fonctionnellement identique si combiné avec l'Opérateur Not (Bit-wise Complement).

   If (420 = 420) Then Print "(420 = 420) is true."
   If Not (69 <> 69) Then Print "not (69 <> 69) is true."


Différences de dialectes

  • Dans le dialecte -lang qb, cet opérateur ne peut pas être surchargé.
