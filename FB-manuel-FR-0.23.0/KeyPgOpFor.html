Opérateur For (Itération)
 
Déclare ou définit les opérateurs utilisés par une boucle For...Next avec des variables de type défini par l'utilisateur

Syntaxe

{ Type | Class | Union } typename
Declare Operator For ()
Declare Operator For ( [ ByRef | ByVal ] stp As typename )
...
End { Type | Class | Union }

Usage

For iterator [ As typename ] = start_value To end_value [ Step step_value ]
[ ...statements... ]
Next

Paramètres

typename
nom du Type, de la Class ou de l'Union
stp, step_value
un objet typename utilisé comme valeur pour l'incrémentation
iterator
un objet typename utilisé comme itérateur
end_value
un objet typename utilisé comme valeur déterminant la fin de la boule
start_value
un objet typename utilisé pour la copie de construction ou à assigner à l'itérateur initial

Description

Opérateur For, Opérateur Next et Opérateur Step peuvent être surchargés dans des définitions de type par l'utilisateur pour allouer des objets de ce type comme itérateur ou valeur d'incrémentation, dans les boucles For...Next.

Opérateur For est appelé immédiatement après la copie de construction ou assigné à l'objet itérateur et permet à l'objet d'effectuer toute initialisation supplémentaire nécessaire à la préparation de la boucle.

La première version de Opérateur For est utilisée si aucune valeur d'incrémentation n'est donné dans la déclaration For...Next. Si une valeur d'incrémentation est donnée, la seconde version est utilisée et une valeur d'incrémentation est passée.
