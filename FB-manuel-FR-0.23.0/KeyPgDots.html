... (Ellipse - suggestion)
 
Utilisé à la place d'un paramètre de procédure pour passer un nombre variable d'arguments, ou comme limite supérieure dans une déclaration de tableau pour indiquer que le nombre d'éléments sera déterminé par l'initialiseur.

Syntaxe

Declare { Sub | Function } proc_name cdecl ( param As datatype, ... )

Dim array_symbol ([lbound To] ...) [As datatype] => { expression_list }

#define identifier( [ parameters, ] variadic_parameter... ) macro_text


Description

Les points de suspension (trois points, ...) sont utilisés dans les déclarations de procédure et les définitions pour indiquer une liste d'arguments variable. Un premier argument doit toujours être spécifié et la fonction doit être appelée avec la convention d'appel cdecl du langage C. Dans le corps de la procédure, va_first, va_arg et va_next sont utilisés pour traiter les arguments variables.

L'utilisation des points de suspension (...) pour remplacer la limite supérieure dans une déclaration de tableau, fait que la limite supérieure sera fixée selon les données qui apparaissent dans expression_list. Lorsque l'ellipse est utilisée de cette manière, un initialiseur doit apparaître, et ne peut être Any.

L'utilisation de l'ellipse dans #define ou une déclaration de #macro permet de créer une macro variable, voir #define.

Exemple

Declare Function FOO cdecl (X As Integer, ...) As Integer


Exemple

Dim As Integer myarray(0 To ...) = {0, 1, 2, 3}
Print LBound(myarray), UBound(myarray)   '' 0, 3
