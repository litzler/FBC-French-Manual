WeekdayName
 
Récupère le nom d'un jour de la semaine à partir de sa représentation entière

Syntaxe

Declare Function WeekdayName ( ByVal weekday As Integer, ByVal abbreviate As Integer = 0, ByVal firstdayofweek As Integer = fbUseSystem ) As String

Usage

#include "vbcompat.bi"
result = WeekdayName( weekday [, abbreviate [, firstdayofweek ] ] )

Paramètres

weekday
le numéro du jour de la semaine
abbreviate
drapeau indiquant que le nom doit être abrégé
firstdayofweek
1er jour de la semaine

Return Value

Retourne le nom du jour dans le langage du système d'exploitation local à partir de weekday valeur de 1 à 7.

Description

La manière dont weekday est interprété dépend du paramètre firstdayofweek.

Si abbreviate est vrai, une abréviation de 3 lettres est retournée, s'il est faux ou omis, le nom complet est retourné.

firstdayofweek est un paramètre facultatif spécifié comme suit:

valeur1er jour de la semaineconstante
omissunday = dimanche 
0paramètres locauxfbUseSystem
1sunday = dimanchefbSunday
2monday = lundifbMonday
3tuesday = mardifbTuesday
4wednesday = mercredifbWednesday
5thursday = jeudifbThursday
6friday = vendredifbFriday
7saturday = samedifbSaturday


Le compilateur ne reconnaît cette fonction que si vbcompat.bi ou datetime.bi est inclus.

Exemple

#include "vbcompat.bi"

Dim a As Double = DateSerial(2005, 11, 28) + TimeSerial(7, 30, 50)

Print Format(a, "yyyy/mm/dd hh:mm:ss "); WeekdayName(Weekday(a))


Différences avec QB
