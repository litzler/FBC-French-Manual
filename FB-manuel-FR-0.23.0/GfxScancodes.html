Scancodes clavier DOS
 
Liste des scancodes clavier.

Description

Voici une liste de scancodes clavier acceptés par la fonction MultiKey. Ceux-ci sont égaux aux scancodes DOS et sont garantis d'être toujours reconnus sur toutes les plate-formes.

Ces constantes sont également définis dans le fichier d'inclusion fbgfx.bi que vous pouvez utiliser dans vos programmes. Si vous utilisez le dialecte lang fb alors tout à l'intérieur de fbgfx.bi est enfermé dans le FB Namespace FB. Pour utiliser ces constantes dans lang fb, soit ajouter le préfixe "FB." au nom de la constante ou mettre "Using FB" après la ligne #include.
Le code hexadécimal est pas nécessaire et est fourni seulement pour la référence.
