DefByte
 
Spécifie un type de données par défaut pour une série de noms de variables

Syntaxe

DefByte start_letter[-end_letter ][, ...]

Paramètres

start_letter
la première lettre dans la série
end_letter
la dernière lettre dans la série

Description

DefByte précise que les variables et les tableaux qui ne sont pas déclarés avec un type de données - ou non déclarés entièrement - sont implicitement déclarés de type Byte si la première lettre de leur nom correspond à une certaine lettre ou se trouve dans une plage inclusive de lettres.

Exemple

Ceci fera de bNumber un Byte car sa première lettre commence par un "b":
'' Compile with -lang fblite or qb

#lang "fblite"

DefByte b
Dim bNumber


Différences de dialecte

  • Disponible dans le dialecte -lang fblite.
  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __Defbyte.

Différences avec QB

  • Nouveau pour FreeBASIC
