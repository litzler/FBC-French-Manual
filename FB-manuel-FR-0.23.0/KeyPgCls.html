Cls
 
Efface l'écran, aussi bien dans le mode texte que le mode graphique

Syntaxe

Declare Sub Cls ( ByVal mode As Integer = 1 )

Usage

Cls mode

Paramètres

mode
Une variable numérique facultative d'une valeur de 0 à 2. Si omise, 1 par défaut.

Description

Un paramètre facultatif mode peut être fourni,

S'il est omis, Cls efface le texte ainsi que les graphiques de la fenêtre. Si une fenêtre graphique a été définie en utilisant View (Graphics), cette fenêtre est effacée. Autrement, la fenêtre texte définie par View Print, est nettoyée. (S'il n'y a pas de fenêtre texte explicitement définie, l'écran entier est nettoyé.).

Si à 0, efface l'écran entier.

Si à 1, efface la fenêtre graphique si elle a été définie. Autrement, efface la fenêtre texte.

Si à 2, efface le fenêtre texte.


Exemple

'' Valide  couleur texte : gris clair et arrire-plan bleu
Color 7, 1

'' efface ecran avec couleur arriere-plan
Cls

'' ecris texte au centre ecran
Locate 12, 33
Print "Hello Universe!"


Dans les modes graphiques, si vous voulez effacer entièrement l'écran avec la couleur 0, il est plus rapide d'utiliser Clear pour écrire des zéros dans la mémoire écran que d'appeler Cls.

Dim scrbuf As Byte Ptr, scrsize As Integer
Dim As Integer scrhei, scrpitch
Dim As Integer r = 0, dr = 1

ScreenRes 640, 480, 8

scrbuf = ScreenPtr: Assert( scrbuf <> 0 )
ScreenInfo( , scrhei, , , scrpitch )
scrsize = scrpitch * scrhei

Do
    
    '' verouille ecran (obligatoire pendant travail directement dans tampon ecran)
    ScreenLock
        
        '' efface ecran (possible utiliser Cls ici):
        Clear *scrbuf, 0, scrsize
        
        '' dessine cercle
        Circle (320, 240), r
        
    ScreenUnlock
    
    '' augment/diminue rayon cercle
    r += dr
    If r <= 0 Then dr = 1 Else If r >= 100 Then dr = -1
    
    '' courte pause pour chaque image (previent saturation CPU)
    Sleep 1, 1
    
touche appuyee
Loop Until Len(Inkey) > 0
