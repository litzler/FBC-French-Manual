CPtr
 
Convertit une expression pointeur en pointeur de type de données spécifié

Syntaxe

CPtr( PointerDataType, expression )

Description

Convertit expression vers PointerDataType.

PointerDataType doit être un Pointer vers un type de données (par exemple, un type de données Ptr ou un Any Ptr), ou un Type (Alias).
expression peut avoir un autre type de pointeur ou être un Integer.

Note: Actuellement, FB n'impose pas effectivement que PointerDataType soit un pointeur. Cela changera probablement dans les futures versions. À l'heure actuelle, il affichera un avertissement si vous tentez de convertir vers un non-pointeur, si vous compilez avec le commutateur de compilateur -w pedantic.

Exemple

Dim intval As Integer
Dim intptr As Integer Ptr
intval = &h0080
intptr = @intval
'' will print -128 and 128, as the first expression will be "seen" as an signed byte
Print *CPtr( Byte Ptr, intptr ), *intptr


Différences de dialecte

  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __Cptr.

Différences avec QB

  • Nouveau pour FreeBASIC
