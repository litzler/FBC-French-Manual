Time
 
Retourne l'heure courante du système sous forme d'une chaîne

Syntaxe

Declare Function Time ( ) As String

Usage

result = Time

Valeur retournée

Retourne l'heure courante du système.

Description

Retourne l'heure courante du système dans le format hh:mm:ss.

Exemple

Print "the current time is: "; Time


Différences avec QB

  • La déclaration QB TIME (pour définir la date système) est maintenant appelée SetTime.
