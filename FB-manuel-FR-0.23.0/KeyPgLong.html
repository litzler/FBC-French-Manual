Long
 
Type de données standard: nombre entier signé ayant la même taille que SizeOf(Any Ptr)

Syntaxe

Dim variable As Long

Description

Selon la plate-forme, identique à un Integer ou un LongInt. Type de données: nombre entier signé de 32 bits ou 64 bits.

Long a la même taille que SizeOf(Any Ptr).

Exemple

  Dim x As Long = &H80000000
  Dim y As Long = &H7FFFFFFF
  Print "Long Range = "; x; " to "; y

Affichage:
Long Range = -2147483648 to  2147483647

Différences de dialecte

  • Dans les dialectes -lang fb et -lang fblite, le type de données Long a la même taille que SizeOf(Any Ptr).
  • Dans le dialecte -lang qb, le type de données Long a toujours 32 bits.

Différences avec QB

  • Dans QB, LONG est toujours 32 bits.
  • Aucune, si compilé dans le dialecte -lang qb.
