#endif
 
Directive conditionnelle du préprocesseur

Syntaxe

#endif

Description

Termine un groupe de directives conditionnelles

Voir #if, #ifdef, ou #ifndef pour des exemples d'utilisation.

Exemple

#define DEBUG_LEVEL 1
#if (DEBUG_LEVEL = 1)
  'Conditional statements
#endif
