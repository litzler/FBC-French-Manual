Read Write (Accès fichiers)
 
Spécificateur d'accès fichier

Syntaxe

Open filename As String For Binary Access Read Write As #filenum As Integer

Description

Spécificateur pour la clause Access dans l'instruction Open. Read Write spécifie que le fichier est accessible en lecture et en écriture.
