Seek (Déclaration)
 
Définit la position de la prochaine opération de lecture/écriture sur un fichier

Syntaxe

Seek [#]filenum, position

Paramètres

filenum
numéro de fichier d'un fichier ouvert
position
la nouvelle position pour les opérations d'entrée/sortie

Description

Définit la position à laquelle la prochaine lecture ou écriture sur un fichier va se produire.

La position est donnée en enregistrements si le fichier a été ouvert en mode d'accès Random, en octets dans les autres cas. La position est basée sur 1 -- le 1er enregistrement est à la position 1.

La fonction Seek est utilisée pour déterminer la position de la prochaine opération de lecture ou d'écriture.

Exemple

' e.g. if you want to skip to the 100th byte in the file for reading/writing:

Dim f As Integer

f = FreeFile
Open "file.ext" For Binary As #f

Seek f, 100

Close #f
