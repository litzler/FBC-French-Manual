WHex
 
Retourne la wstring (Unicode) hexadécimale representation d'un nombre

Syntaxe

Declare Function WHex Overload ( ByVal number As Byte ) As WString
Declare Function WHex ( ByVal number As Byte, ByVal digits As Integer ) As WString

Declare Function WHex ( ByVal number As Short ) As WString
Declare Function WHex ( ByVal number As Short, ByVal digits As Integer ) As WString

Declare Function WHex ( ByVal number As Integer ) As WString
Declare Function WHex ( ByVal number As Integer, ByVal digits As Integer ) As WString

Declare Function WHex ( ByVal number As LongInt ) As WString
Declare Function WHex ( ByVal number As LongInt, ByVal digits As Integer ) As WString

Usage

result = WHex( number [, digits ] )

Paramètres

number
Un nombre ou une expression numérique représentant un nombre.
digits
Facultatif: nombre de chiffres à retourner.

Valeur retournée

Retourne une String hexadécimale représentation de number.

Description

Chiffres hexadécimaux de 0 à 9, ou A à F.

Si vous spécifiez digits > 0, la WString résultat aura exactement cette longueur. Elle sera tronquée ou complétée avec des zéros sur la gauche, si nécessaire.

La longueur de la wstring ne dépassera pas le nombre de bits maximum requis pour le type de expression (8 pour un entier, 16 pour un nombre à virgule flottante ou un entier long)

Exemple

Print Hex(54321)
Print Hex(54321, 2)
Print Hex(54321, 5)

produira la sortie:

D431
31
0D431

Différences de plate-forme

  • Les (w)strings Unicode ne sont pas supportées dans le portage DOS de FreeBASIC.


Différences de dialectes

  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __Whex.

Différences avec QB

  • Nouveau pour FreeBASIC.
