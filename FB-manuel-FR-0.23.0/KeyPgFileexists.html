FileExists
 
Teste l'existence d'un fichier

Syntaxe

Declare Function FileExists ( ByVal filename As ZString Ptr ) As Integer

Usage

#include "file.bi"
result = FileExists( filename )

ou

#include "vbcompat.bi"
result = FileExists( filename )

Paramètres

filename
Nom du fichier dont on teste l'existence.

Valeur retournée

Retourne non-zéro (-1) si le fichier existe, autrement retourne zéro (0).

Description

FileExists teste l'existence d'un fichier.
En interne, il peut y avoir émission d'un Open() et d'une fonction close(), qui peut avoir des conséquences - par exemple, tout Lock existant sur le fichier peut être libéré.
Selon les besoins exacts, une méthode alternative de vérifier l'existence des fichiers est d'utiliser la fonction Dir() (en faisant attention aux attributs et en s'assurant que le chemin ne contient pas jokers), ou d'essayer un Open sur le fichier puis vérifier la valeur de retour et tester la réussite.

Exemple

#include "vbcompat.bi"

Dim filename As String

Print "Enter a filename: "
Line Input filename

If FileExists( filename ) Then
  Print "File found: " & filename
Else
  Print "File not found: " & filename
End If


Différences de plate-forme

  • Linux nécessite que la casse de filename corresponde au nom réel du fichier. Windows et DOS ne sont pas sensibles à la casse.
  • Les séparateurs de chemin de Linux sont des barres obliques /. Windows utilise des barres obliques vers l'arrière \ mais il admet des barres obliques. DOS utilise des barres obliques en arrière \.
