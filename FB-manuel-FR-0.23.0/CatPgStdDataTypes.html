Types de données standard
 
Types numériques et chaînes d'origine.

Types des nombres entiers
Types qui stockent des valeurs entières, dont l'étendue est déterminée par la taille du type de donnée et le fait d'être signé ou non.
Types à virgule flottante
Types qui stockent des valeurs réelles (nombres à virgule), dont l'étendue et la précision est déterminée par la taille du type de donnée.
Modificateurs de type de données
Spécifient les caractéristiques supplémentaires d'une norme définie par l'utilisateur ou d'un type de données.
Types des chaînes
Types qui stockent ou pointent vers un tableau de caractères.

Types des nombres entiers
Byte et UByte
Types de données de 8-bit d'étendue qui stockent des valeurs entières.
Short et UShort
Type de données de 16-bit d'étendue qui stockent des valeurs entières.
Integer et UInteger
Type de données de 32-bit d'étendue qui stockent des valeurs entières.
Long et Ulong
Type de données de 32-bit ou 64-bit d'étendue qui stockent des valeurs entières.
LongInt et ULongInt
Type de données de 64-bit d'étendue qui stockent des valeurs entières.

Types à virgule flottante
Single
Type de données de 32-bit d'étendue qui stockent des valeurs réelles (nbre à virgule).
Double
Type de données de 64-bit d'étendue qui stockent des valeurs réelles (nbre à virgule).
Modificateurs de type de données
Const
Spéecifie un type de données en lecture seulement.
Pointer et Ptr
Modifie les types pour être des pointeurs de types.
Unsigned
Spécifie un type entier non-signé.

Types des chaînes
String
Chaînes à longueur-fixe ou longueur-variable avec gestion intégrée de la mémoire.
ZString
Chaînes à longueur-fixe ou longueur-variable avec marqueur de fin nul.
WString
Chaînes de caractères longs à longueur-fixe ou longueur-variable avec marqueur de fin nul.
