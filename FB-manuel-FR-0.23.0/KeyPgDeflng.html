DefLng
 
Spécifie un type de données par défaut pour une série de noms de variables

Syntaxe

DefLng start_letter[-end_letter ][, ...]

Paramètres

start_letter
la première lettre dans la série
end_letter
la dernière lettre dans la série

Description

DefLng précise que les variables et les tableaux qui ne sont pas déclarés avec un type de données - ou non déclarés entièrement - sont implicitement déclarés de type Long si la première lettre de leur nom correspond à une certaine lettre ou se trouve dans une plage inclusive de lettres.

Exemple

Ceci fera de lNumber un nombre entier Long car sa première lettre est l.
'' Compile with -lang fblite or qb

#lang "fblite"

DefLng l
Dim lNumber ' implicit: As Long

Print Len(lNumber) ' Displays 4, the number of bytes in a long.


Différences de dialecte

  • Disponible seulement dans les dialectes -lang qb et -lang fblite.

Différences avec QB

  • Aucune
