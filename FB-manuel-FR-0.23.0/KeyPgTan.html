Tan
 
Obtient la tangente d'un nombre

Syntaxe

Declare Function Tan ( ByVal number As Double ) As Double

Usage

result = Tan( number )

Paramètres

number
l'angle (en radians)

Valeur retournée

Renvoie la tangente de l'argument number comme un Double dans l'intervalle de -infinité à infinité.

Description

L'argument number est mesuré en radians (non en degrés). L'argument number peut être n'importe quelle expression numérique valide. Si number est une variable non-initialisée, zéro est retourné.

Exemple

Const PI As Double = 3.1415926535897932
Dim a As Double
Dim r As Double
Input "Please enter an angle in degrees: ", a
r = a * PI / 180    'Convert the degrees to Radians
Print ""
Print "The tangent of a" ; a; " degree angle is"; Tan ( r ) 
Sleep


Affichera:
Please enter an angle in degrees: 75
The tangent of a 75 degree angle Is 3.732050807568878
