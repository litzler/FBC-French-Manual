Else
 
Déclaration de contrôle de flux pour le branchement conditionnel

Syntaxe

If expression Then statement(s) [Else statement(s)]
ou
If expression Then : statement(s) [Else statement(s)] : End If
ou
If expression Then
statement(s)
[ ElseIf expression Then ]
statement(s)
[ Else ]

End If
