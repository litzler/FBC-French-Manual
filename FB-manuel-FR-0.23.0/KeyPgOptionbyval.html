Option ByVal
 
Spécifie les paramètres qui doivent être passés par valeur par défaut dans les déclarations de procédure

Syntaxe

Option ByVal

Description

Option ByVal est une instruction qui fixe la valeur par défaut de la convention de passage pour les paramètres des procédures à by value (par valeur), comme s'ils avaient été déclarés avec ByVal. Ceci reste en vigueur pour tout le reste du module dans lequel Option ByVal est utilisé, et peut être annulé en spécifiant ByRef dans les liste de paramètres.

Exemple

'' compile with the "-lang fblite" compiler switch

#lang "fblite"

Sub TestDefaultByref( a As Integer )
  '' change the value
  a = a * 2
End Sub

Option ByVal

Sub TestDefaultByval( a As Integer )
  a = a * 2
End Sub

Dim a As Integer = 1

Print "a = "; a
TestDefaultByref( a )
Print "After TestDefaultByref : a = "; a
Print

Print "a = "; a
TestDefaultByval( a )
Print "After TestDefaultByval : a = "; a
Print


Différences de dialecte

  • Disponible seulement dans les dialectes -lang fblite et -lang qb.

Différences avec QB

  • Nouveau pour FreeBASIC
