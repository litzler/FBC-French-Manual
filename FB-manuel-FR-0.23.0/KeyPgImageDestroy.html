ImageDestroy
 
Détruit et libère une image (stockage en mémoire)

Syntaxe

Declare Sub ImageDestroy ( ByVal image As Any Ptr )

Usage

ImageDestroy( image )

Paramètres

image
Adresse de l'image à détruire.

Description

Détruit l'image pointée par image, qui doit être une adresse retournée par un appel à ImageCreate.

Exemple

Voir la fonction ImageCreate pour un exemple d'utilisation de ImageDestroy.


Différences de Dialecte

  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __Imagedestroy.

Differences avec QB

  • Nouveau pour FreeBASIC
