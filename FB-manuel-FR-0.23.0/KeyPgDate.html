Date
 
Retourne la date courante du système sous forme d'une chaîne

Syntaxe

Declare Function Date ( ) As String

Usage

result = Date

Valeur retournée

Retourne la date courante du système, dans le format mm-dd-yyyy

Description

Aucune

Exemple

Print Date ' prints the current date


Différences avec QB

  • La déclaration QB DATE (pour définir la date système) est maintenant appelée SetDate.
