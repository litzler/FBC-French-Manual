Option du compilateur: -prefix
 
Définit le préfixe du chemin du compilateur

Syntaxe

-prefix < path >

Paramètres

path
Le répertoire, relatif ou absolu de l'endroit où est situé fbc.

Description

L'option de compilateur -prefix définit le préfixe du chemin du compilateur (où le compilateur trouve les répertoires bin, lib, et inc); et par défaut ce sera le chemin où fbc réside, s'il peut être déterminé.
