Sin
 
Retourne le sinus d'un nombre

Syntaxe

Declare Function Sin ( ByVal number As Double ) As Double

Usage

result = Sin( number )

Paramètres

number
l'angle (en radians)

Value retournée

Retourne le sinus de l'argument number comme un Double dans l'intervalle de -1 à 1.

Description

L'argument number est mesuré en radians (non pas en degrés). L'argument obligatoire number peut être n'importe quelle expression numérique valide. Si number est une variable non-initialisée, zéro est retourné.

Example

Const PI As Double = 3.1415926535897932
Dim a As Double
Dim r As Double
Input "Please enter an angle in degrees: ", a
r = a * PI / 180    'Convert the degrees to Radians
Print ""
Print "The sine of a" ; a; " degree angle is"; Sin ( r ) 
Sleep


Affichera:
Please enter an angle in degrees: 30
The sine of a 30 degree angle Is 0.5
