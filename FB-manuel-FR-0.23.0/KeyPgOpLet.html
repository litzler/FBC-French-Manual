Opérateur Let (Affectation)
 
Indique l'opérateur d'affectation en cas de surcharge Operator = (Assignment)

Syntaxe

{ Type | Class | Union | Enum } typename
Declare Operator Let ( [ ByRef | ByVal ] rhs As datatype )
End { Type | Class | Union }

Operator typename.Let ( [ ByRef | ByVal ] rhs As datatype )

Usage

lhs = rhs

Paramètres

typename
Nom de Type, Class, Union, ou Enum
lhs
La variable à affecter.
rhs
La valeur à affecter.

Description

Let est utilisé pour surcharger l'opérateur Operator = (Affectation) et pour le distinguer de l'opérateur de comparaison Operator = (Egal).

lhs = rhs attribuera le terme de droite rhs au terme de gauche lhs en invoquant la procédure d'opérateur Let définie dans typename.

Exemple

Type T
  x As Integer
  y As Integer
  Declare Operator Let( ByRef rhs As T )
End Type

Operator T.let( ByRef rhs As T )
  x = rhs.x
  y = rhs.y
End Operator

Dim a As T = ( 5, 7 )
Dim b As T

'' Do the assignment invoking the LET
'' operator procedure
b = a

Print "a.x = "; a.x
Print "a.y = "; a.y
Print
Print "b.x = "; b.x
Print "b.y = "; b.y

Affiche:
a.x =  5
a.y =  7

b.x =  5
b.y =  7

Différences de dialecte

  • Dans les dialectes -lang qb et -lang fblite, cet opérateur ne peut être surchargé
  • Dans les dialectes -lang qb et -lang fblite, une expression d'affectation peut être précédée par le mot-clef Let keyword.

Différences avec QB

  • Aucune
