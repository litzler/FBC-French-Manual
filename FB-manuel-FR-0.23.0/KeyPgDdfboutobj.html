__FB_OUT_OBJ__
 
Définition intrinsèque (valeur de la macro) fixée par le compilateur

Syntaxe

__FB_OUT_OBJ__

Description

__FB_OUT_OBJ__ indique que le type de sortie fichier spécifié sur la ligne de commande du compilateur au moment de la compilation est un module objet.

Retourne non-nul (-1) si la sortie est un module objet. Retourne zéro (0) autrement.

Seulement un des __FB_OUT_DLL__, __FB_OUT_EXE__, __FB_OUT_LIB__, ou __FB_OUT_OBJ__ sera à non-zéro (-1). Tous les autres seront à zéro (0).

Exemple

#if __FB_OUT_OBJ__ 
        '... specific instructions when compiling to an object file only
#else
        '... specific instructions when not compiling to an object file only
#endif
