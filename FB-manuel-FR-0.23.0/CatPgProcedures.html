Procédures
 
Mots-clés qui fonctionnent avec des procédures.

Description

Ces mots-clés contrôlent la déclaration et la définition des procédures, à la fois, au niveau du module et au niveau des membres de procédures, comment ils sont appelés, comment les arguments sont passés et la manière dont leurs noms sont vus à l'extérieur par les autres modules. Des procédures peuvent également être déclarées d'être exécutées automatiquement avant que tout autre code de niveau module ne soit exécuté.

Déclaration
Mots-clés que déclarent et définissent des procédures.
Liens
Mots-clés qui spécifient comment les noms de procédures sont perçus par des modules externes.
Conventions d'appel
Mots-clés qui précisent comment les arguments sont utilisés lors de l'appel de procédures.
Conventions de passage de paramètre(s)
Mots-clés qui spécifient comment les arguments sont passés aux procédures.
Procédures "Variadic"
Les macros qui permettent de passer un nombre arbitraire d'arguments à une procédure.
Exécution automatique
Mots-clés qui spécifient l'exécution automatique de procédures.
Divers
Divers mots-clés.

Déclaration
Declare
Déclare une procédure au niveau module ou une procédure membre.
Sub
Indique une procédure qui ne renvoie pas d'argument.
Function
Indique une procédure qui retourne un argument.
Overload
Indique que le nom de la procédure peut être utilisé dans des déclarations d'autres procédures.
Static
Spécifie un stockage statique pour toutes les variables et les objets dans le corps de la procédure.
Const (Membre)
Indique un membre de procédure constant dans des définitions de type défini par l'utilisateur.
Static (Membre)
Indique un membre de procédure static dans des définitions de type défini par l'utilisateur.

Liens
Public
Spécifie un lien externe pour une procédure.
Private
Spécifie un lien interne pour une procédure.
Alias
Spécifie un autre nom externe pour une procédure.
Export
Indique qu'une procédure doit être exportée à partir d'une bibliothèque partagée.
Lib
Indique le chargement automatique d'une bibliothèque.

Conventions d'appel
stdcall
Indique la convention d'appel standard pour les langages BASIC, notamment FreeBASIC.
cdecl
Indique la convention d'appel standard en langages C and C++.
pascal
Indique la convention d'appel standard pour les langages Fortran, Pascal et Microsoft QuickBASIC/QBasic.
Conventions de passage de paramètre(s)
ByRef
Indique le passage d'un argument par référence.
ByVal
Indique le passage d'un argument par valeur.
Any
Désactive la vérification de type sur des arguments.

Procédures "Variadic"
... (Ellipse)
Indique une procédure "variadic" dans une déclaration.
va_first
Macro pour obtenir la liste des arguments d'une procédure "variadic".
va_arg
Macro pour obtenir l'argument courant dans une procédure "variadic".
va_next
Macro pour passer à l'argument suivant dans une procédure "variadic".

Exécution automatique
Constructor (Module)
Indique une procédure qui doit être exécutée avant le code au niveau du module.
Destructor (Module)
Indique une procédure qui doit être exécutée après le code au niveau du module.

Divers
Call
Appelle une procédure.
Naked
Spécifie qu'un corps de la fonction ne doit avoir aucun code prologue/épilogue
