Right
 
Retourne la sous-chaîne la plus à droite d'une chaîne de caractères

Syntaxe

Declare Function Right Overload ( ByRef str As Const String, ByVal n As Integer ) As String
Declare Function Right ( ByRef str As Const WString, ByVal n As Integer ) As WString

Usage

result = Right[$]( str, n )

Paramètres

str
La chaîne de caractères source.
n
La longueur de la sous-chaîne, en caractères.

Valeur retournée

Retourne la sous-chaîne la plus à droite à partir de str.

Description

Retourne les n caractères les plus à droite à partir de la droite (fin) de str. Si str est vide, alors la chaîne nulle ("") est retournée. Si n <= 0 alors la chaîne nulle ("") est retournée. Si n > len(str) alors la chaîne entière est retournée.

Exemple

Dim text As String = "hello world"
Print Right(text, 5)

produira la sortie::

world
Un exemple Unicode:

dim text as wstring*20
text = "Привет, мир!"
print right(text, 5) 'displays " мир!"

Différences de plate-forme

  • DOS ne supporte pas la version large WString de la chaîne de Right.

Différences de dialecte

  • Le suffixe de type de chaîne "$" est obligatoire dans le dialecte -lang qb.
  • Le suffixe de type de chaîne "$" est facultatif dans les dialectes -lang fblite et -lang fb.

Différences avec QB

  • QB ne supporte pas Unicode.
