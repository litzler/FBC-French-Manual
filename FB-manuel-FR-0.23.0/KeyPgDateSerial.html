DateSerial
 
Crée une date serial

Syntaxe

Declare Function DateSerial ( ByVal year As Integer, ByVal month As Integer, ByVal day As Integer ) As Integer

Usage

#include "vbcompat.bi"
result = DateSerial( year, month, day )

Paramètres

year
l'année
month
le mois de l'année
day
le jour du mois

Valeur retournée

Retourne une date serial contenant la date formée par les valeurs dans les paramètres year, month et day. La "date serial" retournée n'a pas de partie décimale.

Description

Le compilateur ne reconnaît cette fonction que si vbcompat.bi ou datetime.bi est inclus.

Exemple

#include "vbcompat.bi"

Dim a As Double = DateSerial(2005, 11, 28)

Print Format(a, "yyyy/mm/dd hh:mm:ss") 


Différences avec QB

  • N'existe pas dans QB. Cette fonction est apparue dans PDS et VBDOS
