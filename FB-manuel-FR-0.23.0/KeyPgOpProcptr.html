Opérateur Procptr (Pointeur de procédure)
 
Renvoie l'adresse d'une procédure

Syntaxe

Declare Operator ProcPtr ( ByRef lhs As T ) As T Ptr

Usage

result = ProcPtr ( lhs )

Paramètres

lhs
Une procédure.
T
Le type de procédure.

Valeur retournée

Renvoie l'adresse de la procédure.

Description

Cet opérateur renvoie l'adresse d'une procédure Sub ou Function.

L'Opérateur @ (Adresse de), utilisé avec des procédures, a un comportement identique.

Exemple

' This example uses ProcPtr to demonstrate function pointers
Declare Function Subtract( x As Integer, y As Integer) As Integer
Declare Function Add( x As Integer, y As Integer) As Integer
Dim myFunction As Function( x As Integer, y As Integer) As Integer

' myFunction will now be assigned to Add
myFunction = ProcPtr( Add )
Print myFunction(2, 3)

' myFunction will now be assigned to Subtract.  Notice the different output.
myFunction = ProcPtr( Subtract )
Print myFunction(2, 3)

Function Add( x As Integer, y As Integer) As Integer
    Return x + y
End Function

Function Subtract( x As Integer, y As Integer) As Integer
    Return x - y
End Function


Différences de dialecte

  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __Procptr.

Différences avec QB

  • Nouveau pour FreeBASIC
