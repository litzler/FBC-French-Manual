MKShort
 
Copie binaire d'une variable Short en une String (chaîne), réglage de sa longueur à 2 octets

Syntaxe

Declare Function MKShort ( ByVal number As Short ) As String

Usage

result = MKShort[$](number)

Paramètres

number
Une variable Short dont on fait une copie binaire vers une String (chaîne).

Valeur retournée

Retourne une String (chaîne) copie binaire d'un nombre Short.

Description

Exécute une copie binaire d'une variable Short vers une String (chaîne) dont la longueur est de 2 octets. La chaîne résultat peut être lue à nouveau comme un Short par CVShort

Cette fonction est utile pour écrire une valeur numérique dans un tampon sans utiliser de définition de Type.

Exemple

Dim a As Short, b As String
a = 4534
b = MKShort(a)
Print a, CVShort(b)
Sleep



Différences de dialectes

  • Non disponible dans le dialecte -lang qb sauf si référencé par l'alias __Mkshort.

Différences avec QB

  • Dans QBasic cette fonction est appelée MKI.
