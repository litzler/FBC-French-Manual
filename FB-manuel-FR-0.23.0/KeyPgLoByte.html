LoByte
 
Obtient la valeur de l'octet le moins significatif (LSB = Least-Significant Byte, ou lo-byte) d'une valeur UInteger.

Syntaxe

#define LoByte( expr ) ( Cast(UInteger, expr) And &h000000FF )

Usage

result = LoByte( expr )

Paramètres

expr
Une expression numérique, convertie en une valeur UInteger.

Valeur retournée

Retourne la valeur de l'octet le moins significatif (LSB) de `Cast(UInteger, expr)`.

Description

Cette macro convertit l'expression numérique expr en une valeur UInteger, qu'elle développe ensuite en un UInteger représentant la valeur de son LSB.

Exemple

Dim N As UInteger

'Note there are 16 bits
N = &b1010101110000001
Print "N is                                       "; N
Print "The binary representation of N is          "; Bin(N)
Print "The most significant byte (MSB) of N is    "; HiByte(N)
Print "The least significant byte (LSB) of N is   "; LoByte(N)
Print "The binary representation of the MSB is    "; Bin(HiByte(N))
Print "The binary representation of the LSB is    "; Bin(LoByte(N))
Sleep


La sortie sera:
N Is                                       43905
The Binary representation of N Is          1010101110000001
The most significant Byte (MSB) of N Is    171
The least significant Byte (LSB) of N Is   129
The Binary representation of the MSB Is    10101011
The Binary representation of the LSB Is    10000001


Différences de dialectes

  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __LOBYTE.
