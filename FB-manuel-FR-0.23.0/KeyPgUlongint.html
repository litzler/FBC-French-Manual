ULongInt
 
Type de données standard: nombre de 64 bits non-signé

Syntax

Dim variable As ULongInt

Description

Nombre entier de 64 bits non-signé. Peut contenir des valeurs de 0 à 18 446 744 073 709 551 615

Exemple

Dim x As ULongInt = 0
Dim y As ULongInt = &HFFFFFFFFFFFFFFFFull
Print "ULongInt Range = "; x; " to "; y

Affichage:
ULongInt Range = 0 to 18446744073709551615


Différences de dialecte

  • Non disponible dans le dialecte -lang qb sauf si référencé avec l'alias __Ulongint.
