Option du compilateur: -R
 
Ne pas supprimer le(s) fichier(s) asm

Syntaxe

-R

Description

L'option -R du compilateur implique la non-suppression des fichiers asm file(s) générés pendant le processus de compilation. Notez qu'avec -gen gcc des fichiers *.c sont générés, au lieu de fichiers *.asm normaux.
