Access
 
Clause de l'instruction Open pour spécifier les privilèges demandés

Syntaxe

Open filename for Binary Access {Read | Write | Read Write} as [#]filenum

Usage

open filename for binary Access Read as #filenum
open filename for binary Access Write as #filenum
open filename for binary Access Read Write as #filenum

Paramètres

Read
Ouvrir le fichier en lecture seulement.
Write
Ouvrir le fichier en écriture seulement.
Read Write
Ouvrir le fichier en lecture et écriture.

Description

Access est utilisé avec l'instruction Open pour obtenir l'ouverture d'un fichier en lecture, en écriture ou en lecture et en écriture. Si la clause Access n'est pas spécifiée, Read Write est utilisé.

Exemple


Cet exemple montre comment ouvrir le fichier "data.raw" avec un accès Read (lecture) et puis "data.out" avec un accès en Write (écriture), en mode Binary (binaire), dans un certain nombre de fichiers ouverts retournés par FreeFile.
Dim As Integer o

  '' get an open file number.
  o = FreeFile
  
  '' open file for read-only access.    
  Open "data.raw" For Binary Access Read As #o
    
    '' make a buffer in memory thats the entire size of the file
    Dim As UByte file_char( LOF( o ) - 1 )

      '' get the file into the buffer.      
      Get #o, , file_char()
    
  Close
  
  '' get another open file number.
  o = FreeFile
  
  '' open file for write-only access.    
  Open "data.out" For Binary Access Write As #o

    '' put the buffer into the new file.      
    Put #o, , file_char()
    
  Close

  Print "Copied file ""data.raw"" to file ""data.out"""

  Sleep
