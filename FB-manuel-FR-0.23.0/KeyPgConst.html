Const
 
Déclare une variable non-modifiable.

Syntaxe

Const symbolname [AS DataType] = value [, ...]

Description

Déclare une donnée constante non-modifiable qui peut être un nombre entier ou décimal (à virgule flottante) ou une chaîne. Le type constante sera appliqué si DataType n'est pas donné explicitement.

Exemple

Const Red = RGB(252, 2, 4)
Const Black As UInteger = RGB(0, 0, 0)
Const Text = "This is red text on a black bkgnd."

Locate 1, 1
Color Red, Black
Print Text
Sleep
End


Différences avec QB

  • QB ne supporte pas la syntaxe As datatype.
