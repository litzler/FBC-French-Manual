Option du compilateur: -a
 
Ajouter un fichier objet à la liste de l'éditeur de liens

Syntaxe

[ -a ] < object file >

Paramètres

object file
Nom du fichier objet avec l'extension.

Description

L'option -a du compilateur ajoute un fichier objet compilé à la liste de l'éditeur de liens. Le "-a" est facultative si le nom du fichier objet a une extension de fichier ".o".
