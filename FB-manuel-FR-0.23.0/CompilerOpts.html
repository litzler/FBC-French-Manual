Option du compilateur: -s
 
Définit le sous-système exécutable

Syntaxe

-s < subsystem >

Paramètres

subsystem
Le sous-système exécutable: gui ou console.

Description

L'option -s du compilateur spécifie le sous-système exécutable. Les sous-systèmes admis sont gui et console (par défaut, la console est utilisée). La spécification d'un sous-système gui d'interface graphique empêche la fenêtre de console d'apparaître derrière la fenêtre du programme.

Différences de plate-forme

  • Pris en charge seulement sur les plates-formes Windows.
