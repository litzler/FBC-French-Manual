SetDate
 
Fixe la date du système

Syntaxe

Declare Function SetDate ( ByRef newdate As Const String ) As Integer

Usage

result = SetDate( newdate )

Paramètres

newdate
la nouvelle date à valider

Valeur retournée

Retourne zéro si succès ou non-zéro si échec sur tous les ports sauf DOS.

Description

Pour valider la nouvelle date, formatez seulement newdate et envoyez-la à SetDate dans l'un des formats valides qui suivent: "mm-dd-yy", "mm-dd-yyyy", "mm/dd/yy", or "mm/dd/yyyy" (mm est le mois, dd est le jour, yy ou yyyy est l'année.

Exemple

Dim m As String, d As String, y As String
m = "03" 'march
d = "13" 'the 13th
y = "1994" 'good ol' days
SetDate m + "/" + d + "/" + y


Différences avec QB

  • La déclaration DATE était utilisée dans QB et la syntaxe était "DATE = string"
