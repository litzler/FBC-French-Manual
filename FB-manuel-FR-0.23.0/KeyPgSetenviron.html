SetEnviron
 
Définit une variable d'environnement système

Syntaxe

Declare Function SetEnviron ( ByRef varname As String ) As Integer

Usage

result = SetEnviron( varname )

Paramètres

varname
Le nom de la variable d'environnement à définir.

Valeur retournée

Retourne zéro (0) si succès et non-zéro en cas d'échec.

Description

Modifie les variables d'environnement système. Il y a plusieurs variables disponibles pour l'édition autres que celles par défaut sur votre système. Un exemple de ceci serait "fbgfx", où vous pouvez choisir la forme de pilote graphique de la bibliothèque graphiques FreeBasic à utiliser.

Exemple

'e.g. to set the system variable "path" to "c:":

Shell "set path" 'shows the value of path
SetEnviron "path=c:"
Shell "set path" 'shows the new value of path
