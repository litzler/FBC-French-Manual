CVD
 
Conversion d'une chaîne de 8 octets en une variable double-précision

Syntaxe

Declare Function CVD ( ByRef str As Const String ) As Double

Usage

result = CVD( str )

Paramètres

str
Une chaîne (String) ayant exactement 8 octets de longueur, copie d'une variable double-précision.

Valeur retournée

Retourne une variable Double-Precision qui contient la copie binaire d'un DOUBLE.

Description

Exécute une copie binaire d'une chaîne-8-octets vers une variable double-précision. Le résultat n'aura un sens que si la chaîne contient une valeur double formatée IEEE754, comme celle générée par MKD.

Cette fonction est utile pour lire une valeur numérique à partir d'un tampon sans utiliser de définition de Type.

Exemple

Dim a As Double, b As String
a=4534.4243
b=MKD(a)
Print a, CVD(b)
Sleep
