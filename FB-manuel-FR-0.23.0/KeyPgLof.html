LOF
 
Retourne la longueur d'un fichier disque ouvert

Syntaxe

Declare Function LOF ( ByVal filenum As Integer ) As LongInt

Usage

result = LOF( filenum )

Paramètres

filenum
Le numéro de fichier d'un fichier ouvert sur le disque.

Valeur retournée

La longueur en octets d'un fichier ouvert sur le disque.

Description

Renvoie la longueur, en octets, d'un fichier ouvert précédemment avec Open (=ouvrir) en utilisant filenum.

Avec Open Com, il renvoie la longueur des données en attente d'être lues, dans le tampon de réception.

Exemple

Dim f As Integer
f = FreeFile
Open "file.ext" For Binary As #f
Print LOF(f)
Close #f
