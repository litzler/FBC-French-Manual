Frac
 
Renvoie la partie décimale d'un nombre

Syntaxe

Declare Function Frac ( ByVal number As Double ) As Double

Usage

result = Frac( number )

Paramètres

number
nombre ou expression dont on veut obtenir la partie décimale.

Valeur retournée

Retourne la partie décimale d'un nombre ou d'une expression.

Description

Équivalent à: (number - Fix(number)).

L'opérateur Operator unaire Frac peut être surchargé avec des types définis par l'utilisateur.

Exemple

Print frac(10.625)  '' will print  0.625
Print frac(-10.625) '' will print -0.625 


Différences de dialecte

  • Dans le dialecte -lang qb, cet opérateur ne peut pas être surchargé.
