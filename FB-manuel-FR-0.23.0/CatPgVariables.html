Déclaration des variables
 
Déclarations permettant de définir et d'allouer de l'espace aux variables.

Dim
Déclare une variable de portée locale.
Const
Déclare une variable non-modifiable.
Scope
Commence nouveau un bloc de portée des variables.

Static
Déclare des variables dans une procédure qui conservent leur valeur entre les appels.
Shared
Utilisé avec Dim permet aux variables d'être visibles à travers un module.
Var
Déclare des variables dont le type de données est implicite à partir d'une initialisation.
