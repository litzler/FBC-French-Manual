Bit
 
Obtient l'état individuel d'un bit dans une valeur entière.

Syntaxe

#define Bit( value, bit_number ) (((value) And (Cast(TypeOf(value), 1) Shl (bit_number))) <> 0)

Usage

result = Bit( value, bit_number )

Paramètres

value
La valeur entière.
bit_number
L'indice du bit.

Valeur retournée

Retourne une valeur Integer de -1 si le bit est valide (1), ou 0 si le bit est à 0.

Description

Cette macro se développe en une valeur entière indiquant si oui ou non le bit spécifié par bit_number est validé (à 1) dans value entière. Se comporte comme `(value And 1 Shl bit_number) <> 0`.

Exemple

Print Bit(4,2)
Print Bit(5,1)
Print Bit(&H8000000000000000ULL,63)

produira la sortie:

-1
0
-1

Différences de dialectes

  • Non disponible dans le dialecte -lang qb.

Différences avec QB
