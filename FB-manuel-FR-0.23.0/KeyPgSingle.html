Single
 
Type de données standard: nombre à virgule flottante de 32 bits

Syntaxe

Dim variable As Single

Description

Single est un type de nombre de 32 bits, à virgule flottante utilisé pour stocker des nombres décimaux. Ils peuvent contenir des valeurs de 1.401298e-45 à 3.402823e+38, ou des valeurs négatives de -1.401298e-45 à -3.402823e+38, ou zéro (0). Ils ont une précision au plus de 24 bits ou de 6 chiffres après la virgule.

Ils sont similaires aux Double, mais moins précis.

Exemple

'Example of using a single variable.

Dim a As Single
a = 1.9857665
Print a
