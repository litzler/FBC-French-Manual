Option Static
 
Revient au comportement par défaut de déclaration de tableau

Syntaxe

Option Static

Description

Option Static annule le comportement de Option Dynamic, c'est à dire que les tableaux déclarés avec une gamme d'indices constante sont de longueur fixe. Ceci demeure en vigueur pour le reste du module dans lequel Option Static est utilisé et peut être modifié avec Option Dynamic.

Exemple

'' Compile with the "-lang fblite" compiler switch

#lang "fblite"

Option Dynamic

Dim foo(100) As Integer            ' declares a variable-length array

Option Static

Dim bar(100) As Integer            ' declares a fixed-length array


Différences de dialectes

  • Disponible seulement dans les dialectes -lang fblite et -lang qb.

Différences avec QB

  • Nouveau pour FreeBASIC
