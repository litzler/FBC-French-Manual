BitReset
 
Obtient la valeur d'un entier dont on a positionné un certain bit à zéro (0).

Syntaxe

#define BitReset( value, bit_number ) ((value) And Not (Cast(TypeOf(Value), 1) Shl (bit_number)))

Usage

result = BitReset( value, bit_number )

Paramètres

value
La valeur entière.
bit_number
L'indice du bit à mettre à zéro.

Valeur retournée

Retourne la valeur entière avec le bit spécifié mis à zéro.

Description

Cette macro se développe en une copie de la valeur entière value avec le bit bit_number "nettoyé" (à off, ou `0`). Se comporte comme `value And Not (1 Shl bit_number)`.

La plage valide de valeurs pour bit_number dépend de la taille, en bits, de`TypeOf(value)`, qui est `0` par le biais de `SizeOf(value) * 8 - 1`. Voir Limites des types de données standard pour un tableau des types de données standard et leurs tailles.

Example

Print BitReset(5,0)
Print Hex(BitReset(&h8000000000000001,63))

produira la sortie:

4
1

Différences de dialectes

  • Non disponible dans le dialecte -lang qb.

Différences avec QB

  • Nouveau pour FreeBASIC
