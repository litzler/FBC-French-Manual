Opérateur Mod= (Modulo et Affectation)
 
Divise une valeur et assigne le reste à une variable

Syntaxe

Declare Operator Mod= ( ByRef lhs As Integer, ByRef rhs As Integer )

Usage

lhs Mod= rhs

Paramètres

lhs
La variable Ã  affecter.
rhs
La valeur pour diviserlhs.

Description

Cet opérateur divise deux valeurs de type Integer et attribue le reste à la variable de son côté gauche (lhs). C'est fonctionnellement équivalent à:

lhs = lhs Mod rhs

Cet opérateur élève à la puissance et attribue une valeur à une variable. Il est fonctionnellement équivalent à:

Exemple

Dim n As Integer
n = 11
n Mod= 3
'' The result is 2
Print n
